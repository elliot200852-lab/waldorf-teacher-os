Differentiated Instruction Framework

差異化教學框架

從課程設計階段，
主動為每一位學生
建立多層次的學習路徑。

"差異化教學不是補救教學，
而是預設學生之間存在差異，
在課程設計階段主動為不同學習狀態的學生
建立多層次的學習路徑。" — TeacherOS CreatorHub 差異化教學框架

核心原則

在華德福教育中，
差異化教學並非額外的教學策略，
而是課程本身的內在結構。
每一堂課、每一次評量的設計都必須預先考慮：
這些學生不是一個齊一的群體，
而是各具獨特學習優勢、能力與動機的個體。

  • 不是補救教學

    差異化教學的對象不僅是「落後」的學生，
    而是全班所有學生。
    目標是為每一位學生設計最適切的學習路徑——
    讓高能力學生有延展挑戰，
    讓低能力學生有紮實的鷹架支撐，
    讓低動機學生找到課程與己身的意義連結。

  • 強制性的設計義務

    差異化思維不是可選項，
    而是每一份課程計畫的強制性要求。
    從主課程的 3 週 15 堂到專科課程的 45 分鐘，
    都必須內建差異化的思考。
    這是教師專業責任的一部分。

  • 多層次的學習路徑

    同一堂課程中，
    不同學生可能透過不同的學習入口
    （視覺、聽覺、動作、藝術）進入同一個主題；
    透過不同的支持層級
    （獨立挑戰、扶持成長、意義探索、安全開始）
    深化理解。
    課程的寬度與深度都必須足夠彈性。

雙軸強制設計

差異化教學的核心
是兩個互相獨立但都不可或缺的設計維度。
每一份課程設計都必須同時滿足兩軸的要求。

軸一：學習優勢

原則：每一堂課至少必須涵蓋
2–3 種學習優勢的入口點。
這不是為了迎合學習風格迷思，
而是為了確保課程有足夠的豐富度，
讓每一位學生至少有一種方式
與課程內容產生真實連結。

語言型

透過朗誦、討論、閱讀、寫作進入課程。
這些學生用言語思考，
在語言的流動中建構意義。

圖像型

透過圖表、圖像、視覺化筆記、
黑板畫與空間排列進入課程。
這些學生在圖像與空間中思考，
用眼睛而非耳朵認知。

動作型

透過身體移動、手工藝、
農業實作、戲劇表演進入課程。
這些學生在身體行動中思考，
具身學習是認知的基礎。

藝術型

透過音樂、繪畫、詩歌、雕塑進入課程。
這些學生透過美感與創意表達
連結課程意義，
將知識轉化為個人美感體驗。

軸二：能力 × 動機矩陣

原則：每一份課程設計都必須至少區分
「高挑戰層級」（A 型與 C 型學生）
與「扶持層級」（B 型與 D 型學生），
理想上應該設計四層完整的差異化策略。

2×2 能力 × 動機矩陣

縱軸：學習能力（從低到高）｜ 橫軸：學習動機（從低到高）

A 型：高能力高動機

延展挑戰型

  • 獨立深度探究
  • 同儕教導與領導
  • 跨領域連結與創意應用
  • 設計開放性問題
C 型：高能力低動機

意義探索型

  • 強化課題與個人興趣的連結
  • 減少重複性練習
  • 提供選擇自主度
  • 探討「為什麼重要？」
B 型：低能力高動機

扶持成長型

  • 小步驟成功建立信心
  • 具體的學習鷹架
  • 頻繁的鼓勵與反饋
  • 化抽象為具體示範
D 型：低能力低動機

安全開始型

  • 降低焦慮與威脅感
  • 從具體操作開始
  • 建立安全的學習社群
  • 尋找隱藏的學習優勢

課程設計的強制性檢查清單

每一份課程設計與評量工具
都必須同時滿足「軸一：學習優勢」
與「軸二：能力 × 動機」的要求。
這不是建議，而是課程完整性的必要條件。

  • 多入口點原則

    課程規劃中至少包含
    2–3 種學習優勢的學習入口點
    （語言、圖像、動作、藝術）。
    如果一堂課只有「講述」，表示設計不完整。

  • 四層差異化

    理想上，課程應該設計四層完整的差異化路徑：
    高挑戰（A 型）、意義探索（C 型）、
    扶持成長（B 型）、安全開始（D 型）。
    至少不能少於兩層（高挑戰 vs. 扶持）。

  • 課程寬度與深度

    「同一課題，不同深度」是核心邏輯。
    不是為不同學生設計「不同課題」，
    而是讓他們在同一課程主軸上
    以不同路徑與深度進行。

品質標準：五大核心原則

差異化教學的設計不是冷酷的學生分類，
而是有溫度的教學對話。
這五項品質標準確保課程設計
始終以學生的內在狀態與真實課堂場景為中心。

  • Rule 1：人文優先

    先描寫學生的內在心靈狀態，
    而非直接開列教學策略清單。
    「七年級學生剛從集體意識中醒來，渴望測試界線」
    遠優於
    「A 型學生需要獨立挑戰」。

  • Rule 2：拒絕抽象標籤

    「提供鷹架支撐」說得再好也不夠具體。
    必須展開成：
    「先透過黑板畫示範三個步驟，
    然後邀請學生在小組內重複一次，
    最後才是獨立嘗試。」

  • Rule 3：課堂現場感

    每一個策略都必須能被轉化成
    一個課堂畫面。
    讀者能否在心中看見：
    教室裡發生了什麼？誰站在哪裡？
    學生拿著什麼？老師說了什麼？

  • Rule 4：溫暖的語調

    用專業教師的語調，
    而非冷硬的 AI 分類報告。
    「這位學生正經歷……」
    好過「此學生屬於 D 類」。

  • Rule 5：可操作性

    老師讀完後，應該能明天直接用。
    不是理論，是行動方案。
    「明天我可以……」的語句
    應該很自然地浮現。

四層設計維度

差異化不是單一維度的工作。
課程可以在內容深度、學習過程、
產出形式、甚至環境安排上進行差異化。
以下四層維度提供完整的設計思路。

內容（Content）

同一課題，不同複雜度。
基礎版本聚焦核心概念，
進階版本則探討應用、
批判或跨領域連結。

過程（Process）

同一學習目標，不同路徑。
有人透過直接講述，
有人透過探究、
有人透過藝術創作。

產出（Product）

同一證明理解的期望，
不同表達形式。
可以寫評論文章、畫概念圖、
錄製影片、製作實體模型。

環境（Environment）

調整節奏、座位安排、互動方式。
有人需要獨處思考，
有人需要協作對話，
有人需要一對一確認。

課程設計自我檢查

完成課程設計後，
用以下問題檢驗你的設計
是否真正落實了差異化教學。

這堂課有多少種學習優勢的入口點？
（應至少 2–3 種：語言、圖像、動作、藝術）
A 型學生（高能力高動機）
有沒有延展挑戰與獨立探究的空間？
B 型學生（低能力高動機）
有沒有具體的鷹架支撐與小步驟成功？
C 型學生（高能力低動機）
有沒有被告知「為什麼這很重要」？
D 型學生（低能力低動機）
有沒有安全的開始點，避免立刻被挫折擊倒？
我有沒有從學生的內在心靈狀態開始思考，
而非套用標籤？
每一個教學策略
都能被轉化成一個課堂現場的具體畫面嗎？
這份設計讀起來像是
一個有溫度的教師在思考，
還是冷硬的 AI 分類報告？

與華德福教育的深層連結

差異化教學在華德福教育中，
並非現代教改的附加品，
而是施泰納人智學的本質要求。

  • 認識人的完整性

    施泰納強調每一個人
    都是身、心、靈的完整統一體。
    課程必須同時觸及思考（頭）、情感（心）、意志（手）。
    差異化教學要求我們看見
    每一位學生獨特的身心靈組合，
    而非將其簡化為「一種學習風格」。

  • 七年週期的發展邏輯

    七至九年級的學生
    正經歷劇烈的靈魂轉變。
    同齡的青少年，其內在發展狀態
    可能相差三至五年。
    差異化教學承認這種真實的發展時間差，
    而非強制所有學生進度齊一。

  • 尊重個體的使命

    華德福教育相信每一個孩子
    都肩負獨特的人生使命。
    課程設計不是將知識灌輸到
    被動接收的容器中，
    而是幫助每位學生發掘自身的內在力量
    與獨特的表現方式。

"差異化教學的終極目標不是分類學生，
而是讓每一位學生
都能在課程中找到屬於自己的位置——
一個被看見、被理解、被支持的位置。"

每個設計決定都回答同一個問題：
我們如何確保沒有任何一位學生
在課堂上成為隱形人？

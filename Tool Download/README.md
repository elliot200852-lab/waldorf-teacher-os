# Tool Download

TeacherOS 教師工具箱——可獨立下載使用的工具集。

## 可用工具

### [音檔轉逐字稿（macOS 版）](./音檔轉逐字稿-macOS/)

將錄音檔自動轉錄為 Markdown 逐字稿，含說話者辨識功能。

- **平台：** macOS（Apple Silicon：M1/M2/M3/M4）
- **功能：** 語音轉文字 + 自動區分說話者
- **用途：** 訪談錄音、會議記錄、課堂對話紀錄
- **安裝：** 下載資料夾後執行 `./setup.sh` 即可
- **詳細說明：** 見資料夾內 [README.md](./音檔轉逐字稿-macOS/README.md)

### [英文寫作訂正系統](./英文寫作訂正系統/)

學生貼上英文作文，系統自動標記拼字與文法錯誤，附中英文修正建議。

- **平台：** 瀏覽器（任何裝置，無需安裝）
- **線上使用：** <https://writing-correction-system-tw.web.app/>
- **功能：** 三層分析引擎（LanguageTool / Claude API / 內建規則）、色彩標記、雙語錯誤說明
- **用途：** 九年級英文寫作課堂訂正，含每日動態密碼與上課時間限制
- **附件：** [學生使用手冊](./英文寫作訂正系統/學生使用手冊-Writing-Checker.docx)（可列印公佈於班上）
- **詳細說明：** 見資料夾內 [README.md](./英文寫作訂正系統/README.md)

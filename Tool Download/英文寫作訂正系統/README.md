# 英文寫作訂正系統 English Writing Checker

華德福九年級英文寫作訂正工具——學生貼上作文，系統自動標記拼字與文法錯誤，並提供中英文修正建議。

## 線上使用

**公開網址：** <https://writing-correction-system-tw.web.app/>

直接用瀏覽器開啟即可使用，無需安裝。

## 功能

- **三層分析引擎：** LanguageTool（預設，數千條規則）→ Claude API（深度語意分析）→ 內建規則（離線備援）
- **色彩標記：** 黃色＝拼字/大小寫/標點，紅色＝文法/用字/句構
- **中英文說明：** 每個錯誤附帶雙語解釋，幫助學生理解「為什麼錯」
- **登入控管：** 學生帳號使用每日動態通行碼＋時段限制（週一至五 08:00–16:00／週六 08:00–12:00／週日不開放）
- **匯出功能：** 可匯出訂正報告供存檔

## 設計理念

這個工具是「檢查器」，不是「代寫器」。設計上刻意保留學生自主修正的空間——系統只指出問題與方向，修改的決定權在學生自己手上。每日動態密碼與時間限制的設計，是為了確保工具在教師陪伴的課堂脈絡中使用，避免學生在家過度依賴。

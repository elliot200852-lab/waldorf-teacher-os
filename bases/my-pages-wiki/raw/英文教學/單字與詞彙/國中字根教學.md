Unit1
① 字首：un– = not / opposite（不、相反）
國中出現字：
	•unhappy（不快樂的 → not happy）
	•unlucky（不幸運的）
	•unsafe（不安全的）

記憶法：
un + 形容詞 → 「不、相反」。
先抓出你熟的字：happy → unhappy；lucky → unlucky。
看到 un–，就先在腦中加一句：「不那樣」。
② 字首：re– = again / back（再一次、回到）
國中出現字：
	•return（回來、歸還 → re + turn 轉回去）
	•reread（再讀一次）
	•review（複習 → re + view 再看一次）
記憶法：
re = again / back。
老師常說 review、repeat，本質都是「再來一次」。
看到 re–，可以在心裡先翻成「再…一次 / 回…」。

③ 字首：dis– = not / opposite / away（不、相反、離開）
國中出現字：
	•disagree（不同意）
	•dislike（不喜歡）
	•dishonest（不誠實的）
記憶法：
dis– 常常跟「不、打散、分開」有關。
agree（同意）→ disagree（不同意）
like（喜歡）→ dislike（不喜歡）
看到 dis–，可以先想：「把原本的意思打翻」。
④ 字首：tele– = far（遠距離）
國中出現字：
	•telephone（電話 → 遠距離的聲音）
	•television（電視 → 遠距離的影像）
	•telephoto（ 望遠鏡頭）
記憶法：
tele = far。
telephone：tele（遠）＋ phone（聲音）→ 聲音傳很遠
television：tele（遠）＋ vision（影像）→ 影像傳很遠
只要看到 tele–，就想到「不用走到那裡也能連到遠方」。

⑤ 字根：port = carry（帶、搬運）
國中出現字：
•airport（機場 → air + port：飛機「進出」的港口）
•important（重要的 → im + port + ant：把事情「帶進心裡」）
•report（報告 → re + port：把資訊「再帶回來」給大家）
記憶法：
port 跟「搬、帶」有關：
•airport：飛機來來去去的地方
•report：把你知道的事情「帶回」老師、同學面前
以後看到 port，可以先問自己：「什麼東西在被帶來帶去？」
⑥ 字尾：–er = person / thing that…（做這件事的人 / 東西）
國中出現字：
•teacher（老師 → teach 的人）
•worker（工人 → work 的人）
•singer（歌手 → sing 的人）
•reader（讀者 → read 的人）
記憶法：
動詞 + er → 做這件事的「人」或「東西」。
teach → teacher
work → worker
sing → singer
看到 –er，就先在中文後面加一句：「…的人」。
⑦ 字尾：–est = the most（最…的）
國中出現字：
•biggest（最大的）
•tallest（最高的）
•oldest（最老的 / 最久的）
•youngest（最年輕的）
記憶法：
形容詞 + est → 「最…的」一個。
big → bigger → biggest
old → older → oldest
看到 –est，就想到「比大家都還…」。
⑧ 字尾：–ly = in this way（以這樣的方式地…）
國中出現字：
•slowly（慢慢地）
•quickly（快速地）
•quietly（安靜地）
•carefully（小心地）
記憶法：
形容詞 + ly → 變成「怎麼做」的副詞。
quick（快的）→ quickly（快快地做）
quiet（安靜的）→ quietly（安靜地做）
看到 –ly，可以先翻成「…地」。
⑨ 字尾：–ful = full of（充滿…的）
國中出現字：
•careful（小心的 → 充滿關心、小心）
•useful（有用的 → 充滿用途的）
•helpful（有幫助的 → 充滿幫助的）
記憶法：
ful = full → 滿滿的。
care + ful → 充滿 care 的 → careful
help + ful → 充滿 help 的 → helpful
看到 –ful，就想成「滿滿的…」。
⑩ 字尾：–less = without / no（沒有…的）
國中出現字：
•careless（粗心的 → 沒有小心的）
•useless（無用的 → 沒有用處的）
•hopeless（絕望的 → 沒有希望的）
記憶法：
less = without → 沒有。
careful（小心的）↔ careless（不小心的）
hopeful（有希望的）↔ hopeless（沒希望的）
看到 –less，就先在心裡加一句：「沒有…」。
unit2
① 字根：graph = write / draw（寫、畫、圖）
國中常見 / 生活字：
•graph（圖表）
•paragraph（段落 → 一小段寫出來的文字）
•geography（地理 → geo 土地 + graph 寫／畫，把地球「畫」下來）
記憶法：
graph 就是「用筆或圖，把東西寫下來／畫下來」。
•paragraph：para（旁邊）＋ graph → 一段寫在旁邊的文字
•geography：geo（地）＋ graph（畫）→ 把地球畫成地圖，就是地理
看到 graph，腦中先浮現「筆、圖、寫字」。
② 字根：form = shape（形狀、形式）
國中常見 / 生活字：
•uniform（制服 → uni 一樣 + form 形狀 → 全部穿一樣的樣子）
•information（資訊 → in 裡面 + form 形狀 → 把事情「整理成形」給你）
•transform（改變、變形 → trans 改變 + form 形狀）
記憶法：
form 就是「形狀、樣子」。
•uniform：一樣的樣子 → 制服
•information：資訊不是亂七八糟，是「被整理成有形的內容」
看到 form，可以問：這個字是不是在說「變成什麼樣子」？
③ 字根：vis / vid = see（看見）
國中常見 / 生活字：
•visit（拜訪 → 去看看）
•video（影片 → 可以看的東西）
•television（電視 → tele 遠 + vis 看 → 看遠方的畫面）
記憶法：
vis / vid 都跟「看」有關：
•visit：去別人家「看一看」
•video：你最常「看」的東西之一
看到 vis / vid，可以先聯想到「眼睛」「畫面」。
④ 字根：phon / phone = sound / voice（聲音）
國中常見 / 生活字：
•telephone（電話 → tele 遠 + phone 聲音 → 聲音傳得很遠）
•smartphone（智慧型手機 → 有很多「聲音相關」功能）
•headphone（耳機 → 帶在頭上的聲音）
記憶法：
phone 就是「聲音」。
•telephone：不用走過去也能把聲音送很遠
•headphone：把聲音掛在頭上
看到 phone，就想到「跟聲音、通話有關」。
⑤ 字根：dict = say / speak（說）
國中常見 / 生活字：
•dictionary（字典 → 幫你查怎麼「說」這個字）
•predict（預測 → pre 先 + dict 說 → 先說出會發生什麼）
記憶法：
dict 就是「說話、說出來」。

⑥ 字根：scrib / script = write（寫）
國中常見 / 生活字：
•describe（描述 → de 下 + scribe 寫 → 把細節寫下來）
•description（描述、說明 → describe 的名詞）
•subscribe（訂閱 → sub 底下 + scribe 寫名字 → 在底下寫名字的簽名、訂閱）
記憶法：
scrib / script 都跟「寫」有關。
•describe：把眼前看到的東西「寫」得清清楚楚
•description：那整段「寫下來的說明」就是 description
看到 scrib / script，就想到「筆、寫、文字」。
⑦ 字根：meter = measure（測量）
國中常見 / 生活字：
•meter（公尺 → 測量長度的單位）
•thermometer（溫度計 → thermo 熱 + meter 測量 → 測量熱度）
•speedometer（車速表 → 測量速度的表）
記憶法：
meter 就是「測量」。
看到 meter，可以直接翻成：「用來量…的工具或單位」。

⑧ 字根：micro = small（小）
國中常見 / 生活字：
•microphone（麥克風 → micro 小 + phone 聲音 → 把小小的聲音放大）
•microscope（顯微鏡 → 看很小、很小的東西）

記憶法：
micro = very small。
•microphone：原本你的聲音很小（micro），麥克風讓大家都聽得到
•microscope：讓你看到小到眼睛看不到的東西
看到 micro，可先想到「微小、細微」。
⑨ 字根：bio = life（生命）
國中常見 / 生活字：
•biology（生物學 → bio 生命 + logy 學問）
•biography（傳記 → bio 生命 + graphy 寫 → 寫一個人的生命故事）
記憶法：
bio 就是「生命」。看到 bio，可以先翻成「生命、活著」。
⑩ 字根：cycl / cycle = circle / wheel（圓圈、輪子、循環）
國中常見 / 生活字：
•bicycle（腳踏車 → bi 二 + cycle 輪子 → 有兩個輪子的車）
•recycle（回收 → re 再 + cycle 循環 → 讓東西再回到循環裡）
•motorcycle（機車 → moto 馬達 + cycle 輪子）
記憶法：
cycl / cycle 和「圓、輪子、循環」有關：
看到 cycle，可以想成「一圈一圈、一輪一輪」。

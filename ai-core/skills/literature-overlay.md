# 文學與語文課覆蓋層｜課堂設計（45/90/120 分鐘）

> 本覆蓋層由統一課程引擎 `lesson-engine.md` 自動載入。

## 文學課的隱蔽功能

文學課不是「讀課文、寫心得」。它是華德福教育中最直接觸及學生心魂的科目 — 文本是鏡子，讓學生在別人的故事裡看見自己。

對不同年級而言：

| 年級 | 文學課的發展對應功能 |
|------|---------------------|
| **七年級** | 透過傳記、冒險敘事、異文化故事，讓剛從童年醒來的個體在別人的生命中找到自己的影子。文學是「安全的冒險」— 不用出門就能走遍世界。 |
| **八年級** | 透過革命史、社會批判文學、戲劇衝突，為叛逆的能量提供合法出口。八年級需要「看見不公平」— 文學提供的不是答案，是讓憤怒有方向的素材。 |
| **九年級** | 透過多元觀點、道德灰色地帶、存在焦慮的文本，幫助從「非黑即白」走向能容納複雜性的心智。文學中的「沒有標準答案」是這個年級最需要的養分。九年級需要問：「我是誰？我所在的這個社會是什麼？」文學讓他們在故事中尋找。 |
| **十至十二年級** | 透過哲學性文本、世界文學經典、獨立批判性閱讀，培養自主的文學判斷力與文化理解力。從「被引導閱讀」過渡到「能自己選書、讀書、評價書」。 |

## 文學科專屬設計原則

以下原則**補充**通用引擎的十條原則，不覆蓋。
原則編號以 L（Literature）為前綴。

### 原則 L1：文本分層閱讀

文學課的核心困境：同一個班級的閱讀能力差異極大。解法不是簡化文本，是分層接觸：

- **全班主文本**：每個人都讀。教師選定核心段落在課堂精讀，確保即使未完整閱讀的學生也能參與。
- **教師精讀段落**：教師事先從主文本中選出 2-4 個關鍵段落，在課堂上朗讀帶領。這些段落是「沒讀完全文也能進入」的入口。
- **A/B 延伸閱讀**：一篇與主文本對話的延伸文本（同主題不同角度、同作者不同作品、同時代不同族群）。課堂上不深入討論，作為個人延伸探索。
- **回家閱讀**：下一堂課的主文本。Day N 結束時發放 Day N+1 的文本，附簡短框架（「注意一件事⋯⋯」）引導閱讀方向。

**AI 在階段一必須做的事**：
1. 確認主文本長度，判斷學生是否能在家讀完全文
2. 若全文過長，選出課堂精讀的核心段落（標記起訖點）
3. 建議 A/B 延伸閱讀的文本
4. 設計「注意一件事」的閱讀引導語

### 原則 L2：寫作技法累進

文學課的寫作不是「回答問題」，是「學會用文字做事」。每堂課（或每天主課程）引入一個寫作技法，逐步累加，後期可以融合使用。

**技法累進的設計邏輯**：

| 順序 | 方向 | 說明 |
|------|------|------|
| 第一層 | **感官入口**（意象、描寫） | 用看得見的東西承載看不見的感受。先學會「畫面」。 |
| 第二層 | **視角操作**（敘事觀點、角色聲音） | 用不同的眼睛看同一件事。學會「位置」改變意義。 |
| 第三層 | **意義承載**（象徵、隱喻） | 一件物品或一個畫面承載超過字面的重量。學會「讓讀者自己想」。 |
| 第四層 | **結構性技法**（對比、並置、倒敘） | 兩個東西之間的距離就是意義。學會「安排位置」。 |
| 第五層 | **融合**（學生自選技法組合） | 把工具疊在一起。不再分別使用，而是有機整合。 |

**每堂課必須做的事**：
1. 用 2 分鐘回顧前面已學的技法（「Day 1 學了什麼？」）
2. 引入當日新技法時，從文本中提取實例
3. 教師黑板示範（用自己的文字，不用文本原文）
4. 學生書寫時，明確標示當日技法的最低要求

**教師示範文的品質標準**：
- 不能太平淡（學生看不到技法在工作）
- 不能太華麗（學生覺得自己做不到）
- 要有文學的韻味，但用的是生活場景
- 最後一行要有「翻轉」或「餘韻」— 讓學生看到技法的深層用法

### 原則 L3：教師朗讀為情感錨點

文學課中，教師的朗讀不是「代替學生讀」，是「示範如何用聲音打開一個文本」。

**設計規則**：
- 每堂課至少有一段教師朗讀（主課程 120 分鐘建議 8-15 分鐘朗讀量）
- 朗讀段落必須是文本的**情感轉折點**，不是開頭或背景說明
- 朗讀後停頓 3-5 秒，不立即進入分析。讓文字的餘韻先落地
- 教師前一晚必須出聲排練至少一遍。朗讀不是「讀字」，是「表演場景」

**朗讀要領（通用）**：
- 對話段落：區分角色語氣。角色說話越輕描淡寫，讀者（教師）也越輕描淡寫 — 衝擊來自文字本身，不是教師的詮釋
- 描寫段落：放慢速度。讓每一個意象有時間在學生腦中成像
- 高潮段落：不需要大聲。控制的力量比爆發更有效。「壓低聲音」往往比「提高音量」更震撼
- 結尾段落：讀完停住。不說話。沉默是朗讀的一部分

### 原則 L4：討論引導轉折點

文學課的核心不是「教師告訴學生文本的意義」，是「學生自己在討論中發現」。

**轉折點設計**：
- 每堂課設計 2-3 個「轉折點」— 文本中不顯而易見但重要的洞見
- 轉折點必須透過學生討論浮現，不由教師直接講述
- 教師的角色是「埋問題」，不是「給答案」
- 如果轉折點在討論中沒有自然浮現，教師用一個精準的提問引導

**分層討論結構**：
- **第一輪（全班，所有人能回答）**：感受性問題（「哪個畫面留在你心裡？」「這個角色讓你想到什麼？」）
- **第二輪（引向深層）**：連結性問題（「這跟昨天讀的有什麼關係？」「這是巧合還是結構？」）
- **第三輪（引向轉折點）**：挑戰性問題（「如果 X 是對的，那 Y 怎麼解釋？」「作者為什麼不直接說？」）
- 不需要三輪都完成 — 看教室節奏決定深度

**AI 在階段二必須做的事**：
1. 為每堂課設計 2-3 個轉折點
2. 為每個轉折點設計引導問題（從全班到深層的梯度）
3. 標記哪些轉折點「如果沒浮現，教師用哪個問題引導」

### 原則 L5：書寫連結生命經驗

文學課的書寫不是考試式的「文本分析」，是「從文本出發，抵達自己的生命」。

**書寫設計邏輯**：
- 書寫題目必須同時連結**文本**和**學生的個人經驗**
- 題目的措辭應該是開放的、邀請式的，不是指令式的
- 每次書寫至少有一個技法要求（但不超過兩個 — 技法是工具，不是枷鎖）
- 書寫形式要有變化：短文、詩歌、生活雜記、FB 留言、角色書寫、宣言 — 不能每天都寫同一種

**DI 書寫鷹架的通用結構**：

| 層 | 引導策略 |
|----|---------|
| **D** | 起手式寫在黑板上。教師前 3 分鐘走到 D 層確認動筆。字數要求降低（入門層 ≈ 全班的 60-70%）。重點是「動筆」，不是「寫好」。 |
| **C** | 結構引導（「分 N 段，第一段寫⋯第二段寫⋯」）。有一個技法要求。字數正常。 |
| **B** | 方向引導但不限制路徑。禁止抽象形容（「很重要」「很感動」）。兩個技法要求。 |
| **A** | 自由發揮 + 挑戰。融合多個技法。進階：嘗試作者的手法、從物品的視角寫、對某個對象說話。 |

### 原則 L6：主題弧線意識（主課程專用）

主課程（120 分鐘，連續 3 週）的文學教學有一條跨越整個 Block 的主題弧線。每一天不是獨立的「讀一篇文章」，而是弧線上的一個點。

**弧線設計原則**：
- 每天的主題用 1-2 個字命名（入口、漂流、沉默、根、汗、海、我、回音）
- 前半段向下沉（越來越重、越來越複雜）
- 中段有一個轉折（從苦難到力量、從個人到結構）
- 後半段向上收（回到「我」、回到當下、回到自己的位置）
- 最後一天是回顧 — 不加新素材，讓學生整理、分享、反思

**課堂上的弧線可見性**：
- 板書上每天更新弧線圖（Day 1 → Day 2 → ★今天★ → Day N）
- 每天包含前一天的文本/主題/作者/技法，讓老師不用翻資料
- 每天開場用 2-3 句話回顧前面走過的路（「三天了。Day 1⋯⋯Day 2⋯⋯Day 3⋯⋯」）
- 每天結尾預告明天（「明天，我們要⋯⋯」），種下期待的種子

**AI 在階段一必須做的事**：
1. 確認今天在弧線中的位置
2. 設計從昨天到今天的橋接語
3. 設計今天到明天的預告語
4. 確認板書中弧線圖的完整內容

### 原則 L7：板書作為視覺論證

文學課的板書不只是筆記，是教師用粉筆建構的**視覺論證**。學生看到板書，就看到這堂課的思維結構。

**板書設計規則**：
- 教案中必須有獨立的「板書規劃」段落，讓老師打開就能抄
- 板書分面規劃（左/中左/中右/右），依課程進度逐步展開
- 包含以下類型：
  - **弧線圖**：主題弧線 + 每天的文本/技法
  - **結構圖**：文本中的人物關係、事件結構、對比結構
  - **技法總覽**：累進的技法清單（Day 1 意象、Day 2 觀點⋯⋯）
  - **書寫提示**：題目 + 起手式
- 結構圖不是「文本摘要」— 它是讓學生「看見」某個模式的工具

## 階段一補充：文學科研究任務

以下任務在通用引擎的階段一之上**追加**，不取代。

### L-研究 1：文本教學潛力分析

1. 文本長度與閱讀難度（學生能否在家獨立讀完？需要教師選段嗎？）
2. 情感轉折點在哪裡？（教師朗讀的段落選擇）
3. 哪些段落適合全班精讀？哪些適合只給 A/B？
4. 文本中可提取的寫作技法是什麼？（從文本到技法的轉換路徑）

### L-研究 2：寫作技法定位

1. 今天的技法在累進序列中的位置（第幾層？前面已學了什麼？）
2. 這個技法在文本中的具體實例（至少 2-3 個）
3. 這個技法的教師示範文方向（生活場景 + 技法展現 + 結尾翻轉）
4. DI 寫作引導的四層設計預案

### L-研究 3：主題弧線定位（主課程適用）

1. 今天在弧線中的位置與功能
2. 從昨天到今天的轉折是什麼？（不只是主題轉換，是心理層次的推進）
3. 今天到明天的橋接方向
4. 轉折點設計：2-3 個，附引導問題梯度

### L-研究 4：朗讀段落選擇

1. 從文本中選出 2-4 個教師朗讀段落（標記起訖點）
2. 每段朗讀要領（語氣、節奏、關鍵時刻的處理方式）
3. 估算朗讀時間總計（控制在 8-15 分鐘內）

## 階段二補充：文學科教案元素

### 教案格式補充欄位

在通用教案格式之上，文學課教案必須包含：

```markdown
## 簡案（靈魂版）
> 一頁看完就能站上講台。
- 今天的一句話
- 節奏表（時間 / 段落 / 做什麼 / 靈魂）
- 轉折點清單

## 板書規劃（獨立完整版）
> 老師看這一節就知道黑板上寫什麼。
- 分面板書內容（左/中/右）
- 弧線圖含每天的文本/人物/技法（主課程）
- 結構圖
- 書寫提示 + 起手式

## 教師引導問題總覽
> 上課時翻到這頁就能用。
- 按段落分類的所有問題

## 教師講稿與結語
> 只讀一頁就讀這頁。
- 開場的聲音
- 核心的畫面
- 最重要的一句話
- 結語全文
```

### 教師動作補充

文學課教案中需要特別標記的教師動作：

| 標記 | 說明 |
|------|------|
| **教師朗讀** | 標明段落起訖、朗讀要領（語氣、關鍵時刻、停頓點） |
| **教師引導語** | 不是逐字稿，是「要命中的節拍」— 教師保留即興空間 |
| **教師示範文** | 用生活場景展現技法，有文學韻味但不過度華麗 |
| **板書時機** | 標明「邊讀邊畫」或「討論後補上」 |
| **[DI]** | 標記有差異化分層設計的環節 |
| **⚡** | 最容易超時的環節 + 備案 |

## 階段三補充：文學科稽核維度

在通用引擎的 10 項稽核維度之上追加：

| 審核維度 | 檢查問題 |
|---------|---------|
| **文本分層閱讀** | 是否為不同閱讀能力的學生提供了多層次的文本接觸方式？C/D 層即使沒讀完全文也能參與核心討論嗎？ |
| **教師朗讀品質** | 是否選擇了情感轉折點而非背景段落？是否提供了朗讀要領？朗讀量是否合理（8-15 分鐘）？ |
| **寫作技法累進** | 今天的技法是否建立在前幾天的基礎上？是否回顧了已學技法？是否有教師示範文？ |
| **討論轉折點** | 是否設計了 2-3 個轉折點？是否都有引導問題的梯度（全班→深層）？是否標記了「如果沒浮現怎麼辦」？ |
| **書寫連結生命** | 書寫題目是否同時連結文本和學生個人經驗？是否有 DI 四層鷹架？起手式是否寫在黑板上？ |
| **弧線連貫性**（主課程） | 是否清楚標示今天在弧線中的位置？是否有橋接語和預告語？板書弧線圖是否完整？ |

## 階段四補充：文學科產出規格

### 教師教案補充

文學課教案在通用格式之上，必須包含以下獨立段落（不嵌入流程中，方便教師翻閱）：

1. **簡案（靈魂版）** — 教案最前面，一頁內
2. **板書規劃** — 獨立段落，所有面板書的完整內容
3. **教師引導問題總覽** — 獨立段落，按課程段落分類
4. **教師講稿與結語** — 教案最後面，含開場/核心/最重要的一句話/結語

### 文學課無獨立學習單

主課程文學課的書寫在工作本上進行，不另印學習單。教案中的「書寫提示 + 起手式」直接寫在板書上。

例外：若有特殊活動（討論卡、角色卡、選段閱讀單），由教案內說明格式。

## 使用範例（文學科觸發情境）

**教師可能的說法**：
- 「設計一堂臺灣文學主課程」→ 120 分鐘，載入 literature-overlay
- 「幫我備 Day 5 的課」→ 從 session.yaml 推斷科目，載入 overlay
- 「這篇文章可以怎麼上？」→ 確認班級和格式後載入
- 「我有一篇黃春明的小說想用在九年級」→ 確認科目後載入

**AI 啟動流程**：
1. lesson-engine 載入 → Step 0 確認參數
2. 偵測科目為 `ml-taiwan-literature` / `literature` / `language-arts` → 載入 `literature-overlay.md`
3. 若有科目哲學 reference（`ai-core/reference/subject-history.yaml` 等），一併載入
4. 進入五階段工作流，每個階段執行通用 + 本覆蓋層的補充任務

## 注意事項

- 本覆蓋層適用於 **七至十二年級** 的文學與語文類課程
- 低年級（一至六年級）的故事教學節奏與本覆蓋層差異較大，未來需要獨立的 `storytelling-overlay.md`
- **寫作技法累進序列不是固定的**。L2 原則中的五層是建議方向，教師可依文本特性調整順序
- 45 分鐘的專科語文課可以使用本覆蓋層，但主題弧線原則（L6）僅適用於連續課程（主課程或連堂）
- 教師朗讀是本覆蓋層的核心設計元素。如果一堂文學課沒有教師朗讀，稽核時應標記 ⚠️
- 討論轉折點的「不講述，引導浮現」是硬性要求。AI 設計教案時不可把轉折點寫成「教師向學生解釋⋯⋯」
- 板書規劃必須獨立成段落，不可散落在流程各處。教師要能「翻到那一頁就抄」

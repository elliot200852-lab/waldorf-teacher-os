# skill: story-writer — 臺灣的故事・三件套撰寫

將素材包精煉為三份產出文件：故事內容、教師說書稿、投影圖像清單。

## 適用對象

David（管理員）。

## 根目錄

以 Repo 根目錄為基準。

## 讀取的檔案

1. `{workspace}/projects/stories-of-taiwan/current-task.yaml` — 今日選題
2. `{workspace}/projects/stories-of-taiwan/stories/[區塊]/[ID]/raw-materials.md` — 素材包
3. `{workspace}/projects/stories-of-taiwan/index.yaml` — 索引（確認前後篇連結）
4. `{workspace}/projects/stories-of-taiwan/themes/theme-skeleton.yaml` — 區塊脈絡

## 入口驗證（由 story-daily pipeline 強制執行）

**權威來源：** `story-daily.md` Checkpoint 矩陣 Step 2→3 列。以下為快速參照，若有衝突以 checkpoint 矩陣為準。

啟動本技能前，必須確認以下條件全部滿足，否則回報 FAIL 並停止：

1. `stories/[區塊]/[ID]/raw-materials.md` 存在
2. `raw-materials.md` 字數 > 500
3. `raw-materials.md` 的「資料來源」段落至少包含 3 筆含 URL 的來源
4. `projects/stories-of-taiwan/current-task.yaml` 存在且 `story_id` 非空

## 執行步驟

### Step 1 — 產出故事內容（content）

撰寫 `[ID]/content.md`（存入故事子資料夾），遵循以下規格：

**故事本文**（以自然篇幅為準，不設硬性字數上限）：
- 敘事體，有場景、有人物、有感官細節
- 語言程度適合五年級（10-11 歲）——具體、可視覺化、情感可觸及
- 不說教、不灌輸——讓故事自己說話
- 華德福說故事原則：圖像先於概念，體驗先於結論

**史實錨點**：
- 每個核心事實標註來源（從素材包對應）
- 至少兩個獨立來源交叉確認的事實才能以斷言形式呈現
- 僅有單一來源的事實，以「據 XXX 記載」等方式標註

**地理標記**：
- 故事發生地的地名（歷史名稱 + 現代名稱）
- 經緯度（如有）
- 簡述現在這個地方的樣貌

**延伸線索**：
- 與前一篇、後一篇的連結
- 跨齡提示（六到九年級可如何深化）

### Step 2 — 產出教師說書稿（narration）

撰寫 `[ID]/narration.md`（存入故事子資料夾），遵循以下規格：

**開場**（1-2 分鐘）：
- 口語化引入，可回顧前一天的故事
- 設置懸念或提問，引起好奇

**說書本文**（5-8 分鐘）：
- 逐字稿，可直接朗讀
- 標註語氣變化：（低沉）（加速）（停頓三秒）等
- 華德福說故事節奏：吸氣（緊張/高潮）↔ 呼氣（舒緩/沉澱）
- 適當安排投影圖像展示的時機點

**收尾**（1 分鐘）：
- 不做結論——留下問題或圖像讓學生自己消化
- 連結明天的線索（如有）

**教師備註**：
- 可能的學生提問與回應方向
- 情感注意點（哪些內容可能觸動特定學生）
- 差異化提示：A 類（高能力高動機）可延伸什麼；D 類（低能力低動機）需要什麼支持

### Step 3 — 產出投影圖像清單（images）

撰寫 `[ID]/images.md`（存入故事子資料夾），遵循以下規格：

- 主圖：與故事核心最相關的一張圖像
- 輔助圖：地圖、文物、建築等（2-4 張）
- 每張圖標註：來源、授權、在故事中的角色
- 投影建議：展示順序與時機（對應說書稿的段落）
- 替代方案：無法投影時的實體列印建議

## 輸出格式

三份 Markdown 文件，存入 `{workspace}/projects/stories-of-taiwan/stories/[區塊]/[ID]/`：
- `content.md`
- `narration.md`
- `images.md`

## 寫作品質錨點

1. **不是百科全書**——是一個讓十歲孩子聽了會記住的故事
2. **有人物**——即使是地質或遺址，也要找到「人」的切入角度
3. **有畫面**——讀者應該在腦中看到場景，不只是讀到資訊
4. **有情感**——不煽情，但要觸動——讓學生感到「這跟我有關」
5. **有誠實**——不迴避困難的歷史（如殖民、衝突），但用適齡的方式呈現

## 五年級寫作校準（從 A002 兩版比較提取）

1. **開場必有回顧 + 懸念**——先用一兩句連結前一篇（「上一次我們說了⋯」），再拋出一個學生想知道答案的問題。不要直接跳入新主題
2. **比喻要用學生身體或日常經驗**——「比你的頭還大」「像從模子裡倒出來的」「像手指的指紋」。十歲孩子做過黏土、看過自己的手，這些比喻立刻有畫面。不要用他們沒經歷過的抽象意象
3. **短句對比製造張力**——「一邊往上長，一邊被水往下切」這種結構比長段解釋更有力。用對立的畫面讓孩子感受到力量的拔河
4. **收尾要勾住明天**——最後一段不是結論，是懸念。「你猜到了嗎？」「下一次我們要認識⋯」讓學生帶著好奇離開教室。不要用漂亮的修辭收在自己身上
5. **用擬人化串連系列**——地質力量是「雕刻師」、是「角色」，不是抽象概念。讓五年級覺得水和火是有個性的人，有脾氣、有耐心
6. **連結學生此刻的身體位置**——「你腳下踩的平原」「蹲下來看看旁邊的水溝」，讓知識落地到他們放學會經過的地方。最好的收尾是讓學生今天回家路上就會想到這個故事

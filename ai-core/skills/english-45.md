# 英文科覆蓋層｜45 分鐘課堂設計

> 本覆蓋層必須與通用引擎 `subject-lesson-45.md` 一起載入。
> 引擎定義方法論（五階段工作流、六條通用原則、節奏模板），本覆蓋層定義英文課的科目專屬內容。

---

## 英文課的隱蔽功能

英文課不只是語言訓練——它是華德福教育中少數能同時觸及「思考的精確性」與「情感的細膩度」的科目。

對不同年級而言：

| 年級 | 英文課的發展對應功能 |
|------|---------------------|
| **七年級** | 透過故事、傳記、角色扮演，讓剛從集體中醒來的個體找到「用另一種語言表達自我」的新鮮出口。英文作為「第二身份」的安全實驗場。 |
| **八年級** | 透過辯論、觀點表達、因果論述，為叛逆的能量提供理性出口。英文中的邏輯連接詞（because, however, therefore）是內在因果思維的外化工具。 |
| **九年級** | 透過文學中的道德掙扎、人物的灰色地帶、語言中的細微差異，幫助從二元極端走向能容納複雜性的心智。詩歌的多義性是打破非黑即白的最佳媒介。 |

## 英文科專屬設計原則

以下原則**補充**通用引擎的六條原則，不覆蓋。

### 原則 E1：文本可及性優先

選用文本必須考慮班級英語程度的實際現況。原則：

- **原始文本**保持完整，不刪改（尊重作者與文本的完整性）
- 若文本難度偏高 → **同時準備簡易散文版（Prose Retelling）**作為入門層鷹架
- 散文版不是翻譯，而是用較簡單的英文重述核心意涵，保留關鍵意象與情感
- 入門層學生先讀散文版建立理解基礎，再嘗試接觸原始文本的片段

### 原則 E2：文法與句型教學

英文課的文法教學有兩種模式，由教師依據該堂課的主要目的決定：

- **模式 A（有機整合）**：文學/情感導向課堂。文法從文本中有機長出來，遇到重要句型時**提點**但不打斷情感流動。
- **模式 B（獨立教授）**：文法/句型導向課堂。文法是主角，但仍需提供有意義的語境，而非抽離操練。

無論哪種模式，文法教學都遵循認知遞進：**發現**（在語境中注意到）→ **感受**（理解為什麼有效）→ **運用**（在自己的表達中嘗試）。

**鷹架策略**：給學生支撐結構，讓他們在安全框架內嘗試產出。層次從「句型框架 + 填空」（入門層，如 `I chose ___ because ___.`）到「開放運用」（挑戰層，用目標句型自由造句）。入門層的鷹架要夠具體但不能只有一個正確答案；挑戰層要有真實認知挑戰而非單純增量。所有層次印在同一張學習單上，學生可自行選擇。

**造句貼近文本**：造句練習的題目與情境應盡可能貼近當堂課的閱讀文本（小說、故事、詩歌），讓句型操練與閱讀理解互相強化。避免使用與文本無關的抽象例句——學生在造句時同時也在回味、重新理解故事的內容與情感，這比機械式操練更有效。

AI 在階段一研究時，需判斷適用模式 A 或 B，並在研究報告中標記。

### 原則 E3：聲音先於文字

英文課的開場應盡可能包含「聲音」——朗誦、合唱朗讀、教師敘述。華德福外語教學的核心信念：語言先通過耳朵進入心靈，再通過眼睛進入頭腦。

開場模式建議（依年級調整）：

| 年級 | 開場聲音活動 |
|------|-------------|
| 七年級 | 合唱朗讀（Choral Reading）、節奏拍手配合短詩 |
| 八年級 | 教師敘述引導 + 學生接力朗讀、戲劇式對話朗讀 |
| 九年級 | 教師全詩朗誦（不看文本）、安靜聆聽後的個人感受分享 |

### 原則 E4：詩歌教學法

詩歌是華德福英文課的核心素材。詩歌教學遵循以下順序（不可跳過）：

1. **第一印象**：教師朗誦全詩（學生不看文本），學生只聽、只感受
2. **再次聆聽**：發下文本，教師第二次朗誦，學生跟讀或默讀
3. **文本探索**：從意象、聲音、情感切入（不從「主旨」切入）
4. **個人連結**：學生找到與自己生命經驗的共鳴點
5. **語言分析**：最後才進入文法結構、修辭技巧的討論

**禁止**：直接從「這首詩在說什麼」開始教學——這會扼殺詩歌的生命力。

## 階段一補充：英文科研究任務

在通用引擎的研究任務之後，追加以下英文科專屬任務：

### E-研究 1：語言難度分析

- 估算文本的 Lexile 等級或可讀性指數（若為已知文本）
- 識別高頻障礙詞彙（超出該年級常見詞彙範圍的詞）
- 判斷是否需要準備 Prose Retelling 版本
- 標記文本中的文化特定參照（需要教師額外解釋的背景知識）

### E-研究 2：文法整合機會

- 若教師指定了要融入的文法結構 → 在文本中標記目標句型的自然出現位置
- 若教師未指定 → 從文本中識別最自然可教的 1-2 個文法結構，建議給教師
- 規劃文法整合的認知遞進路線：發現 → 感受 → 運用

### E-研究 3：朗誦與聲音潛力

- 分析文本的聲音特質（韻律、重複結構、對話性）
- 建議適合的朗誦方式（教師獨誦 / 合唱朗讀 / 分角色 / 接力）
- 若為詩歌：確認五步詩歌教學法的可行性

### 教師動作補充

英文科教師動作中，除了通用引擎要求的「提示點而非逐字稿」之外，需特別標記：

- **需要朗誦的段落**：標記「教師朗誦」並註明建議的朗誦方式（語速、情緒、停頓點）
- **黑板板書規劃**：列出需要寫在黑板上的鷹架句型、關鍵詞彙
- **過渡語言**：英文課中活動轉換的過渡語建議（用英文），幫助維持英語氛圍

## 階段四補充：英文科產出規格

### 教師教案補充

英文科教師教案除通用格式外，必須包含：

- **文法整合總覽表**（見階段二格式）
- **朗誦標記**：需要教師朗誦的段落用 `🎤` 標記
- **黑板規劃**：用 `📝 Board` 標記需要板書的內容

### 學生學習單補充

英文科學習單除通用格式外：

**視覺引導圖標**（英文科專用）：
- 📖 閱讀素材（Reading Material）
- 🔎 偵探/分析任務（Detective Task）
- 🔬 深度挑戰（Deep Challenge）
- 💬 討論/選擇（Discussion/Choice）
- ✍️ 書寫（Writing）
- 🔍 Follow-up

**正面額外內容**：
- 若有 Prose Retelling → 原始文本與散文版並排或上下對照
- 關鍵詞彙提示欄（Key Vocabulary）：列出 5-8 個重要詞彙，附簡短英文定義或同義詞

**背面額外內容**：
- 挑戰層活動可包含「選擇性延伸」：如果時間充裕，學生可自行挑戰的額外問題

## 使用範例（英文科觸發情境）

教師的輸入可能是：

- 「幫我設計一堂九年級英文課，主題是 moral dilemma」
- 「我想用 Robert Frost 的 The Road Not Taken 來上一堂課」
- 「九年級下週要上 narrative writing，文法想融入 relative clauses」
- 「這首詩可以怎麼上？」（附上一首詩）
- 「8A 英文，主題是 description of places，想練 there is / there are」
- 「七年級英文，用一個冒險故事來上」

AI 的啟動流程：
1. 從觸發語解析出班級代碼與科目 → 載入通用引擎 + 本覆蓋層
2. 執行 Step 0（載入發展樣態、科目哲學、班級 DI 資料）
3. 進入五階段工作流（通用階段 + 本覆蓋層的補充項目）

## 注意事項

- 本覆蓋層適用於七至九年級的英文課，不限於九年級
- 詩歌教學法（原則 E4）的五步順序是硬性約束，不可跳過或調換
- 文法整合總覽表是英文科教案的必備區塊，即使該堂課無特定文法焦點，也要標記文本中自然出現的句型
- Prose Retelling 的品質很重要——它不是「翻譯成簡單英文」，而是用較簡單的語言重新敘述，保留文本的情感核心與關鍵意象
- 英文課中英文的使用比例：教師指令與過渡語盡量用英文，深度討論與情感分享可視學生程度切換中英文

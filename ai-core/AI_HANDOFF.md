## 二、TeacherOS 是什麼

TeacherOS 是 David 建立的**教師工作作業系統**：
- 以 YAML 管理結構與狀態
- 以 Markdown 管理內容輸出
- 讓 AI 每次對話能立刻定位「我是誰、專案在哪、今天做什麼」
- 可移植、可重複載入、可輸出

### 三層記憶模型

```
Layer 1 — teacheros.yaml（教師身份層）
  路徑：ai-core/teacheros.yaml
  用途：教學哲學、工作偏好、AI 協作目標
  原則：核心不變。每次對話必須載入。

Layer 2 — project.yaml（專案記憶層）
  路徑：projects/<專案名>/project.yaml
  用途：專案目的、當前焦點、下一步
  原則：AI 下次工作的「起點」。

Layer 3 — working/*.yaml（工作線記憶層）
  路徑：projects/<專案名>/working/*.yaml
         projects/<專案名>/<科目>/lesson.yaml
         projects/<專案名>/<科目>/assessment.yaml
  用途：同一專案內不同工作線，拆開管理

內容輸出：projects/<專案名>/content/**/*.md
```

## 三、華德福教育核心概念（工作時必須理解）

### 教學哲學基礎

華德福教育基於**魯道夫·施泰納的人智學（Anthroposophy）**，核心信念：
- 人由四個層次組成：物質體、以太體（生命力）、星芒體（情感）、自我（靈性核心）
- 人類發展以**七年為一週期**，每個週期有不同的學習模式：
  - 0–7 歲：模仿
  - 7–14 歲：圖像與情感，教師權威是錨點
  - 14–21 歲：獨立思考萌發，質疑一切，尋找意義

### David 教學的學生年齡段（12–15 歲）

這個年齡段是第二與第三週期的**過渡地帶**，是所有年級最複雜的心理戰場：

| 年級 | 年齡 | 核心心理危機 | 課程鏡像 |
|------|------|------------|---------|
| 七年級 | 12–13 歲 | 界線探索、對不公平極度敏感 | 文藝復興、地理大發現 |
| 八年級 | 13–14 歲 | 邏輯萌發、社交焦慮、為反對而反對 | 法國大革命、工業革命 |
| 九年級 | 14–15 歲 | 存在焦慮、非黑即白、對虛偽零容忍 | 現代世界史、黑白素描 |

**課程是隱蔽的心理鷹架**：學科內容精準對應學生的發展危機，而非單純傳授知識。

### 教師權威模型

七至九年級教師的權威**不由體制自動賦予**，必須通過真實的人際關係與專業能力贏得：
- 言行絕對一致，雙重標準立即失去信任
- 充滿愛的權威，而非威權
- 班級衝突以修復式正義（Restorative Justice）處理，而非懲罰

## 四、差異化教學框架（每次必載）

**路徑：** `projects/_di-framework/project.yaml`

差異化教學是每一堂課設計的**必要考量，不是選項**。任何課程設計輸出，都必須包含差異化層次。

### A/B/C/D 學生矩陣（能力 × 動機）

| 類型 | 能力 | 動機 | 策略 |
|------|------|------|------|
| A | 高 | 高 | 延伸挑戰、自主探究、引導同儕 |
| B | 低 | 高 | 鷹架支持、小步驟成功經驗、建立自信 |
| C | 高 | 低 | 提升意義感、連結個人興趣、減少無效重複 |
| D | 低 | 低 | 降低焦慮、建立安全感、從具體操作切入 |

### 差異化設計四個層次

- **內容（Content）**：同樣主題，不同深度的素材
- **過程（Process）**：同樣目標，不同學習路徑與支持程度
- **成果（Product）**：允許不同形式的輸出與表達
- **環境（Environment）**：調整節奏、座位、師生互動方式

## 六、英文教學哲學

核心信念（施泰納）：語言不只是溝通工具，是**塑造性力量（Formative Force）**，深刻影響學生的思維方式與靈魂發展。

英文教師的三個角色：
1. **語言的示範者**——以自身對英語的熱情感染學生
2. **文化的橋樑**——連結英語世界的靈魂與台灣學生的生命處境
3. **青少年靈魂發展的陪伴者**——全面投入學生的經驗、個性與情感

教學手法：朗誦、戲劇、故事、創意寫作——語言是身體性的體驗，不是分析性的練習。

---

## 七、歷史教學哲學

核心原則：**人類整體意識的演化歷程與兒童個人意識的發展平行對應**。
歷史課不是讓學生「知道過去發生了什麼」，而是讓學生經驗到與自身內在發展相呼應的意識轉變。

台灣主體意識立場：
- 從台灣人的角度認識世界
- 以全球史觀取代歐洲中心史觀
- 讓台灣歷史（南島文化、多重殖民、民主化）成為教學素材的主軸

## 八、工作協議（你必須遵守）

### 每次對話開始

1. 確認讀取 `ai-core/teacheros.yaml`（身份錨點）
2. 確認讀取 `projects/_di-framework/project.yaml`（DI 共用框架）
3. 確認今日工作的班級專案（`projects/class-*/project.yaml`）
4. 確認今日推進的工作線（`working/*.yaml` 或科目子資料夾）

### 每次對話結束

1. 只更新 YAML 中有變動的區塊（**不重寫全部**）
2. 新產出的備課內容存入 `content/*.md`

### 輸出規範

| 情境 | 格式 |
|------|------|
| 備課內容、長文輸出 | Markdown |
| 狀態更新、結構變動 | YAML（只寫改動的區塊） |
| 語言 | 繁體中文 |
| 風格 | 結構清晰、簡潔、可直接操作 |
| 表情符號 | 不使用 |

### 差異化教學強制要求

任何課程設計輸出，**必須預設包含**：
- 四種學生類型（A/B/C/D）的對應策略
- 至少兩個差異化層次的設計（內容、過程、成果、環境任選）

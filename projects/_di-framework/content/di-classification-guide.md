# DI 學生分類指南
## 差異化教學（Differentiated Instruction）實踐工具

---

## 一、系統總覽

本指南協助您系統化地辨識和分類學生的學習特性。我們使用**能力×動機矩陣**與**學習優勢**兩個維度，幫助您為每位學生設計最合適的教學策略。

### 重要提醒

- **標籤僅供教師與AI使用**，永遠不向學生或家長公開透露
- **分類具有流動性**，應定期更新（每個月/每個教學階段）
- **同一學生在不同科目可能類型不同**（例如：在數學是B型，在語文是A型）
- **分類是工具，不是判斷**——目的是為了更好地支持每位學生

---

## 二、能力×動機矩陣

### A型：高能力、高動機
**特徵**: 學生已掌握基本概念，主動投入學習，樂於挑戰

**教學策略**
- 延伸挑戰與深化探究
- 引導自主學習與獨立研究
- 培養學生成為同儕教練/領導者
- 提供開放式、跨領域的複雜任務

**在班級中的角色**: 學習榜樣、小組領導者、同儕支援資源

---

### B型：低能力、高動機
**特徵**: 學生願意嘗試但掌握得較慢，需要更多支持才能成功

**教學策略**
- 細分步驟，設計「小步成功」的經驗
- 提供具體的鷹架支持（示範、檢查清單、模板）
- 頻繁的正面回饋與成功確認
- 建立自信心，強調進步而非完美

**在班級中的角色**: 認真學習者、需要信心建立、與同儕互助的絕佳對象

---

### C型：高能力、低動機
**特徵**: 學生有能力完成任務，但缺乏投入感或興趣

**教學策略**
- 連結個人興趣與生活經驗，提升意義感
- 減少無意義的重複練習
- 提供選擇權與自主性
- 引導反思「為什麼學這個」

**在班級中的角色**: 潛力未發揮者、需要意義建立、可能成為優秀貢獻者

---

### D型：低能力、低動機
**特徵**: 學生既不掌握基本概念，也尚未展現投入動機

**教學策略**
- 優先降低學習焦慮，建立安全感與信任
- 從具體操作、感官體驗切入
- 設定極小且明確的目標，確保初期成功
- 一對一陪伴或小組支持
- 探索根本原因（學習困難、家庭因素、先備知識缺乏等）

**在班級中的角色**: 需要密切注意的學生、需要特別的愛與耐心、可能需要轉介進一步評估

## 三、決策樹：如何分類學生

```
這位學生在【科目/領域】中...

                             開始
                              ↓
              他/她能獨立完成大部分課堂任務嗎？
                    ↙                    ↘
                  否                      是
                  ↓                       ↓
          他/她願意嘗試新挑戰嗎？    他/她對學習感興趣嗎？
            ↙            ↘              ↙            ↘
           是              否            是              否
           ↓               ↓             ↓               ↓

         B型            D型           A型             C型
      低能力           低能力         高能力           高能力
      高動機           低動機         高動機           低動機

      (需要鷹架)     (需要安全感)   (需要挑戰)      (需要意義)
```

### 決策樹使用說明

**第一道問題：能力評估**
- 觀察：學生是否能獨立完成課堂大部分任務？
- 參考指標：
  - 是否需要頻繁提醒或幫助？
  - 是否理解課堂核心概念？
  - 作業的完成度和正確率如何？
  - 是否有先備知識的缺口？

**第二道問題：動機評估**
- 觀察：學生是否主動投入或樂於挑戰？
- 參考指標：
  - 課堂參與度如何？
  - 面對困難時的反應？
  - 是否主動提出問題或想法？
  - 對新概念的好奇心程度？

## 四、觀察行為問卷

使用以下問題觀察和記錄學生在特定科目的表現。這些都是**可觀察的具體行為**，不涉及抽象判斷。

### 能力相關問題

**問題1：先備知識**
- 這位學生是否掌握該課題的基本前置知識？
  - [ ] 是（多數情況掌握）
  - [ ] 部分（有些概念不清楚）
  - [ ] 否（缺乏基本基礎）

**問題2：獨立完成度**
- 這位學生是否能獨立完成課堂作業，不需頻繁協助？
  - [ ] 是
  - [ ] 部分（需要偶爾提醒或澄清）
  - [ ] 否（需要頻繁幫助）

**問題3：理解與應用**
- 這位學生是否能理解核心概念並在新情況中應用？
  - [ ] 是（能應用到新問題）
  - [ ] 部分（理解但應用有困難）
  - [ ] 否（難以理解或應用）

**問題4：作業品質**
- 這位學生完成的作業品質如何？（準確度、完整度、深度）
  - [ ] 高品質（幾乎無誤，完整深入）
  - [ ] 中等（基本完成，偶有錯誤）
  - [ ] 低品質（常有錯誤，不完整）

### 動機相關問題

**問題5：課堂參與度**
- 這位學生在課堂中是否主動參與？（舉手、提出想法、專注聆聽）
  - [ ] 高度參與
  - [ ] 中度參與（有時主動，有時被動）
  - [ ] 低度參與

**問題6：面對困難的態度**
- 當遇到困難題目時，這位學生的反應如何？
  - [ ] 願意嘗試（「讓我再試一次」）
  - [ ] 猶豫不決（需要鼓勵才嘗試）
  - [ ] 放棄（「我不會」「太難了」）

**問題7：主動性**
- 這位學生是否主動提出問題、想法或想要學更多？
  - [ ] 常常（展現好奇心，追問「為什麼」）
  - [ ] 有時（需要提示才提問）
  - [ ] 很少（被動學習）

**問題8：任務完成率**
- 這位學生是否主動完成所有指派任務？
  - [ ] 是（總是完成，有時超越預期）
  - [ ] 通常（完成度在80%以上）
  - [ ] 不穩定（有些任務完成，有些沒有）
  - [ ] 否（常常未完成）

## 五、分類計分指南

根據上述問卷的答案，使用以下標準確定學生類型：

### 能力評分

**高能力指標**（至少符合3項以上）
- ✓ 問題1：多數情況掌握
- ✓ 問題2：是（能獨立完成）
- ✓ 問題3：能應用到新問題
- ✓ 問題4：高品質

**低能力指標**（至少符合3項以上）
- ✓ 問題1：缺乏基本基礎 或 部分掌握
- ✓ 問題2：需要頻繁幫助
- ✓ 問題3：難以理解或應用
- ✓ 問題4：低品質或不完整

### 動機評分

**高動機指標**（至少符合3項以上）
- ✓ 問題5：高度參與
- ✓ 問題6：願意嘗試
- ✓ 問題7：常常提問
- ✓ 問題8：是/主動完成

**低動機指標**（至少符合3項以上）
- ✓ 問題5：低度參與
- ✓ 問題6：放棄或需要鼓勵
- ✓ 問題7：很少提問
- ✓ 問題8：不穩定或常未完成

### 最終分類

| 能力 | 高動機 | 低動機 |
|------|--------|--------|
| 高能力 | **A型** | **C型** |
| 低能力 | **B型** | **D型** |

## 六、學習優勢檢核表

學生可能具有多種學習優勢。觀察以下可觀察的具體行為，幫助您了解每位學生最有效的學習方式。

### 語言型學習者
**特徵**：透過文字、口語討論、閱讀發展理解

**可觀察行為**
- [ ] 喜歡聽故事和解釋
- [ ] 能夠清楚用言語表達想法
- [ ] 偏好閱讀和寫作作為學習方式
- [ ] 在討論中表現積極
- [ ] 容易記住聽到的資訊
- [ ] 能寫出連貫的段落或故事

**教學調整**
- 提供閱讀材料、有聲書、口頭講述
- 設計討論和辯論活動
- 使用詩歌、故事、比喻說明概念

---

### 圖像型學習者
**特徵**：透過視覺化、圖表、圖像理解和記憶

**可觀察行為**
- [ ] 喜歡看圖表、地圖、圖解
- [ ] 能快速理解視覺模式
- [ ] 在筆記中使用圖畫和箭頭
- [ ] 有良好的空間感和方向感
- [ ] 能想像事物的樣子
- [ ] 喜歡看影片和動畫說明

**教學調整**
- 使用圖表、心智圖、視覺組織者
- 提供彩色筆記和視覺化示例
- 整合藝術和視覺化表達

---

### 動作型學習者
**特徵**：透過身體移動、實作、觸覺體驗學習

**可觀察行為**
- [ ] 在課堂上不易靜坐，需要活動
- [ ] 透過動手做才能真正理解
- [ ] 喜歡實驗和親身嘗試
- [ ] 能耐心進行細緻手工
- [ ] 在操作中學得最快
- [ ] 喜歡走動和變換位置

**教學調整**
- 設計動手實驗、操作活動
- 納入身體動作輔助記憶（例如：用肢體表演故事）
- 提供各種材料進行實作
- 允許在學習時走動和改變姿勢

---

### 藝術型學習者
**特徵**：透過音樂、視覺藝術、戲劇、詩歌表達和學習

**可觀察行為**
- [ ] 能哼唱歌曲來記憶資訊
- [ ] 喜歡透過繪畫或塑塑表達
- [ ] 熱愛表演、角色扮演、戲劇
- [ ] 能感受韻律和節奏
- [ ] 喜歡用創意方式展示學習成果
- [ ] 在藝術創作中感到投入和快樂

**教學調整**
- 融入音樂、歌唱、節奏輔助記憶
- 提供藝術表達和創意展示機會
- 使用戲劇角色扮演探索概念
- 鼓勵以藝術形式呈現學習成果

## 七、實例演練

### 學生案例：小明（五年級，自然科學課）

#### 背景資訊
- 課題：植物的生長與繁殖
- 觀察期間：最近四週的課程

#### 問卷填寫結果

**能力相關**
- 問題1：先備知識 → 部分（上學年學過基本概念但已遺忘）
- 問題2：獨立完成度 → 部分（需要偶爾提醒或澄清）
- 問題3：理解與應用 → 部分（理解但應用有困難）
- 問題4：作業品質 → 中等（基本完成，偶有錯誤）

**動機相關**
- 問題5：課堂參與度 → 中度（有時主動，有時被動）
- 問題6：面對困難的態度 → 願意嘗試（「讓我再試一次」）
- 問題7：主動性 → 有時（需要提示才提問）
- 問題8：任務完成率 → 通常（完成度在80%以上）

#### 分類過程

1. **能力評分**：符合3項低能力指標（問題1、2、3、4都指向中等/低）→ **低能力**
2. **動機評分**：符合2項高動機指標（問題6、8）→ **中等動機（偏高）**

#### 初步判定：**B型（低能力、高動機）**

#### 學習優勢檢核

- [ ] 語言型：有時喜歡聽解釋，但筆記有時不清楚
- [x] 圖像型：**強**——喜歡看植物圖片，用圖畫記錄觀察筆記
- [x] 動作型：**強**——在實驗課表現最好，動手操作時專注力提高
- [ ] 藝術型：中等——對繪畫有興趣但不是主要優勢

#### 最終分類結論

**小明 = B型 + 圖像優勢 + 動作優勢**

#### 教學建議

1. **鷹架支持**
   - 提供已簡化的概念圖和視覺參考
   - 在課堂講授時多使用圖表和示意圖
   - 提供複習檢查清單（「植物生長需要：水、光、溫度...」）

2. **小步驟成功**
   - 將「植物繁殖」分解為更小的單位：根、莖、葉、花、種子
   - 每堂課聚焦一個部分，確保完全掌握
   - 頻繁提供進度確認

3. **活用優勢**
   - 讓小明用**圖畫和圖表**記錄觀察（而不是全文字）
   - 多安排**動手實驗**：種子發芽、根的生長觀察
   - 用實物和模型輔助說明

4. **建立自信**
   - 強調進步：「上週你不懂種子，現在你能解釋發芽過程了」
   - 在他成功時給予具體的正面回饋
   - 邀請他在班級分享他的圖畫觀察紀錄

5. **動機提升**
   - 小明已展現願意嘗試的態度，保持鼓勵
   - 讓他看到學習的成果和進度

#### 複習週期

下次評估：6週後（下一個教學單元開始）
- 能力是否有進步？
- 動機是否保持？
- 學習優勢是否得到有效發揮？

## 八、從Waldorf教育觀點的補充思考

在Waldorf教育中，我們不僅看成績或能力，而是**理解整個孩子**：

### 關注整體發展

- **認知面**：思考能力、理解深度
- **情感面**：課題對他有什麼意義？他是否感到被看見和理解？
- **意志面**：他是否有行動的意願？內在動力來自哪裡？

### 動機的深層原因

C型（有能力但動機低）學生可能：
- 找不到學習與自己生活的連結
- 感到被高估或被忽視
- 有其他更深層的關注或困擾

D型（低能力低動機）學生可能：
- 經歷過學習挫折，建立了無助感
- 在課堂中感到不安全或不被接納
- 有未被發現的學習困難或其他需求

### 教師的任務

分類不是為了貼標籤，而是為了：
1. **更深入地認識每位學生**
2. **找到他們獨特的潛能和優勢**
3. **創造每個人都能成功和茁壯的環境**

記住：沒有「差的學生」，只有被**理解得不夠深** 的學生。

## 九、使用提示與注意事項

### 何時進行分類

- **年年度開始**：了解新班級
- **新教學單元開始**：同一學生在不同科目可能分類不同
- **定期檢視**：每月或每個月底更新，追蹤進度
- **發現變化時**：當學生的參與或能力顯著改變

### 常見誤區

❌ **誤**：將D型學生視為「問題學生」或「放棄對象」
✓ **正**：D型學生是在說「請幫助我建立安全感」，他們最需要耐心和愛

❌ **誤**：假設A型學生自己會學，就完全放手不管
✓ **正**：A型學生需要被看見和挑戰，需要充分的延伸和深化機會

❌ **誤**：認為學習優勢是固定的
✓ **正**：在適當的教學設計下，學生可以發展多種學習方式

❌ **誤**：用分類預測學生的「天花板」
✓ **正**：分類是當下的快照，不是永久的標籤

### 與其他教育工具的整合

此分類系統可與以下工具搭配使用：
- **多元智能理論**：除了4種學習優勢，可進一步細化
- **學習風格問卷**：了解更多細節偏好
- **標準化測驗結果**：客觀的診斷數據
- **家長對話**：家庭背景和期望
- **學生自我反思**：他們對自己的認知

## 十、快速參考表

### 快速判斷流程圖（簡化版）

| 觀察 | A型 | B型 | C型 | D型 |
|------|-----|-----|-----|-----|
| **會不會** | ✓ 會 | ✗ 不會 | ✓ 會 | ✗ 不會 |
| **想不想** | ✓ 想 | ✓ 想 | ✗ 不想 | ✗ 不想 |
| **最需要** | 挑戰 | 支持 | 意義 | 安全 |
| **優先策略** | 延伸 | 鷹架 | 連結 | 陪伴 |
| **課堂角色** | 領導者 | 學習者 | 潛力者 | 陪伴者 |

### 檢核清單：我準備好分類了嗎？

在進行分類前，確認：

- [ ] 我已在課堂觀察這位學生至少2-3週
- [ ] 我已看過他在多種活動/任務中的表現（不只是傳統作業）
- [ ] 我理解他在這個特定科目/課題的表現（他在其他科目可能不同）
- [ ] 我沒有受到第一印象或刻板印象的影響
- [ ] 我已考慮到個人或家庭因素可能對表現的影響
- [ ] 我準備好用這個分類**幫助學生，而非限制他們**

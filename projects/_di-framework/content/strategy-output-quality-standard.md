# 策略分析產出品質標準（AI 必須遵守）

> **本文件為所有 Block 產出策略分析時的強制品質指令。**
> AI 在 Phase 1 策略層產出任何差異化建議時，必須達到本文件定義的標準。
> 品質範例參見：`projects/_di-framework/reference/strategy-analysis-quality-example.md`

---

## 一、ABCD 差異化策略建議的寫作標準

### 結構：先說「人」，再說「策略」

每一類學生（A / B / C / D）的策略建議，必須包含兩個部分，**順序不可顛倒**：

```
第一段：心理素描（這類學生的內在狀態是什麼）
    ↓
第二段起：具體教學策略（老師在課堂中可以做什麼）
```

#### 心理素描的要求

- 用第二人稱（「這群學生」「他們」）與教師對話，而非用第三人稱做分類報告
- 描述學生的**內在感受與需求**，不只是外在行為
- 每段心理素描的結尾，必須暗示策略的核心方向
- 每類學生的標題，必須包含一句**策略精神**的副標題，例如：
  - A 類：「給他們值得掙扎的問題」
  - B 類：「讓熱情先走，能力會跟上」
  - C 類：「連結到他們在乎的事情」
  - D 類：「先讓他們覺得安全，安全感是學習的前提」

#### 具體教學策略的要求

- 每一條策略必須**描述課堂中實際會發生的畫面**，老師讀完後能直接想像自己在做
- 禁止使用無法操作的抽象標籤，例如：
  - 「提供鷹架支持」（鷹架是什麼？長什麼樣子？）
  - 「延伸挑戰」（延伸到哪裡？挑戰什麼？）
  - 「連結個人興趣」（怎麼連結？在課堂的哪一刻？）
- 每一條策略至少包含一個**具體操作指引**（材料、時機、動作、話語）
- 如果策略涉及任務設計，必須說明**任務的具體內容**，而不只是任務的分類
- 策略建議的數量：每類學生至少 3 條，可以多但不可少

### 禁止出現的寫法

| 禁止 | 應該改為 |
|------|---------|
| 「提供關鍵詞彙表」 | 「課前發一張 A5 大小的單字地圖，將本課出現的 10 個核心詞彙用詞族方式（前綴、詞根、後綴）呈現，讓學生在閱讀時可以隨時對照」 |
| 「引導同儕討論」 | 「在小組中讓他們擔任『首席讀者』，用自己的話為組員敘述指定段落的故事——不是朗讀，而是說故事」 |
| 「降低焦慮」 | 「給他們極度具體的任務：在文本中找出 5 個認得的英文單字，圈起來，告訴旁邊的人這些字是什麼意思。成功完成 = 一個小小的勝利」 |
| 「提升意義感」 | 「不要問課本題——改問：『你的生活中，有哪一件事是別人替你決定的，但你其實想自己選？』讓他們先從自身經驗出發，再回頭看文本」 |

## 二、學習優勢入口建議的寫作標準

### 結構要求

學習優勢的建議不是列一張「活動類型清單」，而是描述**這個優勢在這個單元中可以長什麼樣子**。

- 每一種優勢（語言型 / 圖像型 / 動作型 / 藝術型）都必須給出**至少一個完整的活動描述**
- 活動描述必須跟**本單元的具體內容**結合，而非泛用型建議
- 活動描述必須包含：做什麼、怎麼做、為什麼這樣做能夠觸及這類學生

### 禁止出現的寫法

| 禁止 | 應該改為 |
|------|---------|
| 「心智圖」 | 說明這張心智圖的中心要放什麼、分支怎麼展開、學生用什麼媒材畫、花多少時間 |
| 「角色扮演」 | 描述扮演的角色是誰、場景是什麼、規則是什麼、之後怎麼收尾回到課程 |
| 「朗讀」 | 說明朗讀的具體段落、如何安排聆聽的氛圍、朗讀之後要做什麼 |

## 三、整體語氣與風格

| 面向 | 標準 |
|------|------|
| **語言** | 繁體中文，用教師之間的專業對話語氣 |
| **溫度** | 每一段建議背後都能感受到「你理解這類學生」 |
| **畫面感** | 老師讀完後，腦海中能浮現課堂上的具體場景 |
| **華德福底蘊** | 策略建議自然融入華德福教育對人的理解，但不刻意引用術語 |
| **可操作性** | 老師可以直接拿走改造，明天就能用 |

## 四、品質自檢清單（AI 產出後必須核對）

產出策略分析表後，AI 必須以此清單自我核對，未通過的項目必須修正後再呈現給教師：

- [ ] 每類學生（A/B/C/D）是否都以心理素描開頭？
- [ ] 每類學生的標題是否包含策略精神副標題？
- [ ] 每條策略是否描述了課堂中實際會發生的畫面？
- [ ] 是否消除了所有抽象標籤（鷹架支持、延伸挑戰、連結興趣等）？
- [ ] 每條策略是否包含至少一個具體操作指引？
- [ ] 學習優勢建議是否跟本單元的具體內容結合？
- [ ] 整體語氣是否像教師之間的專業對話，而非 AI 式清單？
- [ ] 老師讀完後能否直接挑選一條策略帶進明天的教室？

# 策略分析品質範例 — The Giver 單元（9C 班）

## ABCD 差異化策略建議

### A 類學生（高能力 × 高動機）——給他們值得掙扎的問題

這群學生已經準備好獨立思考，他們渴望被認真對待。與其給他們「更多」的任務，不如給他們「更深」的問題，讓他們感受到自己的思考被尊重。

- 讓他們擔任某一章節的「首席讀者」：課前獨立閱讀指定段落，在課堂上用自己的話為全班敘述這段故事——不是朗讀，而是「說故事」。這本身就是一種語言的變形與內化。
- 提出沒有標準答案的倫理問題，例如：「Jonas 的社區消除了痛苦，也消除了選擇。如果你是立法者，你願意用什麼交換安全感？」讓他們寫一段 150 字的立場宣言，在第四節課進行小型辯論。
- 邀請他們為 B 或 D 類同學設計一張「閱讀地圖」——這不是降級任務，而是讓他們用教學者的視角重新消化文本，真正理解什麼叫做「把複雜的東西說清楚」。

---

### B 類學生（低能力 × 高動機）——讓熱情先走，能力會跟上

這些學生最怕的是「公開出醜」。他們想參與，但害怕自己的英文不夠好會被笑。你的工作是在他們還沒開口之前，就讓他們覺得自己有東西可以說。

- 課前提供簡化版的段落摘要（中英對照，但不是逐句翻譯，而是用簡單英文重述關鍵場景），讓他們帶著理解進教室，而不是帶著焦慮。
- 在小組討論中，給他們「指定角色」而非「自由發言」：例如「你負責記錄組員提到的三個關鍵詞」——這讓他們有明確的事情可做，同時自然地沉浸在英文討論中。
- 閱讀後的回應任務，允許他們用「圖文混合」的方式表達：畫一幅 Jonas 第一次看見顏色的場景，旁邊用 3-5 句英文描述畫面。降低文字門檻，但不降低思考深度。

---

### C 類學生（高能力 × 低動機）——連結到他們在乎的事情

他們的問題不是「不會」，而是「不在乎」。你需要讓素材燒到他們個人的某條神經，讓他們突然覺得「這跟我有關」。

- 不要問「Jonas 的社區有什麼規定」這種課本題——改問：「你的生活中，有哪一件事是別人替你決定的，但你其實想自己選？」讓他們先從自身經驗出發，再回頭看文本，發現 Jonas 的處境與自己的某個片刻重疊。
- 讓他們選擇自己的輸出方式：可以寫一封 Jonas 的日記、錄一段 60 秒的 podcast 評論、或設計一張這個「完美社區」的宣傳海報並在背面寫下「海報沒有告訴你的事」。重點不是格式，而是讓他們感覺有主權。
- 如果他們擅長某件事（畫畫、音樂、辯論），刻意在課堂中為他們開一個展現的窗口——被看見，本身就是動機的來源。

---

### D 類學生（低能力 × 低動機）——先讓他們覺得安全，安全感是學習的前提

不要急著讓他們「跟上」。他們需要的是一個不會被評判的小角落，讓他們慢慢靠近英文這個對他們來說很遙遠的東西。

- 第一節課給他們極度具體的任務：在文本中找出 5 個他們認得的英文單字，圈起來，告訴旁邊的人這些字是什麼意思。成功完成 = 一個小小的勝利。
- 配對制度：讓他們跟一位 B 類學生坐在一起（不是 A 類——程度差距太大會加深挫折感），一起完成簡化版的任務。
- 閱讀回應可以用「選擇題 + 一句話回答」的形式，不強迫長篇書寫。但那「一句話」要求他們寫出自己的想法，而不是從文本抄一句——即使只是「I think Jonas is brave」也好。

## 學習優勢入口建議

| 優勢類型 | 在 The Giver 單元中可以長什麼樣子 |
|---------|-------------------------------|
| **語言型** | 朗讀 Jonas 接收記憶的段落時，讓全班閉眼聆聽，之後每人寫下「你聽到時腦海中浮現的第一個畫面」。用聲音喚醒語言的體感。 |
| **圖像型** | 用黑白與彩色兩種媒材：前兩節課只用灰階素描記錄 Jonas 的世界，第三節課 Jonas 開始看見顏色時，突然加入色彩——讓視覺衝擊對應文本的意識轉變。 |
| **動作型** | 「社區規則體驗」：全班站起來，由一位學生擔任「長老」，宣布三條規定（不能選座位、不能決定午餐、不能自己選朋友），其他人必須照做 3 分鐘。結束後討論：「剛才什麼感覺？」用身體經驗打開文本的入口。 |
| **藝術型** | 為 The Giver 設計一首「社區之歌」——歌詞必須用英文，旋律自創或改編。這首歌要聽起來很美好、很安全，但歌詞細看之後讓人不寒而慄。用藝術的形式體驗「表面與真相」的反差。 |

## 此範例的品質檢核點

| 面向 | 標準 |
|------|------|
| 語氣 | 教師可以對話的語言，不是分類標籤 |
| 具體度 | 描述課堂中實際會發生的畫面，而非抽象指令 |
| 教學哲學 | 每一類學生的建議都回扣華德福對人的理解 |
| 可操作性 | 老師看完後可以直接挑選、改造、帶進明天的教室 |
| 人文溫度 | 每段建議的開頭，先說明「這類學生的內在狀態」，再展開策略 |

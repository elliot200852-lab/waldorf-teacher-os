# 8A 英文課產出目錄（科目視角）

本資料夾與 `projects/class-8a/content/english/` **內容完全同步**。
每次產出或更新時，兩個路徑必須同時寫入相同版本的檔案。

## 命名規則

```
{類型}-v{版本號}-{YYYYMMDD}.md
```

## 同步路徑

| 路徑 | 說明 |
|------|------|
| `projects/class-8a/content/english/` | 班級內容視角 |
| `projects/class-8a/english/content/` | 英文科目視角（本路徑） |

> 老師自行決定哪個版本要列印給學生，系統不代為決定。

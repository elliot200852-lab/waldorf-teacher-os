# **九年級下學期英文差異化教學與專案式學習：架構、策略與實踐深度研究報告**

## **緒論：九年級下學期英語教學的轉型與素養定位**

在台灣國民中學教育的最後一個學期，九年級學生面臨的不僅是國中教育會考的壓力，更是一個從基礎教育邁向專業分流的重要轉折點。108 課綱的推行，將英語學習的核心從單純的語言知識記憶轉向具備生活情境應用的「素養」能力，這要求學生不再只是「看得懂」，更要「想得通」並能「用出來」1。針對即將畢業的學生，如何透過差異化教學（Differentiated Instruction）使其對自身的英文能力（單字、文法、聽說讀寫）進行精確的自我定位，成為教學設計的首要目標。在這種教學情境下，課堂不再是單向的知識灌輸，而是一個模擬真實世界的動態實驗室，透過小組合作學習（Collaborative Learning）與專案式學習（Project-Based Learning, PBL）的交互運用，協助不同成就、動機與優勢的學生在畢業前建立對英語學習的長遠信心 2。

本報告旨在探討如何於每週僅兩節課且嚴格限制數位資源使用的情況下，建構一個高效能的學習生態系統。透過設置兩台固定於教室內的電腦主機作為教學任務管理與進度檢核的中心，教學者能有效實施車站旋轉教學法（Station Rotation），並在不增加學生家庭作業負擔的前提下，達成深度學習的目標 4。下文將從核心能力向度、教學策略架構、數位資源整合、以及分組合作機制等層面進行詳盡分析。

## **英語核心能力的自我定位向度分析**

為了讓學生在畢業前具備清晰的自我定位，教學活動必須對標國際語言標準（如 CEFR）以及國內課程綱要。定位的過程不僅是評量學生的現有程度，更是引導其發展出自主學習與整理的習慣 1。

### **英文單字的深度與廣度定位**

在國中階段，學生被要求掌握約 2000 個基礎單字 1。然而，畢業前的定位不應僅停留在拼字正確性，而應擴展至語境運用能力。高成就學生應致力於掌握單字的多義性、搭配詞（Collocations）與詞性變化，而基本能力尚待加強的學生則應優先鞏固最核心的 1200 個高頻詞彙 1。

| 單字掌握層次 | 核心表現指標 | 自我定位標準 |
| :---- | :---- | :---- |
| **基礎層次** | 能正確拼寫 1200 個常用字，並理解基本字義。 | 具備基礎溝通能力，能處理日常熟悉主題 6。 |
| **進階層次** | 掌握 2000 個字彙，並能識別主題相關或字形相似字的差異。 | 能應對高中程度的加深加廣課程需求 1。 |
| **精熟層次** | 能運用主題關聯性（Thematic Association）進行詞彙歸納。 | 具備邏輯推理能力，能從語境推敲生字字義 1。 |

### **聽說讀寫四項技能的 CEFR A2 定位**

九年級畢業生的目標是達到 CEFR A2（初級）至 B1（中級）的門檻。在聽力方面，學生需能聽出對話的邏輯，而非僅是捕捉孤立的單字 1。在閱讀方面，教學應強調略讀（Skimming）與尋讀（Scanning）技巧，以應對長篇閱讀與圖表整合題型 1。

| 技能向度 | CEFR A2 定位描述 | 課堂評量任務建議 |
| :---- | :---- | :---- |
| **聽力 (Listening)** | 能理解日常生活中最相關的短語，如個人與家庭基本資訊。 | 聽取短篇廣播或說明，並完成資訊擷取表 7。 |
| **口說 (Speaking)** | 能在簡單、日常的任務中進行溝通，交換熟悉事物的資訊。 | 針對熟悉話題進行短暫社交互動或簡單描述 8。 |
| **閱讀 (Reading)** | 能在日常材料（如菜單、廣告）中找到特定且可預見的資訊。 | 閱讀長篇生活文章，並推導文章主旨與細節 1。 |
| **寫作 (Writing)** | 能寫簡單的筆記或信件，描述個人背景、環境與需求。 | 撰寫關於過去經驗、未來計畫或個人觀點的短文 9。 |

### **文法架構的邏輯化定位**

九年級文法的重點在於時態的整合運用（如現在完成式）以及被動語態 1。學生應具備建立「錯題筆記」的能力，記錄錯誤原因並釐清觀念 1。教學內容應側重於文法的「功能性」而非「規則性」，例如在專案報告中如何使用被動語態來描述過程。

## **雙主機環境下的差異化教學架構**

在數位資源有限（僅兩台主機）且限制於教室使用的情況下，車站旋轉教學法（Station Rotation）是實現差異化教學的最優策略。這種模式允許學生在固定的課堂時間內，輪流進入數位站進行自主學習或進度檢核 5。

### **車站旋轉模型的配置與動線設計**

將全班分為三個核心站點，每節課進行一次或兩次旋轉。透過這樣的安排，兩台電腦能發揮最大效益而不造成排隊等待的混亂 12。

| 站點名稱 | 學習模式 | 主要功能與任務 |
| :---- | :---- | :---- |
| **數位自主站 (Online Station)** | 獨立或雙人合作 | 使用電腦主機 A 或 B，進行適性學習平台測試、單字闖關或專案進度錄入 14。 |
| **教師引導站 (Teacher-led Station)** | 小組指導 | 教師針對特定程度學生進行文法補強、寫作指導或口說反饋 12。 |
| **離線協作站 (Offline Station)** | 小組互動 | 進行拼圖法教材研討、專案草案撰寫或同儕抽背單字 12。 |

在硬體管理上，課內電腦主機（稱為「任務分派主機」）可預先載入各組的專案路徑圖，而課外電腦主機（稱為「進度檢核主機」）則讓學生在完成離線任務後，登錄完成情況，實現學習歷程的可視化 5。

### **差異化任務的分派策略**

為了應對不同成就與動機的學生，教師應設計「選擇權看板」（Choice Boards），讓學生在每個站點根據自己的學習偏好選擇任務 16。例如，在數位站，動機較弱的學生可以選擇從趣味性的單字遊戲（如 Blooket 或 Gimkit）切入，而高成就學生則可以直接進行 CEFR 模擬測驗 13。

## **小組合作與拼圖法：建立同儕互助機制**

合作學習的核心在於建立「積極相互依賴」的關係。拼圖法（Jigsaw Method）是達成此目標的經典策略，尤其適用於處理長篇閱讀或複雜的專案資訊 19。

### **拼圖法的實踐步驟**

1. **基本小組 (Home Groups)**：將學生混齡或混成就分組。每個成員被分配一個特定的「專家子題」。  
2. **專家小組 (Expert Groups)**：負責相同子題的學生（來自不同基本小組）聚集在一起，共同研究特定內容。此階段可使用電腦主機查找資料 19。  
3. **教學回饋 (Teaching Phase)**：專家回到原組，負責將所學教給其他成員。這強化了「說」的能力，並讓低成就學生在負責特定區塊時產生責任感 22。

這種機制確保了每位學生在畢業前都能定位自己在團隊中的角色——有些人擅長統整資訊（閱讀優勢），有些人擅長清晰解釋（說話優勢），有些人則擅長管理團隊動能 3。

### **同儕教練 (Peer Coaching) 與互助機制**

在每週至少一次的小組研討中，應落實「同儕力量（Peer Power）」。高成就學生擔任「教練」，協助其他同學進行單字抽背或文法訂正。研究顯示，這種互助模式能有效加速學習，並提升整體的課堂動機 1。為了確保公平，教練的評量應包含「其教導之組員的成長幅度」，以此激勵深度合作而非僅是單向抄寫。

## **專案式學習 (PBL) 與總監制度之實踐**

專案式學習旨在讓學生透過解決真實世界的難題來掌握知識 2。在畢業前的最後一個學期，專案應具備整合性質，並賦予學生高度的自主權。

### **雙總監制度 (Dual Director System) 的設計**

每個專案由兩位同學擔任總監，分別負責不同的管理面向。這種分工能分散壓力，並讓具備不同才能的學生都有發揮空間 25。

| 總監職稱 | 核心職責 | 關鍵技能需求 |
| :---- | :---- | :---- |
| **行政總監 (Admin Director)** | 負責進度控管、時間管理、材料收集與電腦主機進度更新 25。 | 組織力、溝通協調能力。 |
| **品質總監 (Content Director)** | 負責內容正確性（如英文文法與單字使用）、學術誠信監督與最終呈現審核。 | 學術成就較高或具備細緻度。 |

這種設置能有效減少對教師的過度依賴，提升團隊的自治能力。教師的角色則轉變為「教練」或「諮詢顧問」，在旋轉教學的教師站中提供針對性的引導 25。

### **九年級下學期建議教學專案內容**

專案的主題應與學生的生活經驗及未來規劃緊密連結。根據 108 課綱精神與國際教育目標，以下為兩個具代表性的專案建議 28：

1. **我的英語履歷與未來力 (Personal Portfolio & Future Skills)**  
   * 內容：結合生涯規劃，學生需分析自己的性格特質與未來職涯興趣。  
   * 語言目標：運用描述個人特質的形容詞（單字）、過去完成式與現在完成式（文法）來描述過往成就與未來願景 30。  
   * 自我定位：學生能定位自己是否具備基礎的國際競爭力語言基礎。  
2. **地球特派員：全球議題專案 (Earth Commissioner: Global Issues)**  
   * 內容：選擇一個 SDGs 議題（如氣候變遷、性別平等），研究其現況並提出英文解決方案 29。  
   * 語言目標：進行長篇資料閱讀（讀）、撰寫提案報告（寫）、製作宣傳短片或進行口頭發表（聽與說） 31。  
   * 合作機制：透過拼圖法，將「現狀調查」、「成因分析」、「解決方案」分派給不同成員，最後由雙總監彙整。

## **數位資源管理與進度檢視策略**

在教室內設置兩台主機的特殊環境下，管理策略必須極度簡化且具備高度指向性。

### **任務分派與進度視覺化**

電腦主機 A 作為「教學任務分派站」，其桌面或入口網頁應包含當週的「學習路徑圖」。每個小組在開始活動前，由行政總監至此確認今日目標。電腦主機 B 作為「進度與評量站」，學生在此完成單字小測驗或上傳小組討論筆記 5。

為了不增加家庭負擔，所有數位任務都應設計為可在 15 分鐘（一個車站旋轉週期）內完成的微型任務。對於進度較慢的小組，教師可透過主機 B 上的記錄即時發現問題，並在旋轉到教師導引站時進行介入 12。

### **自我評量與反射性學習**

在畢業前，學生需要建立一套屬於自己的「技能清單（Can-Do Statements）」。這套清單應數位化存放在電腦主機中，學生每週進行一次勾選 7。

| 定位維度 | 檢核指標範例 |
| :---- | :---- |
| **單字力** | 我能不看字典理解 2000 個基礎單字中的 80% 1。 |
| **溝通力** | 我能在專案研討中，使用簡單的英文句子提出反對意見或建議 8。 |
| **理解力** | 我能從長篇文本或圖表中，正確推論出作者的態度 1。 |
| **文法力** | 我能在撰寫報告時，正確使用時態來區分過去與現在的事件 33。 |

## **結論與教學行動建議**

九年級下學期的英語學習不應只是枯燥的考題練習，而應是一個賦予學生「自我定位權」的過程。透過差異化教學與車站旋轉模型，我們能在有限的數位資源下，確保每位學生都能獲得最適合其程度的指導 11。拼圖法與雙總監制的 PBL 模式，不僅解決了動機差異的問題，更培養了未來職場必備的協作能力 3。

最後，建議教學者將「不增加回家負擔」視為一種創新的動力，利用課堂內的高度集中與數位檢核，讓學生在放學離開教室前，就已經完成所有的學習循環。當學生能夠在畢業之際，看著自己在電腦主機上積累的學習歷程紀錄，並自信地說出「我能用英文完成一個專案」時，這場教學規劃便達成了真正的教育意義 2。

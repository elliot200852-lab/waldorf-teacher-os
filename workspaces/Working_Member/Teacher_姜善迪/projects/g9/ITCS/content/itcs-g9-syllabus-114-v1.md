# 114 學年 ITCS 九年級教學大綱 (Syllabus)

- **教師**：姜善迪
- **班級**：九年級 (g9)
- **科目**：資訊科技與電腦科學 (ITCS)
- **核心定位**：WHAT is this? — 數位世界的構成、邏輯與協作工具
- **節奏**：每週 1 次 90 分鐘，共 15 週

---

## 課程目標
1. **理解硬體因果**：認識電腦硬體的運作邏輯與科學史觀。
2. **具身資訊科學**：透過 CS Unplugged 活動，在不插電的情況下體驗演算法與資料表示法。
3. **精進數位協作**：熟練 Google Workspace 操作，將其作為學習與團隊協作的加速器。

---

## 15 週教學進度規劃

| 週次 | 主題 | 內容要點 |
| :--- | :--- | :--- |
| **第一階段：電腦的身體與靈魂 (Hardware & Logic)** | | |
| W1 | 課程開場與數位世界概論 | 什麼是資訊科技？九年級 ITCS 的核心探問 |
| W2 | 電腦硬體基礎 (1) | 拆解與認識：CPU、記憶體、儲存裝置的因果邏輯 |
| W3 | 電腦硬體基礎 (2) | 馮紐曼架構：資料如何在硬體間流動？ |
| W4 | 電腦科學史 (1) | 從算盤到圖靈機：人類計算意志的展現 |
| W5 | 電腦科學史 (2) | 數位革命與現代電腦的誕生 |
| **第二階段：不插電的資訊科學 (CS Unplugged)** | | |
| W6 | 二進位數 (Binary Numbers) | 點卡活動：電腦如何用 0 與 1 表示龐大資訊 |
| W7 | 影像表示 (Image Representation) | 數字畫：認識像素 (Pixel) 與壓縮原理 |
| W8 | 偵錯與修正 (Error Detection) | 魔術翻牌：資料傳輸中的錯誤偵測邏輯 |
| W9 | 排序演算法 (Sorting Algorithms) | 天秤與排序網路：電腦如何處理大量資料的順序 |
| W10 | 搜尋演算法 (Searching Algorithms) | 戰艦遊戲：線性搜尋與二分搜尋的效率差異 |
| **第三階段：數位協作工具 (Google Workspace)** | | |
| W11 | Gmail 與行事曆管理 | 專業溝通禮儀、時間規劃與提醒機制 |
| W12 | 雲端硬碟 (Drive) 邏輯 | 檔案組織、共用雲端硬碟與協作權限管理 |
| W13 | 文件 (Docs) 與簡報 (Slides) | 內容產出、格式規範與多人即時編輯協作 |
| W14 | 試算表 (Sheets) 初探 | 資料整理、基本函數應用與邏輯運算 |
| W15 | 成果整合與學期回顧 | 運用 Workspace 工具完成小組專案展示、自我超越反思 |

## 差異化教學 (DI) 預留空間
- **A 類 (高能力高動機)**：在 CS Unplugged 中擔任活動引導者，或在 Workspace 階段進行進階函數/外掛探索。
- **B 類 (低能力高動機)**：提供具體的步驟鷹架，確保每堂課都有具體的產出成就感。
- **C 類 (高能力低動機)**：連結個人專案興趣，強調工具如何幫助其「變懶 (效率提升)」。
- **D 類 (低能力低動機)**：降低技術門檻，從直觀的視覺化活動 (如點卡、數字畫) 切入建立自信。

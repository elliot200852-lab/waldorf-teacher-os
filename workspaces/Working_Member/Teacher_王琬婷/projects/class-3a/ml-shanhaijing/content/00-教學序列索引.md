# 山海經神話主課程・教學序列索引

> 三年級春季語文主課程（春季第 4-6 週）
> 核心提問：人類如何從神的護佑走向自立？

---

## 教學弧線

```
第一階段：天地與生命的誕生（神創造世界）
  01 盤古開天 → 02 女媧造人 → 03 女媧補天

第二階段：宇宙秩序的守護者（神獸維持節律）
  04 燭龍 → 05 龍的誕生 → 06 應龍畫地

第三階段：英雄與意志的試煉（人開始行動）
  07 后羿射日 → 08 精衛填海 → 09 夸父追日

第四階段：人類自立——從對抗到協作（人建立文明）
  10 鯀的悲歌 → 11 禹的行走 → 12 萬物齊心 → 13 相柳與豎亥
```

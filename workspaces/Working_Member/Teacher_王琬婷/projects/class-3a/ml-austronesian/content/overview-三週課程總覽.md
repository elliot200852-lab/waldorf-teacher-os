# 三年級南島神話主課程——三週總覽

## 課程定位

本課程以南島語系民族的創世神話為素材，設計三年級主課程區塊（3 週 / 15 天 / 每天 120 分鐘）。

### 課程目標

1. **語文能力**：透過聆聽、口語重述、工作本書寫，發展三年級的聽說讀寫基礎
2. **土地認識**：讓孩子認識自己腳下這塊土地——臺灣作為南島語系民族海洋文化原鄉的深遠意涵
3. **創世紀精神**：回應三年級「九歲轉變」（Rubicon）的發展需求——孩子開始感受到自我與世界的分離，創世故事回答「世界從何而來？」「我在這個世界中的位置是什麼？」
4. **三大支柱**：融入華德福三年級的三大課程支柱——創世故事、農耕、建造

### 教育學基礎

三年級孩子（8-9 歲）正經歷「九歲轉變」——第一次深刻感受到自我與世界的分離。他們開始質疑：世界是怎麼來的？大人真的什麼都知道嗎？這是一個充滿不安但也蘊含成長力量的時期。

創世故事正是華德福教育對這個發展階段的回應：
- **宇宙有秩序**：天地的分離不是混亂，而是秩序的開始
- **分離中有連結**：天空之神的眼淚成為雨，大地女神的淚水匯成河——分離不代表失去
- **人能掌握世界**：從取得種子、馴養動物、獲得火種，到射下太陽——人類有能力面對挑戰
- **毀滅之後有重建**：洪水毀滅一切，但兄妹存活、石磨碎片化為萬物——世界可以重新開始
- **走向未知的勇氣**：騎鯨人帶著種子和構樹出海——面對未知不是恐懼，而是冒險

## 三週主題弧線

```
第一週：天地與生命（創造與分離）
  從混沌到秩序——世界如何被創造出來？

第二週：人與自然的關係（文明的工具）
  種子、動物、火、光——人類如何學會生活？

第三週：災難與新生（毀滅與重建）
  洪水毀滅一切——人類如何重建並走向海洋？
```

這個弧線同時呼應了人類文明的發展歷程，以及三年級孩子的內在發展：從「世界是安全的」到「世界有挑戰」再到「我有能力面對挑戰」。

## 每日主課程節奏（120 分鐘）

```
┌─────────────────────────────────────────┐
│  Part 1  晨圈活動           20 分鐘     │
│  ─ 歌唱 + 肢體動作（意志活動）          │
│  ─ 念謠（甦醒活動）                     │
├─────────────────────────────────────────┤
│  Part 2  前日故事回顧與討論   30 分鐘     │
│  ─ 口語重述 / 教師提問 / 戲劇扮演       │
│  ─ 華德福「睡眠消化」原則               │
├─────────────────────────────────────────┤
│  Part 3  工作本書寫           30 分鐘     │
│  ─ 黑板畫 + 引導文（約 30 字）          │
│  ─ 自由書寫接續                         │
├─────────────────────────────────────────┤
│  Part 4  新故事講述           20 分鐘     │
│  ─ 教師說書稿                           │
│  ─ 故事結束即下課，不分析不討論         │
│  ─ 讓故事「帶著走」，隔夜沉澱          │
└─────────────────────────────────────────┘
  剩餘 20 分鐘：過渡、整理、課間緩衝
```

**節奏設計邏輯**：新故事放在最後，孩子帶著鮮活的故事圖像離開教室，經過一夜睡眠沉澱，隔天早晨在 Part 2 回顧時，理解與感受自然深化。這是「先給予圖像，再提取概念」的具體實踐。

## 書寫引導設計（延遲一天原則）

**核心規則**：書寫內容寫的是「前一天」的故事，不是當天的。黑板畫也跟著延遲一天。
- 新故事在 Part 4 講述後，孩子帶著圖像離開，經一夜沉澱
- 隔天 Part 2 回顧後，Part 3 才書寫——這時理解最深
- D01 無前日故事，Part 3 改為「畫工作本封面」

### 書寫漸進三階段

**第一週（D01-D05）**：畫封面 + 引導抄寫 + 1-2 句接續
- D01：畫工作本封面（不書寫）
- D02-D05：引導文約 30 字 + 自由接續提示

**第二週（D06-D10）**：半自由書寫，加入感官描寫
- D06：統整日綜合引導書寫（「從天地初始到砍斷天梯，這個世界經歷了什麼？」）
- D07-D10：引導文 + 感官描寫延伸

**第三週（D11-D15）**：完整自由書寫
- D11：統整日綜合引導書寫（「人從自然取得了哪些禮物？每個禮物背後都有誰的犧牲？」）
- D12-D15：自由書寫為主，D15 結業創作（「如果你是騎鯨人，你會帶什麼去新天地？」）

### 能力 × 動機矩陣

#### Part 2 回顧討論

| 類型 | 策略 |
|------|------|
| A（高能力高動機）| 擔任重述故事的敘事者、提出延伸問題、比較不同故事 |
| B（低能力高動機）| 教師引導問題搭鷹架，從「誰做了什麼」事實層開始 |
| C（高能力低動機）| 連結個人經驗的開放問題（「如果你是…你會怎麼做？」）|
| D（低能力低動機）| 先用身體演出故事片段（戲劇扮演），再過渡到口語 |

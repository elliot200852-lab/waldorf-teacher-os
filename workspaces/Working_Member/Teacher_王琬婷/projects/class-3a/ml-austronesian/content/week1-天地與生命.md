# 第一週：天地與生命（創造與分離）

> 三年級「九歲轉變」——孩子開始感受到自我與世界的分離。創世故事回答「世界從何而來？」的深層需求。本週五天從混沌走向秩序，從合一走向分離，呼應孩子內在正在經歷的意識變化。

---

## 晨圈素材庫（第一週建立，後兩週持續使用並擴充）

### 歌曲一：〈大海的呼吸〉（自編，全三週使用）

```
大海呼吸，浪來浪去，（雙手如波浪左右擺動）
風吹過山，雲走過溪。（雙臂高舉畫弧，再低垂如溪流）
大地沉沉睡，星星亮晶晶，（蹲低抱膝，再站起雙手張開如星芒）
天和地之間，我們站在這裡。（雙腳穩穩踏地，雙手平舉）
```

**動作要點**：波浪動作要從指尖帶動、流暢不急。「我們站在這裡」時全班同時踏步三下，感受大地的承載。

### 歌曲二：〈Rangi 和 Papa〉（D02 起加入，配合毛利族故事）

```
Rangi Papa, Rangi Papa,（雙手合十置於胸前）
天擁著地，地擁著天。（雙臂交叉擁抱自己）
孩子推呀推呀推，（雙手由胸前緩慢向上推）
光——照進來了！（雙手打開如破曉，全身伸展）
```

**動作要點**：「推呀推呀推」時身體從蹲姿慢慢站起，配合用力感。「光照進來了」時突然展開、跳起。這段動作培養意志力品質。

### 念謠一：〈在那之前〉（全三週使用）

```
在那之前——（全班齊聲，低沉緩慢）
沒有山，沒有海，（每說一個「沒有」踏步一下）
沒有風，沒有雨，（繼續踏步）
沒有白天，沒有黑夜，（踏步漸重）
只有——黑暗。（全班蹲下，雙手遮眼，靜默三秒）
然後——（站起，聲音漸亮）
光來了。（雙手向上張開）
```

### 念謠二：〈半石半木〉（D03 起加入，配合泰雅族故事）

```
高山上，有棵樹，（踏步上行）
半邊石頭半邊木。（右手握拳如石，左手掌開如木）
風吹它，雨打它，（搖晃身體）
雷——劈下來！（拍手一聲，蹲下）
根變獸，枝變蟲，（四肢著地爬行，再扭動手指）
葉變鳥，幹變人！（雙臂拍動如翅，再挺直站立）
```

## D01（週一）：天地初始（毛利族）

### Part 1 晨圈活動（20 分鐘）

**流程**：

1. **集合站圓**（2 分鐘）：全班手牽手站成圓圈，深呼吸三次，教師輕聲說：「我們要回到很久很久以前，世界還沒有開始的時候。」
2. **念謠〈在那之前〉**（5 分鐘）：教師先示範一次完整念謠與動作，再帶全班一起做兩次。強調踏步的沉穩與蹲下時的安靜。
3. **歌曲〈大海的呼吸〉**（8 分鐘）：教師先唱一遍，再逐句教唱。第一天重點放在旋律與波浪動作，不求完美。
4. **意志力活動：推天遊戲**（5 分鐘）：兩人一組面對面，雙手掌心相對，一人扮演「天」一人扮演「地」，輪流用力推對方（安全距離內）。教師引導：「感覺你的力量從腳底升起，穿過身體，到達手掌。」

### Part 2 主題導入（30 分鐘）

> D01 無前日故事，改為主題導入。

**教師開場**（5 分鐘）：

「今天開始，我們要聽一些很古老很古老的故事。這些故事比你的爺爺的爺爺的爺爺還要老。它們來自住在海邊的人、住在山上的人、住在島嶼上的人。他們說著不同的語言，卻都在問同一個問題——」（停頓，看著孩子們）「世界是怎麼開始的？」

**引導討論**（15 分鐘）：

教師提問：「閉上眼睛，想像一下——如果世界還沒有開始，那會是什麼樣子？」

讓孩子自由發言。教師不評判對錯，只是溫暖地接住每個回答。

進一步引導：
- 「沒有太陽的話，會是什麼樣子？」
- 「沒有地的話，你站在哪裡？」
- 「你覺得，第一道光是怎麼來的？」

**DI 差異化**：
- **A 學生**：鼓勵描述完整的想像場景，追問「然後呢？」
- **B 學生**：提供具體選項——「你覺得是很冷，還是很熱？是安靜的，還是吵的？」
- **C 學生**：連結個人經驗——「你有沒有在很黑很黑的地方待過？感覺像什麼？」
- **D 學生**：邀請用身體表現——「你可以用身體讓我們看看『什麼都沒有』長什麼樣子嗎？」

**小活動：黑暗體驗**（10 分鐘）：

拉上教室窗簾，關掉所有燈。全班安靜坐著，感受黑暗一分鐘。教師緩緩說：「這就是世界開始之前的樣子。什麼都沒有，只有黑暗，只有安靜。」然後慢慢點亮一根蠟燭。「然後——光來了。」讓孩子看著燭光，安靜片刻。

### Part 4 新故事講述（20 分鐘）

#### 說書稿：天地初始（毛利族）

很久很久以前，比任何人記得的還要久，世界上什麼都沒有。

沒有山，沒有海，沒有河流，沒有樹木。沒有風吹過，沒有鳥叫聲。連白天和黑夜都沒有——因為世界被黑暗整個包住了，像一條又厚又重的毯子，把所有的光都擋在外面。

在這片黑暗裡面，有兩個巨大的存在。

一個是朗伊努伊（Ranginui），天空之神。他的身體寬廣無邊，從這一頭延伸到那一頭，沒有人知道他的邊界在哪裡。

另一個是帕帕圖阿努庫（Papatuanuku），大地女神。她沉穩厚實，溫暖而柔軟，像一張永遠不會塌下去的大床。

朗伊努伊和帕帕圖阿努庫深深地愛著彼此。他們緊緊地、緊緊地擁抱在一起，從來沒有分開過。一千年過去了，一萬年過去了，他們依然擁抱著，不肯鬆手。

在他們擁抱的縫隙裡，生命悄悄誕生了。

一個孩子，又一個孩子，又一個孩子——他們有了好多好多孩子。可是這些孩子被擠在父親的胸膛和母親的身體之間，黑漆漆的、窄窄的，連翻個身都困難。他們從來沒有見過光，不知道什麼是白天，不知道什麼是顏色。

孩子們漸漸長大了。其中有一個最強壯的，叫做塔內（Tane），他是森林之神。塔內的身體愈長愈大，他覺得愈來愈擠，呼吸愈來愈困難。

塔內對他的兄弟們說：「我們不能永遠住在黑暗裡。我要把父親和母親分開，讓光照進來。」

有一個兄弟叫做塔維里馬提亞（Tawhirimatea），他是風暴之神。他大聲反對：「不行！你不可以把父親和母親分開！他們相愛，他們應該永遠在一起！」

但是塔內已經決定了。

他躺了下來，把背靠在母親——大地女神溫暖的身體上。他彎起雙腿，把腳底頂住父親——天空之神寬廣的胸膛。

然後，他開始推。

他用盡全身的力氣推。他的腿在發抖，他的身體在流汗，他咬緊牙關，一直推，一直推，一直推。

天空之神不肯鬆手，他緊緊地抓著大地女神。大地女神也不肯放手，她的手指深深地抓進天空之神的衣裳裡。

但是塔內沒有放棄。他推了一天，推了一個月，推了一年。

終於——

「嘎嘎嘎嘎——」

一聲巨大的撕裂聲響徹天地。

天空之神被推開了。他被推得好高好高好高，高到看不見邊際。

就在那一瞬間，第一道光照了進來。

那道光是金色的，溫暖的，亮得讓所有的孩子都睜不開眼睛。他們從來沒有見過這樣的東西。光照在他們身上，暖洋洋的。他們第一次看見了彼此的臉，第一次看見了顏色。

這是世界的第一個黎明。

天空之神在很高很高的地方，俯視著大地女神。他好想念她，他的心好痛好痛。他流下了眼淚，一滴、兩滴、好多好多滴——他的眼淚落了下來，變成了雨。

大地女神在下面，抬頭望著天空之神。她也好想念他。她也流下了眼淚，眼淚從她身上流過，匯集成小溪，小溪匯集成河流，河流匯集成大海。

從此以後，每當天空下雨的時候，那是天空之神在想念大地女神。每當大地上的河水流向大海的時候，那是大地女神在回應他的思念。

而那個憤怒的兄弟塔維里馬提亞呢？他追隨父親飛上了天空，從此成為暴風雨之神。每當狂風大作、雷電交加的時候，那就是他在生氣——他到現在還在生氣，因為他的父母再也無法擁抱在一起了。

世界就這樣有了天空和大地，有了雨水和河流，有了陸地和海洋。

但是，世界上還沒有樹，還沒有動物，還沒有花。那些，要等到明天的故事了。

## D02（週二）：生命的出現（泰雅族）

### Part 1 晨圈活動（20 分鐘）

**流程**：

1. **集合站圓 + 深呼吸**（2 分鐘）
2. **念謠〈在那之前〉**（3 分鐘）：複習昨天學的念謠，全班一起做兩次。今天孩子應該已能跟上動作。
3. **歌曲〈大海的呼吸〉**（5 分鐘）：複習昨天教的歌曲。教師唱一遍，再全班齊唱兩遍。
4. **新增歌曲〈Rangi 和 Papa〉**（5 分鐘）：教師先完整唱一遍並做動作，再逐句教唱。今天先學前兩句。
5. **意志力活動：樹的生長**（5 分鐘）：全班蹲在地上縮成一顆種子。教師緩慢數「一、二、三......」，孩子慢慢站起來，雙手如樹枝向上展開。教師說「風來了」，孩子搖擺；「雨來了」，孩子用手指輕敲自己肩膀模擬雨滴；「雷來了！」，全班拍手一聲然後定格。

### Part 2 前日故事回顧與討論（30 分鐘）

**口語重述**（15 分鐘）：

教師提問：「昨天我們聽了一個故事。誰記得，在最最最開始的時候，世界是什麼樣子的？」

**DI 差異化**：
- **A 學生**：邀請擔任主要敘事者，從頭到尾重述故事。教師追問：「塔內推開天地之後，他的兄弟塔維里馬提亞做了什麼？為什麼他這麼生氣？」
- **B 學生**：教師用引導問題搭鷹架——「朗伊努伊是誰？帕帕圖阿努庫是誰？他們之間有什麼？」一步步帶出情節。
- **C 學生**：連結個人經驗——「塔內很用力地推，推了很久很久。你有沒有做過一件事，需要用很大的力氣、堅持很久？」
- **D 學生**：先用身體演——「你可以表演塔內推開天空的樣子嗎？躺下來，用腳頂住天花板，用力推！」演完再口述。

**深度討論**（10 分鐘）：

教師引導核心問題：
- 「天空之神和大地女神被分開了，他們很難過。可是如果他們不分開呢？孩子們會怎麼樣？」
- 「塔維里馬提亞反對分開父母，他錯了嗎？」（開放性問題，不給標準答案）
- 「你覺得，光照進來的那一刻，感覺像什麼？」

**戲劇扮演**（5 分鐘）：

選五個孩子，分別扮演天空之神、大地女神、塔內、塔維里馬提亞、和光。其餘孩子當「黑暗」圍在外圍。全班一起演出分離的片段。

### Part 4 新故事講述（20 分鐘）

#### 說書稿：生命的出現（泰雅族）

天和地分開了。天在上面，高得看不到頂。地在下面，寬得走不到邊。白天有太陽，晚上有月亮，風從東邊吹到西邊，雨從天上落到地上。

可是，地面上光禿禿的。

沒有一棵草，沒有一朵花，沒有一棵樹。只有石頭。大石頭，小石頭，圓石頭，尖石頭。整個世界都是石頭。走到哪裡，踩到的都是石頭。冷冰冰的、硬邦邦的石頭。

風吹過石頭，發出嗚嗚嗚的聲音。雨打在石頭上，發出啪啪啪的聲音。太陽曬著石頭，石頭變得又熱又燙。到了晚上，石頭又變得又冷又硬。

就這樣，風吹了一年又一年，雨打了一年又一年。石頭慢慢地裂開了縫隙，碎成了小石子，小石子磨成了沙，沙又變成了土。有了土，世界就不一樣了。

土是軟軟的，濕濕的，暖暖的。種子可以躲在土裡面。

不知道從哪裡來的第一顆種子，落進了土裡。然後是第二顆，第三顆，好多好多種子。草長出來了，花開出來了，樹也長出來了。光禿禿的世界變成了綠油油的世界。

可是，還沒有動物，也沒有人。

在一座很高很高的山上——那是全世界最高的山——長出了一棵奇怪的樹。

這棵樹跟其他的樹都不一樣。它的左半邊是石頭做的，灰灰白白的，硬邦邦的，敲起來會發出「叩叩叩」的聲音。它的右半邊是木頭做的，棕棕暖暖的，軟軟的，摸起來有溫度。

半邊是石頭，半邊是木頭。

這棵半石半木的大樹，愈長愈高，愈長愈大，比山還要高，比雲還要高。它的根伸得好深好深，伸進大地的最底下。它的枝葉伸得好廣好廣，幾乎遮住了半邊天空。

奇怪的是，常常有聲音從這棵樹裡面傳出來。

有時候聽起來像風在唱歌，嗡嗡嗡嗡。有時候聽起來像心在跳動，咚咚咚咚。有時候聽起來像有什麼東西在裡面翻來翻去，想要出來，卻出不來。

那聲音一天比一天大，一天比一天急。

有一天黃昏，天空突然暗了下來。

烏雲從四面八方湧過來，黑壓壓的，把太陽整個遮住了。風開始吹，愈吹愈猛。雨開始下，愈下愈大。

然後——

一道閃電。

不是普通的閃電。是一道從天頂直直劈下來的、又粗又亮的閃電，帶著「轟隆隆隆——」的巨大雷聲。

那道閃電不偏不倚，正好劈中了那棵半石半木的大樹。

「喀啦——！」

大樹從正中間裂成了兩半。

就在裂開的那一瞬間——

樹根動了。粗壯的樹根扭動著，變成了四條腿、變成了尾巴、變成了毛皮——各種各樣的野獸從樹根裡跑了出來。有鹿，有山豬，有熊，有兔子。牠們踩著新生的腳，搖搖晃晃地走進了山林。

樹枝也動了。細長的樹枝彎啊彎，變成了翅膀、變成了羽毛、變成了尖尖的喙——各種各樣的鳥從樹枝上飛了出來。有老鷹，有藍鵲，有畫眉，有小小的繡眼。牠們拍著翅膀，呼啦啦地飛上了天空。

樹葉呢？樹葉一片一片掉下來，落在地上，扭啊扭，變成了蛇，變成了蜥蜴，變成了各種在地上爬行的生物。牠們滑溜溜地鑽進了草叢和石縫裡。

最後是樹幹。

那最粗壯、最中心的樹幹，慢慢地裂開了。從裡面走出來的，不是獸，不是鳥，不是蟲——是人。

第一個人站了起來。他抬頭看看天，低頭看看地，伸出手摸摸風，張開嘴嘗嘗雨。

然後第二個人走出來，第三個人走出來。男人，女人，他們睜著大大的眼睛，看著這個全新的世界。

他們就是泰雅族的祖先。

從那天起，山裡有了動物的腳印，天上有了鳥的歌聲，草叢裡有了蟲的低語，而人——站在天地之間，開始了生活。

### Part 4 新故事講述（20 分鐘）

#### 說書稿：天界與人間分離（上）——尼弗奴造人

天和地分開了，生命也出現了。山上有動物在走，天上有鳥在飛，河裡有魚在游。人呢？人也有了，但是很少很少，少到走一整天的路都碰不到另一個人。

在天上——是的，那個時候天上是住著神的——住著一位創造女神，她的名字叫尼弗奴。

尼弗奴經常從天上往下看。她看到大地那麼大，山那麼多，河流那麼長，可是人呢？這裡一個，那裡一個，東一個西一個，稀稀落落的，像撒在大桌子上的幾粒米。

尼弗奴搖了搖頭，說：「不行不行，人太少了。這麼大的土地，這麼多的山和河，需要更多的人來照顧才行。」

於是尼弗奴決定親自下到人間。

她從天上走下來——不是走樓梯，那時候天和地之間還有一條路可以走——她一腳踩在一座山上，另一腳踩在另一座山上。

這兩座山，一座叫做特富野，一座叫做達邦。它們都在現在的阿里山。如果你去過阿里山，你可能踩過尼弗奴踩過的那片土地。

尼弗奴站在兩座山頭之間，她的身體好大好大，比山還要高。她往四周看了看，到處都是綠色的森林，但是聽不到人說話的聲音，聽不到孩子笑的聲音。太安靜了。

她看到山坡上長著一棵好大的楓樹。秋天到了，楓樹的葉子有的紅得像火，有的黃得像金子，樹上還結滿了圓圓的果實。

尼弗奴走到楓樹旁邊。她伸出手，握住楓樹粗壯的樹幹，然後——

她搖了搖。

不是輕輕地搖，是用力地搖。

楓樹的枝幹搖晃起來，嘩嘩嘩嘩，葉子開始落下來。紅色的葉子，黃色的葉子，一片一片一片，像下了一場彩色的雨。

那些楓葉飄啊飄，轉啊轉，在風裡跳著舞。它們落到地上的時候，奇妙的事情發生了——每一片葉子碰到地面，就變成了一個女人。紅葉變的、黃葉變的、大葉變的、小葉變的，每一個都不一樣。她們站了起來，看看天，看看地，看看彼此，露出微笑。

圓圓的果實也掉了下來。咚咚咚咚，果實滾在地上，每一顆果實碰到土，就變成了一個男人。他們站了起來，伸伸手腳，深深地吸了一口山裡的空氣。

尼弗奴又搖了搖，又搖了搖。葉子不停地落，果實不停地滾，女人愈來愈多，男人也愈來愈多。

他們開始走動了。有的往北走，有的往南走，有的留在山裡，有的走向平原。他們找到水源就住下來，蓋起了房子，生了孩子，孩子又生了孩子。

一個村落，兩個村落，三個村落......慢慢地，山上山下都有了人的聲音。炊煙升起來了，歌聲傳出來了，故事開始被說出來了。

尼弗奴站在山頂上看著，滿意地點了點頭。

「好了，」她說，「這些人會好好地照顧這片土地的。」

然後她沿著天地之間的那條路，走回了天上。

可是你知道嗎？故事還沒有結束。在南邊的山上，有一戶人家的門口，擺著一個古老的陶壺。那個陶壺裡面，正在發生一件非常非常不可思議的事......

不過，那是後天的故事了。

#### 說書稿：天界與人間分離（下）——陶壺蛇蛋、嬤蓋蓋與天梯

在南方的山上，有一戶人家。

有一天，這家人發現壺裡有了一樣東西。

一顆蛋。

不知道從哪裡來的蛋，白白的，光滑的，靜靜地躺在壺底。

第二天，壺裡又多了一樣東西——一條百步蛇。

百步蛇的身體是深深的棕色，上面有一個一個三角形的花紋，排列得整整齊齊，像有人精心畫上去的。牠的眼睛又圓又亮，安靜地盯著那顆蛋，然後慢慢地、慢慢地把身體盤繞在蛋的周圍。

牠不是要吃蛋。牠是在保護蛋。

這家人嚇了一跳，但是他們沒有趕走百步蛇。因為在魯凱族的古老傳說裡，百步蛇是守護的神靈，是祖先的化身。百步蛇出現在你家，那是天大的祝福。

日子一天一天過去。百步蛇一直守著蛋，不吃不喝，不離開半步。太陽每天從東邊升起來，陽光照進陶壺裡，照在蛋上面，暖洋洋的。

一天，兩天，三天......很多天過去了。

有一天早晨，太陽剛剛升起來，第一道金色的陽光照進了陶壺裡。

「喀」——蛋裂開了一條縫。

「喀喀」——縫愈來愈大。

百步蛇把身體縮到旁邊，安靜地看著。

蛋殼一片一片掉落——從裡面走出來的，是一個小小的女孩。

她的皮膚光潔如玉，她的頭髮烏黑如夜，她的眼睛像是裝滿了星星的湖水。陽光照在她身上，整個人發著淡淡的光。

這家人驚呆了。他們小心翼翼地把女孩抱起來，給她取了一個名字——嬤蓋蓋。

嬤蓋蓋長得好快。她一天比一天高，一天比一天漂亮。她的美不是普通的美，是那種讓人看了會忘記說話的美，像山上清晨的第一道雲霧，像溪水裡映著陽光的石頭。

嬤蓋蓋長大的消息，傳過了一座山，又傳過了一座山，傳到了天上。

天上有一位年輕的男神，名叫古勒勒。古勒勒聽說人間有一位從蛋裡出生的美麗女子，他好奇極了。他從天上往下看——他一看到嬤蓋蓋，心就怦怦怦地跳了起來。

那時候，天和地之間還有一條路可以走。那是一座石頭做的梯子，從地面一直延伸到天上，一階一階的，很長很長。神可以沿著石梯走下來，人如果願意爬的話，也可以爬上去，雖然要爬很久很久。

古勒勒沿著石梯走了下來。

他來到嬤蓋蓋的面前，對她說：「我從天上來。我想娶妳為妻。」

嬤蓋蓋看著這個從天上走下來的男神。他的眼睛裡有星光在轉動，他的聲音裡有雲在流動。

她答應了。

村子裡的人為他們舉辦了盛大的婚禮。大家穿上最美麗的衣服，戴上最漂亮的琉璃珠。歌聲響了三天三夜，舞跳了三天三夜。百步蛇安靜地盤在門口的石板上，守護著這場婚禮。

可是，婚禮結束之後，嬤蓋蓋的心裡開始不安。

她想：「古勒勒是天上的神。他的家在天上。如果有一天他想回去呢？他只要沿著石梯往上走，就可以回到天上，然後......就再也不回來了。」

這個念頭像一根刺，扎在嬤蓋蓋的心裡，拔不出來。

一天夜裡，古勒勒睡著了。月亮很亮，風很涼。

嬤蓋蓋悄悄地起了床。她拿起一把石斧，輕手輕腳地走出了石板屋。

她來到天梯旁邊。

石梯在月光下閃閃發光，一階一階地往上延伸，一直伸進雲裡面，看不到盡頭。

嬤蓋蓋舉起石斧。

她的手在發抖。她知道，砍斷這座天梯意味著什麼——天上的神再也下不來，地上的人再也上不去。天和人之間的路，永遠斷了。

可是她更怕失去古勒勒。

她咬緊嘴唇，用力砍了下去。

「喀——！」

一聲巨響，第一階石梯裂成了兩半。

「喀——！喀——！」

她一下又一下地砍。石頭碎片飛濺到她臉上，她的手磨出了血，但她沒有停下來。

終於，天梯斷了。

斷裂的石階轟隆隆隆地崩落下來，砸在山坡上，碎成一塊一塊的石頭，滾進了山谷裡。

從那以後，天上就是天上，人間就是人間。

神再也沒有沿著梯子走下來過。人抬頭看天，天好高好高，高到不管怎麼伸手都摸不到。

古勒勒醒來之後，看到天梯斷了。他沒有生氣。他轉頭看著嬤蓋蓋，什麼也沒有說。

他留了下來。

他留在人間，和嬤蓋蓋一起生活。他們有了孩子，孩子有了孩子。他們的後代成了魯凱族的祖先。

而那些散落在山坡上的天梯碎片呢？

人們說，那就是魯凱族石板屋的石板。每一塊石板，都是天梯的一小片。所以，每一間石板屋裡面住的人，都跟天有一點點關係。

每當夜裡月亮很亮的時候，魯凱族的老人會抬頭看天，然後對孩子說：「從前，天和地是連在一起的。從前，神和人是走在同一條路上的。」

## D05（週五）：第一週統整——從混沌到秩序

### Part 1 晨圈活動（20 分鐘）

**流程**：

1. **集合站圓 + 深呼吸**（2 分鐘）
2. **完整晨圈串連**（13 分鐘）：將本週學的所有素材連續進行——
   - 念謠〈在那之前〉（一次）
   - 念謠〈半石半木〉（一次）
   - 歌曲〈大海的呼吸〉（一次）
   - 歌曲〈Rangi 和 Papa〉（一次）
   全程不中斷，教師用眼神和手勢帶領轉換。到了第五天，孩子們應已能流暢完成。
3. **意志力活動：創世之舞**（5 分鐘）：教師口述，全班用身體演出本週故事的核心意象——
   - 「黑暗」（全班蹲下、縮小）
   - 「推開天地」（慢慢站起、雙腿用力撐開）
   - 「光照進來」（張開雙臂、挺胸抬頭）
   - 「石頭變土、樹長出來」（身體從僵硬慢慢變柔軟，手臂如樹枝展開）
   - 「雷電劈中大樹」（拍手、蹲下再跳起）
   - 「楓葉飄落」（輕輕旋轉、緩緩蹲下）
   - 「嬤蓋蓋砍天梯」（做砍的動作三下）
   - 「天和地分開了」（雙手從合十慢慢分開至兩側，靜止）

### Part 2 前日故事回顧與討論（30 分鐘）

**回顧嬤蓋蓋的故事**（10 分鐘）：

教師提問：「昨天的故事裡，嬤蓋蓋在夜裡做了什麼？她為什麼要砍斷天梯？」

**DI 差異化**：
- **A 學生**：「嬤蓋蓋砍斷天梯是對的嗎？她有沒有別的選擇？如果你是她，你會怎麼做？」引導辯證思考。
- **B 學生**：「嬤蓋蓋害怕什麼？她拿了什麼東西去砍？砍斷之後發生什麼事？」事實層提問。
- **C 學生**：「你有沒有因為害怕失去什麼人，而做了一個很大的決定？」連結個人經驗。
- **D 學生**：「你來演嬤蓋蓋。先偷偷起床——走到天梯旁邊——舉起石斧——砍！」身體演出再口述。

**整週串連回顧**（15 分鐘）：

教師在黑板上畫出一條長長的時間線，邀請孩子一起回顧：

```
黑暗 → 天地分開 → 石頭變土 → 植物生長 → 半石半木大樹 → 雷電 → 動物與人 → 尼弗奴造更多人 → 陶壺裡的蛋 → 嬤蓋蓋 → 天梯斷裂 → 天人分離
```

教師引導核心主題：「這一個禮拜，我們聽了好幾個不同民族說的故事。每一個故事都在說同一件事——世界是怎麼從什麼都沒有，慢慢變成現在這個樣子。你覺得這些故事裡，最重要的一刻是哪一刻？」

讓每個孩子說出自己覺得「最重要的一刻」。教師不評判，溫暖地接住每個回答。

**跨故事比較**（5 分鐘）：

- 「毛利族的塔內推開了天和地，嬤蓋蓋砍斷了天梯。兩個人都讓什麼東西分開了。分開，是好事還是壞事？」
- 「分開了之後，什麼新的東西出現了？」

## 附錄：副課程延伸活動建議

以下活動由副課程教師執行，與主課程故事內容相互呼應，在不同感官通道深化孩子的體驗。

### 一、溼水彩（搭配 D01 天地初始）

**主題**：黑暗中的第一道光

**操作指引**：
1. 溼水彩紙浸水後平放於畫板
2. 全紙先以大筆刷上深藍色與深紫色，讓兩色在紙上自然交融，呈現「黑暗」的感覺
3. 等深色稍微穩定（約 2-3 分鐘），在畫面中央偏上用乾淨的筆沾取金黃色，輕輕點觸紙面
4. 不要塗抹，讓金黃色自己向外暈染擴散，形成從中心向外的光芒效果
5. 如果光芒不夠亮，可以在中心點再加一點檸檬黃
6. 全程不畫具象物體，只有顏色與光的互動

**教師語言引導**：「先讓整張紙都沉入黑暗裡。深藍色和深紫色就是那個什麼都沒有的時候。然後——光來了。你把金色輕輕放上去，看它自己張開。不要急，光會自己找到路。」

**對應故事意象**：天空之神與大地女神被推開的那一瞬間，第一道光照進黑暗世界。

### 二、蜜蠟（搭配 D02 生命之樹）

**主題**：半石半木的大樹 / 雷電變出的動物

**操作指引**：
1. **選項 A——半石半木大樹**：取灰色蜜蠟捏成一半樹幹（表面刻出石頭紋理），棕色蜜蠟捏成另一半（表面刻出木紋）。兩半合在一起成為一棵樹。頂部加綠色枝葉。
2. **選項 B——雷電動物**：選擇一種從樹裡跑出來的動物（鹿、山豬、藍鵲、蜥蜴等），用蜜蠟捏塑。鼓勵孩子先搓揉暖化蜜蠟，感受從硬（石）到軟（木到生命）的變化。

**教師語言引導**：「先把蜜蠟放在手心裡搓。它一開始是硬的，像故事裡的石頭。搓啊搓，你的手的溫度讓它變軟了。就像故事裡的石頭經過風吹雨打變成了土一樣。然後你可以開始把它變成你想要的形狀。」

**對應故事意象**：泰雅族創世故事中的半石半木大樹，以及雷電劈開大樹後動物與人的誕生。

### 三、形線畫（搭配 D03 楓葉與果實）

**主題**：楓葉對稱形、果實圓形——對稱與曲線練習

**操作指引**：
1. **楓葉對稱練習**：教師先在黑板上畫一條直線（中軸線），然後在左邊畫出半片楓葉的鋸齒輪廓。請孩子在右邊畫出對稱的另一半。練習 3-4 次後，在工作本上自行完成一片完整的對稱楓葉。
2. **果實圓形練習**：從一個點開始，畫出螺旋逐漸擴大成圓（象徵果實從小到大）。練習不同大小的圓形，最後在頁面上安排一組「楓葉與果實」的形線畫——楓葉用鋸齒線條，果實用圓滑曲線，兩者交替排列。
3. **色彩**：楓葉用紅色/橘色蠟筆，果實用棕色/金色蠟筆。

**教師語言引導**：「楓葉的兩邊長得一模一樣，就像照鏡子。你的左手畫什麼，右手就畫什麼。仔細看——每一個尖角，每一個彎曲，都要對得上。果實是圓的，你的手要像在畫一個永遠不結束的圈。」

**對應故事意象**：尼弗奴搖落楓樹的葉子與果實，葉子變成女人、果實變成男人。對稱象徵男女互補，圓形象徵生命的完整。

### 四、建築（搭配 D04 石板屋）

**主題**：魯凱族石板屋模型

**操作指引**：
1. **材料**：收集扁平小石板（或使用厚紙板裁成石板形狀）、黏土作為接合劑、小樹枝作為支撐
2. **觀察**：先展示魯凱族好茶舊社石板屋的照片（2-3 張），讓孩子觀察石板屋的結構——牆壁是一層一層疊上去的，屋頂是斜的用大石板蓋住
3. **動手蓋**：每組 3-4 人，合力用小石板一層一層疊出一面牆，再疊出四面牆圍成一間小屋，最後蓋上屋頂。門口放一個用黏土做的小陶壺
4. **延伸**：如果用真石板有困難，可以用木塊或積木替代，重點在「一層一層堆疊」的建造經驗

**教師語言引導**：「魯凱族的人不用水泥，不用鋼筋，他們用山上的石板一塊一塊疊出自己的家。每一塊石板要放穩才能放下一塊，就像故事裡的世界——先有黑暗，再有光；先有石頭，再有土；先有土，再有生命。一步一步，不能跳過。你們蓋房子也是一樣，一塊一塊來。」

**對應故事意象**：嬤蓋蓋故事中的魯凱族石板屋，以及天梯碎片化為石板的傳說。建造活動呼應三年級三大支柱之一。

【秋假作業】
一、認讀語文工作本的每一個字給爸爸、媽媽聽。在小白本裡練習秋季語文課的七個生字。
二、在數學工作本裡寫阿拉伯數字1~12、國字一~十二，和爸爸、媽媽一起找找看生活中哪裡還有1~12（口頭分享即可）。
三、學會寫自己的名字。
四、每天做家事、跳跳繩至少100下，已經會連續跳的人至少200下。

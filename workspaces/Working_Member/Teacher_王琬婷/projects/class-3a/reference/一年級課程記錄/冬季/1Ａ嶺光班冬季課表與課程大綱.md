一、週期性主題課程
（一）語文（3週）
透過格林童話及各國童話故事的講述帶入部首字，以及延伸之字、詞及簡短的句子。每個故事介紹二至三個部首字及二至三個衍生字，本季延續秋季主題，以與自然界相關的字為主，如雨、天、風、飛、火、山、冰、光、飛及鳥、隹、羽等。故事暫定為：姆指孩兒漫遊記、六隻天鵝、冬天的玫瑰、風雪婆婆、星星銀幣等。

（二）數學（4週）
數字1-120；加減乘除精靈；50以內的四則運算；數的分合；序數；奇數、偶數。故事暫定為：三根羽毛、跳破了的鞋、六人闖遍天下、睡美人、鞋匠和小精靈、勇敢的小裁縫等。
（三）自然（3週）
配合語文課程內容，透過自然故事的講述，以及散步、手工、藝術等活動，感受自然環境的存在與變化。故事暫定為小風兒、愛抱怨的小溪、去找北風討回麵粉的青年人、冬至公主、吃黑暗的怪獸、天空藍的斗蓬、芬芳潔白的水仙花、燈猴的故事、老鼠娶親等。

二、專科課程
蜜蠟：配合當週課程故事，進行基礎的動、植物形塑。
濕水彩：配合主課程故事或季節的變化，以紅黃藍三原色中擇二色搭配，進一步到三原色同時使用，表現自然環境或故事中的意象，藉以感受顏色與顏色的相遇。
形線畫：螺旋、直線與曲線結合的線性圖形，水滴等封閉圖形…。
 
面對孩子的學習，我們所要做的除了支持，還有提供好的環境以滋養感官，讓孩子以自己的力量成長。

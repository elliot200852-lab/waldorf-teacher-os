# 12A 畢業專題：兩萬字書面報告寫作鷹架（三次課程草案）

> **發布日期**：2026-03-15
> **課程階段**：畢業專題書面報告撰寫期
> **核心痛點**：12年級學生面對兩萬字書面報告的意志力挑戰與存在焦慮
> **系統對應**：現象學式學習、意志力鍛鍊、差異化教學 (A/B/C/D)

---

## 課程設計核心理念

對 12 年級的學生來說，寫兩萬字不僅是邏輯與學術能力的挑戰，更是一場「意志力」與「存在焦慮」的心理戰。我們規劃透過三次課程，將寫作拆解為可執行的步驟，並提供心理上的安全感與結構支撐。

---

## 三次課程結構大綱

### 第一堂課：拆解巨獸（結構與心法 / 降低焦慮）

**核心目標：** 讓「兩萬字」不再是一個不可觸碰的數字，提供穩定的外骨骼。
**教學流程：**
1. **確立邊界：** 介紹報告的基本結構（前言、文獻/背景、實作歷程、反思、結論）。
2. **骨架搭建：** 帶領學生做「章節骨架與心智圖」。不碰內文，只決定五個大章節及各章節預計字數。
3. **心理建設：** 探討寫作焦慮的來源，並提供「先求有，再求好」的心理策略。
**華德福對應：** 為氾濫的思緒提供「外骨骼」支撐，讓焦慮的青年靈魂有明確的方向感與安全感。

### 第二堂課：血肉填補（素材轉化與論述）

**核心目標：** 解決面對空白 Word 檔案的寫作恐懼，啟動頭部（Head）的邏輯思考。
**教學流程：**
1. **素材盤點：** 教導如何將實作紀錄、照片、訪談逐字稿、心路歷程「轉譯」成正式文字。
2. **寫作衝刺 (Sprint)：** 進行 15 分鐘的小型寫作衝刺活動（例如：寫下專題動機或實作中最挫折的一件事），立即產出 300 字。
3. **段落結構：** 介紹基礎段落架構設定（主題句、支持句、結論/反思）。
**華德福對應：** 將感官衝動轉化為因果邏輯推演，強迫思考（Head）的介入，將星芒體的能量具體化為文字。

### 第三堂課：節奏與打磨（意志力衝刺與個人寫作計畫）

**核心目標：** 讓短期的產出變成長期的寫作習慣，鍛鍊意志力（Will）。
**教學流程：**
1. **同儕力量：** 互相檢閱彼此已產出的段落，提供建設性的回饋（修復性對話的延伸應用）。
2. **建立查核點：** 共同建立每週交稿進度表。
3. **個人計畫表：** 學生制定專屬自己的「交稿時間表」與寫作時段承諾。
**華德福對應：** 從穩定的常規行動中鍛鍊意志（Will），建立對自身專題的絕對責任感。

---

## 差異化教學 (DI) 策略支援矩陣

在上述三次課程中，必須針對班上不同狀態的學生提供以下四層次的任務引導：

| 類型 | 能力與動機狀態 | 對應策略 (Strategy) | 課堂實際操作畫面 |
| :--- | :--- | :--- | :--- |
| **A** | **高能力 / 高動機** | **延伸挑戰、擔任小老師** | 引導他們建立獨特的文獻探討視角。在第三堂課中，指派他們擔任特定寫作風格（如數據分析、文獻回顧）的同儕檢閱員。 |
| **B** | **低能力 / 高動機** | **鷹架支持、小步驟成功** | 提供非常具體的「段落模板」（例如：第一句寫ＸＸ，第二句寫ＯＯ）。在第二堂課的寫作衝刺中，給予明確的填空式引導，讓他們能迅速累積字數建立自信。 |
| **C** | **高能力 / 低動機** | **連結興趣、減少重複** | 允許他們跳出標準結構，從自己最有熱情的那一塊（實作、訪談、心得）開始寫起。暫時不要求起承轉合，先有文字產出即可。 |
| **D** | **低能力 / 低動機** | **極致降焦、建立安全感** | 絕口不提「兩萬字」。第一堂課的目標設定為「寫出前言的 200 字」或利用「語音轉文字工具」產出口語材料。降低認知負荷，陪伴建立基礎產出。 |

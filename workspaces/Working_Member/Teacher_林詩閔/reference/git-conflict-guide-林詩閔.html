Git 衝突排除 — 三步完成

林詩閔老師・2026-04-08

你的工作成果都已經安全保存在雲端（GitHub）上。
以下操作只是讓你的電腦重新與雲端同步，不會遺失任何東西。
1
開啟 Git Bash
在你的 waldorf-teacher-os 資料夾內，
按右鍵 → 選「Open Git Bash here」。
2
貼上這三行指令
在黑色視窗中，把下面三行一次全部複製貼上，然後按 Enter： 
# 取消目前的衝突狀態
git merge --abort
# 抓取雲端最新版本
git fetch origin
# 讓本機與雲端完全同步
git reset --hard origin/main
完成後，你會看到類似 HEAD is now at xxxxx 的訊息，代表成功。
3
確認完成
輸入以下指令確認沒有問題： 
git status
應該會顯示「nothing to commit, working tree clean」。

看到這個訊息就代表衝突已排除，可以正常開工了。
以後每次開始工作前，先執行 git pull origin main 就能避免衝突。

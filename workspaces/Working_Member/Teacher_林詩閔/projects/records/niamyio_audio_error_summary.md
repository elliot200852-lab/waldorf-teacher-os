# 🪀 台語唸謠語音合成 (TTS) 功能異常與除錯總結

**建立日期：** 2026-03-24
**相關技能：** `niamyio` (做唸謠)
**測試範圍：** 童玩唸謠系列 (干樂、毽子、扯鈴)

---

## 🛑 問題描述 (What Happened)

在為《台語童玩唸謠》講義建立「一鍵即聽」語音功能時，我們遇到了下列三個階段的失敗：

1.  **最初方案：意傳 (ÌTHUÂN) 台語 TTS 連結**
    網址直接失效 (DNS Lookup Error / 404 Not Found)。這表示該第三方語音合成網站的公開 API 正在關閉維護或已更改了網址格式。
2.  **備案一：教育部 (NAER) TTS API 連結**
    在部分瀏覽器或手機裝置上，直接點擊這些帶有 `text=...` 參數的外部 API 連結，會因為跨網域安全性政策 (CORS) 或缺少瀏覽器授權而被擋下，無法自動發聲。
3.  **備案二：透過 Python 腳本即時下載音檔給本地使用**
    執行下載腳本時遭遇 `getaddrinfo failed` (DNS 解析失敗 / 網路連線阻擋) 錯誤。作業系統層級（Waldorf Teacher OS）可能針對不明的命令列網路請求進行了安全阻擋。

---

## 🔍 原因剖析 (Why It Errored)

*   **過度依賴外部開源資源 (External Dependency Risks)**：
    台語語音合成資源（如意傳、Taigi AI Labs）多為學術或實驗性質機構維護，伺服器極不穩定，且網址格式常常隨著版本更新而失效。
*   **平台安全限制 (Security Restrictions)**：
    無論是網頁上的外部 API 呼叫，或是終端機的 HTTP 下載指令，都容易被現代瀏覽器和防毒/防火牆系統判定為不安全行為而直接攔截。
*   **裝置相容性問題 (Device Compatibility)**：
    將帶有複雜字串（如台羅拼音 `se̍h-lûn-tǹg`）的網址傳到手機 (Line / Safari)，很容易出現編碼錯誤導致連結斷裂。

---

## ✅ 最終解決方案 (The Solution)

為了確保您在教室（甚至是無網路環境）或手機上都能 **100% 穩定播放**，我們徹底改變了策略：

1.  **放棄「即時線上生成」(Abandon Live Links)**：
    不再將不穩定的外部 API 網址寫進備課講義中。
2.  **改用「本地實體音檔」(Local Audio Files)**：
    在您的工作區建立了一個實體的 `niamyio_audio` 資料夾 (`workspaces/Working_Member/Teacher_林詩閔/niamyio_audio`)，用來存放真正的 `.mp3` 檔案。
3.  **建立「離線版 HTML 講義」(Offline HTML Guide)**：
    將講義改寫為 `niamyio_toy_guide_2026.html`，裡面的播放器直接讀取同一個資料夾內的音檔。
    
**👉 未來行動呼籲：**
由於本次系統網路阻擋，我們暫時建立的是假音檔 (Dummy Files) 來維持資料夾結構。未來建議老師在備課資源豐富時，**手動錄製真實人聲（或穩定生成後下載）覆蓋這三個檔案**，即可達成完美的本地語音講義庫！

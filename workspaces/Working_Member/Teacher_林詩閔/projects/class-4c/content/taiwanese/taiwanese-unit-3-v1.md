# 台語單元教學設計：自我介紹整合發表 (L8)

> **單元名稱**：第三單元 — 我的果子、我的馬、我自編的介紹  
> **適用班級**：四年級 C 班  
> **重點**：將碎片化的語言（水果、喜好、故事、馬匹）整合為具備個人主體性的完整表達。

---

## 一、 核心表達：我的台語名片 (Guá ê Tâi-gí bîng-phìnn)

透過直接、具節奏性的對仗句，讓學生在不需解釋原因的情況下，建立流暢口語：

1.  **打招呼 (Ta̍k-ke hó)**：  
    逐家好 (ta̍k-ke hó)，我係 [名字]。

2.  **水果記憶 (Kó-tsí kì-it) — 趣味生活全集**：  
    我上愛食 [水果]，[水果][水果]，[趣味描述]。
    *   正月：我上愛食**甘蔗**，甘蔗甘蔗，**愈哺愈甜** (jú póo jú tinn)。
    *   二月：我上愛食**楊桃**，楊桃楊桃，**切開變天星** (tshiat-khui piàn thinn-tshinn)。
    *   三月：我上愛食**gî-pê**，gî-pê gî-pê，**食起來真快活** (tsia̍h khí-lâi tsin khuìnn-ua̍h)。
    *   四月：我上愛食**木瓜**，木瓜木瓜，**軟膏膏真好哺** (nńg-kô-kô tsin hó-póo)。
    *   五月：我上愛食**李仔**，李仔李仔，**酸甲流頷垂** (sng kah lâu-ām-suî)。
    *   六月：我上愛食**西瓜**，西瓜西瓜，**摃開一肚紅** (kòng--khui tsi̍t-tōo âng)。
    *   七月：我上愛食**荔枝**，荔枝荔枝，**古早皇帝愛** (kóo-tsá hông-tè ài)。
    *   八月：我上愛食**龍眼**，龍眼龍眼，**甜滋滋** (tinn-tsis-tsis)。
    *   九月：我上愛食**梨仔**，梨仔梨仔，**清甜止喙焦** (tshing-tinn tsí tshuì-ta)。
    *   十月：我上愛食**蓮霧**，蓮霧蓮霧，**脆甜多汁好滋味** (tshè-tinn to-tsiah hó-tsu-bī)。
    *   十一月：我上愛食**柳丁**，柳丁柳丁，**芳芳營養逐家愛** (phang-phang îng-ióng ta̍k-ke ài)。
    *   十二月：我上愛食**柑仔**，柑仔柑仔，**大吉利** (tuā-kiat-lī)。

3.  **馬匹記憶 (Bé ê kì-it) — 趣味形象描述**：  
    我上愛騎 [馬匹]，[馬匹][馬匹]，[趣味描述]。
    *   **烏馬**：我上愛騎烏馬，烏馬烏馬，**走步真威風** (tsáu-pōo tsin ui-hong)。
    *   **白馬**：我上愛騎白馬，白馬白馬，**白雪雪真古錐** (pe̍h-suat-suat tsin kóo-tsui)。
    *   **斑馬**：我上愛騎斑馬，斑馬斑馬，**穿一領花衫** (tshīng tsi̍t-niá hue-sann)。
    *   **赤兔馬**：我上愛騎赤兔馬，赤兔赤兔，**走敢若飛** (tsáu kán-ná pue)。
    *   **寶馬**：我上愛騎寶馬，寶馬寶馬，**有翅仔飛天頂** (ū si̍t-á pue thinn-téng)。
    *   **鐵馬**：我上愛騎鐵馬，鐵馬鐵馬，**免食草嘛會走** (bián tsia̍h-tsáu mā ē tsáu)。
    *   **碌硞馬**：我上愛騎碌硞馬，碌硞馬碌硞馬，**搖到心快活** (iô-kah sim khuìnn-ua̍h)。

4.  **結語 (Kiat-gú)**：  
    祝逐家平安 (Tsiok ta̍k-ke pîng-an)，勞力 (Ló-la̍t)！

## 二、 課堂教學設計：成果發表

*   **重點**：回歸平穩的發表，重點在於發聲與語感。
*   **教學規劃**：
    *   **暖身**：集體唸誦一至十二月唸謠，由老師以響木帶動節拍。
    *   **發表**：每位學生依上方結構進行簡單發表。
    *   **回饋**：全班整齊拍手兩次，說：「讚啦 (Tsand--lah)！」

## 三、 DI 差異化操作建議 (L8)

| 學生類型 | 心理素描 | 具體發表設計策略 |
| :--- | :--- | :--- |
| **A 類 (高能高動)** | **「故事的說書人」**：不滿足於填空，想要展現個人風格。 | 1. 鼓勵他們加入故事細節（例如：我愛食甘蔗，因為甘蔗是天馬救命的桃樹變的）。<br>2. 嘗試使用更豐富的形容詞調色。 |
| **B 類 (低能高動)** | **「儀式感的先行者」**：透過動作與服飾增強自信。 | 1. 提供簡單的「馬頭手杖」或「信使帽」作為道具。<br>2. 容許他們看著自己畫的圖卡提示詞發表。 |
| **C 類 (高能低動)** | **「冷靜的紀錄者」**：需要真實的聽眾與意義。 | 1. 安排他們擔任「信使大會紀錄員」，發表完後，請他點評一兩位同學最有特色的地方（用台語簡單說一兩個詞）。 |
| **D 類 (低能低動)** | **「最小單位的勝任者」**：需要極簡化的結構與支持。 | 1. **極簡版結構**：招呼語 ➔ 水果名 ➔ 馬名 ➔ 勞力。<br>2. 由老師或 A 類同學陪同上台，進行「回聲式發表」（老師說一句，他跟著說一句）。 |

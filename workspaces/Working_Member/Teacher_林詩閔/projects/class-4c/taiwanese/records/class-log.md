# 四年級C班 - 台語教學日誌 (Class Log)

這是一份紀錄全班性教學觀察、課堂氣氛、節奏調整以及集體學習動態的日誌。

### 台語能力五級定義 (2026-03-24 更新)

*   **1 級 (流利領袖)**：能在生活中流利使用台語，課堂學習內容能全然跟上，能自行延伸運用，並協助同學。
*   **2 級 (積極自信)**：在生活中有使用台語的機會，課堂學習內容聆聽理解佳，口語表達較 1 級學生弱一點，但能學習後正確回應，有時致可自行延伸應用。
*   **3 級 (普通/需輔助)**：生活中沒有較多機會使用台語，課堂學習內容聆聽後需要搭配肢體動作或是說明或是華語翻譯才能完全理解，口語表達需要多次練習，練習後能正確回應。
*   **4 級 (基礎/挑戰發音)**：生活中沒有什麼機會使用台語，課堂學習內容聆聽後需要搭配肢體動作或是說明或是華語翻譯才能完全理解，口語表達需要多次練習，練習後有時仍無法在發音及聲調上完全正確回應。
*   **5 級 (初階/困難生)**：生活中完全沒有使用台語，聆聽理解具有難度，依賴肢體動作猜測意思或需要老師同學用華語說明，口語表達在發音上較困難正確，也不容易說完整的句子。

### [2026-03-23] 轉向無視覺學習的轉型期觀察 (林詩閔老師)

四年級的教學持續進行歌曲、唸謠和故事，學習的內容我會寫在黑板，學生除了聽覺及言說的學習，也進入視覺學習，認讀常用的語詞，許多語詞的台文寫法和華語相同，學生是容易記住，心田班在視覺學習方式上，圖卡、字卡、黑板書寫，學習單黏貼在工作本，多數能在多次唸讀後認讀。

### [2026-03-24] 《水果唸謠》賓果遊戲：非競爭帶動學習 (林詩閔老師)

今日利用《水果唸謠》玩賓果遊戲：
*   **操作方式**：老師抽籤，抽到學生即時大聲唸出他想圈的語詞，正確獲得老師通過即可圈。
*   **教學觀察**：老師沒有以「競賽、第一名、獎勵排序」的方式進行遊戲，因此心田班的學生可以心平氣和地在達成一條、兩條甚至多條連線的過程中感到滿足。
*   **遊戲性與學習**：這種非競爭式的設計，對於能力較弱的孩子非常有包容性，大家即使進度快慢不一，都能從中獲得開口表達的成就感。

1. 干樂 (kan-lo̍k) - 陀螺

API 強化：正字「干樂」 | 技術：拋、轉
纏線纏線纏予紧，拋落地下 se̍h-lûn-tǹg！
意志力動作： 模擬雙手快速交叉纏繞 -> 右手臂向外順滑拋出 -> 原足原地旋轉。

2. 毽子 (kian-tsí) - 毽子

API 強化：正字「毽子」 | 技術：踢、盤
毽子輕輕飛起去，一下踢、兩下盤，大步跳過山！
意志力動作： 小碎步踏步 -> 做出腳跟向後踢起手勢 -> 兩手用力向上向上展。

3. 扯鈴 (tshé-lîn) - 扯鈴

API 強化：正字「扯鈴」 | 技術：抽、扯
一下抽、一下扯，扯鈴旋甲真大聲。
意志力動作： 左右手輪流向上向下拉動繩索 -> 雙手手掌對齊合掌摩擦 (模擬旋轉聲)。

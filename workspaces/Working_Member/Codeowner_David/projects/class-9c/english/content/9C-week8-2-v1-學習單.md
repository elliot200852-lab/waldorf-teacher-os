## 選擇你的任務 Choose Your Task

### 📖 Task D — 配對連線 Match

Draw a line to connect each sentence beginning with its ending. Look at the connector (連接詞) to help you.

| Sentence Beginning | | Sentence Ending |
|:---|:---:|:---|
| 1. I didn't know **where** | | a. Aunt Lala said so. |
| 2. I wore the navy blue dress **that** | | b. the envelopes were going. |
| 3. I borrowed money **because** | | c. I wouldn't get paid yet. |
| 4. I wouldn't get paid **till** | | d. made me look older. |
| 5. I went to North Broadway **where** | | e. the next Friday. |
| 6. I lied about my age **like** | | f. Aunt Lala worked. |
| 7. I didn't know **where** | | g. she told me to. |
| 8. I started **and** | | h. to sit down. |

**提示 Hint：** 看看每個連接詞在做什麼工作——

- **where** = 什麼地方 / 哪一個地方
- **that** = 哪一個（描述前面的名詞）
- **because** = 原因
- **till** = 直到
- **like** = 像...一樣
- **and** = 然後

### ✍️ Task A — 自由書寫 Your Own Long Sentence（回家作業 Homework）

Think of a day when EVERYTHING happened at once — a busy day, a confusing day, a day you couldn't stop.

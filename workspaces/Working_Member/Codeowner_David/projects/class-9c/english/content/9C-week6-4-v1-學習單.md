### Task A — Writer's Craft Analysis

Cisneros is famous for writing in a style that sounds like a young person talking. Choose **ONE** of the following and write a short paragraph (5–8 sentences).

**Option 1 — The Letter**
Rewrite the mother's letter to Sister Superior. Keep the same message, but make it longer and more formal. Then answer: *Which version is more powerful — the original or yours? Why?*

**Option 2 — The Last Line**
The story ends with a strong image. Why is it a good ending? What does the reader feel? How does Cisneros create that feeling with so few words?

**Option 3 — Esperanza's Voice**
Write a short paragraph (like Cisneros) about a time when you wanted to belong somewhere. Use short sentences. Use connectors. Try to make the reader *feel* what you felt.

## Connector Reference / 連接詞參考表

| Connector | 中文 | What it does |
|-----------|------|-------------|
| **if** | 如果 | condition 條件 |
| **because** | 因為 | reason 原因 |
| **when** | 當...的時候 | time 時間 |
| **before** | 在...之前 | order 順序 |
| **after** | 在...之後 | order 順序 |
| **although** | 雖然 | contrast 對比 |
| **even though** | 即使 | strong contrast 強烈對比 |
| **until** | 直到 | time limit 時間界線 |
| **while** | 當...的同時 | same time 同時 |

*New friends on the tree: even though, until, while — you'll meet them officially next week.*

## Connector Reference / 連接詞總表

<div class="two-column">
<div class="column">

### Subordinating Connectors 從屬連接詞

| Connector | 中文 | Category |
|-----------|------|----------|
| **if** | 如果 | Condition 條件 |
| **unless** | 除非 | Condition 條件（反面） |
| **because** | 因為 | Cause 原因 |
| **so (that)** | 為了 | Purpose 目的 |
| **when** | 當...的時候 | Time 時間 |
| **before** | 在...之前 | Time 時間 |
| **after** | 在...之後 | Time 時間 |
| **until** | 直到 | Time 時間（終點） |
| **although** | 雖然 | Concession 讓步 |
| **even though** | 即使 | Concession 讓步（更強） |

</div>
<div class="column">

### Coordinating Connectors 對等連接詞

| Connector | 中文 | What it does |
|-----------|------|-------------|
| **and** | 和、而且 | addition 添加 |
| **but** | 但是 | contrast 對比 |
| **or** | 或者 | choice 選擇 |
| **so** | 所以 | result 結果 |

> **Notice:** "so" appears in both lists! As a coordinating connector, it means "therefore / 所以." As part of "so that," it becomes subordinating and means "in order to / 為了."

</div>

### Task D -- Choose the Correct Connector

Each dialogue has a blank. Two options are given. **Circle** the correct one.

**1.**
> A: Are you going to look into that problem?
> B: Yes, I'll look into it ____________ I have some free time.

( **when** / **but** )

**2.**
> A: We need to set up the classroom for the presentation.
> B: I know, ____________ we don't have enough chairs.

( **until** / **but** )

**3.**
> A: Why did they put off the field trip?
> B: They put it off ____________ it was raining all week.

( **because** / **or** )

**4.**
> A: Will you carry on with the project?
> B: I'll carry on ____________ the teacher tells us to stop.

( **so** / **until** )

**5.**
> A: We'll run out of paper ____________ we don't get more today.
> B: I'll go buy some.

( **if** / **so** )

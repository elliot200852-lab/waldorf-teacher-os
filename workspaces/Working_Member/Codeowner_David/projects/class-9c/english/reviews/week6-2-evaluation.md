## 二、核心問題（校準後新增）

### 過度差異化
概念引入課不需要 Task D / Task C / Task B 三層。鷹架清楚 + 兩層（引導版 / 開放版）就夠。四層把課搞複雜，教師操作負荷高，學生被切得太碎。

### 缺乏有語境的應用
分層練習是「填連接詞」而非「用連接詞」。應該讓學生在 pair work 或小組中，用連接詞完成有意義的溝通任務（例：用連接詞描述一個真實事件、兩人互相造句挑戰對方）。

### 活動結構未產生自然互動
全堂幾乎沒有 pair work 或分組。分組本身就會產生移動和互動，不需要硬塞「站起來跟讀」這種人工動覺活動。

## 三、10 原則 + DI

- 其餘 pass

## 四、跨課堂

## 五、整體印象

學生會記住：Connector Family 樹。但「用連接詞做了什麼事」的記憶不深——因為應用環節太薄。

---

## 修正方向

1. 差異化從四層砍為兩層（鷹架版 + 開放版）
2. 分層練習改為 pair work 應用——有語境、有互動、有驗收
3. 「新成員登場」12 分鐘太長且全是教師帶——穿插 pair work 微任務

## 亮點

1. Connector Family 樹——跨週生長的視覺鷹架
2. 動詞片語以配角嵌入——節律性重複

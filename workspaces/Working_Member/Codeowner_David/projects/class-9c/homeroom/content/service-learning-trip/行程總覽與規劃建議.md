# 九年級 C 班・服務學習之旅｜行程總覽與規劃建議

**日期**：2026/6/8（一）～ 6/12（五），5 天 4 夜
**人數**：22 位學生 + 2 位教師（共 24 人）
**出發地**：宜蘭
**主要城市**：台中 → 台南 → 高雄
**交通方式**：全程大眾運輸（台鐵、高鐵、捷運、公車、渡輪）
**住宿預算**：每人每晚 500–800 元

---

## 一、旅程定位

這不是一般的畢業旅行，而是一趟**服務學習之旅**。核心精神：

1. **服務**：在每個城市安排至少一項志願服務或社會參與活動
2. **學習**：以博物館、歷史場域、生態體驗為主軸，取代純觀光行程
3. **自主**：學生分組負責路線規劃、住宿聯繫、交通查詢、景點研究
4. **在地**：透過小吃街區探索，認識台灣不同城市的飲食文化與庶民生活

- 每日一位輪值「領航員」，帶領全班完成當日行程

- 最後一天車上完成旅程總結

- 各組製作旅程報告（含服務學習成果）
- 個人完成一篇完整反思文章（800-1000 字）

## 十、注意事項

1. **安全**：全程以組為單位行動，不單獨離隊；每組配備一支手機保持聯繫
2. **環保**：自備水壺、餐具、環保袋；服務學習精神從自身做起
3. **預算**：學生自行管理餐費，練習記帳；超支部分自理
4. **天氣**：六月南台灣炎熱（30-35°C），注意防曬、補水；備雨具
5. **禮儀**：博物館、服務機構均需遵守場所規範；代表學校形象
6. **紀錄**：鼓勵拍照但不沉迷手機；重要的是觀察與體驗

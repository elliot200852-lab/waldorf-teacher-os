# 9C 春季班親會 — 主持逐字稿

## 一、開場

各位家長好，好久不見。先跟大家說一聲新年快樂，雖然已經過了一陣子，但這是我們今年第一次見面，還是要講。

這個學期開學到現在，其實節奏很快。冬季末孩子們剛結束戲劇體驗，緊接著春季一開學就進了農場實習，一個接著一個，中間幾乎沒有喘息的空間。我們刻意把班親會安排在農場實習結束之後，讓大家跟孩子們都沉澱一下，然後今天我們坐下來，好好聊聊孩子現在的樣子，也聊聊接下來這最後幾個月的路。

今天的流程大概是這樣：我會先帶大家回顧戲劇跟農場實習這兩段經歷，然後報告一下接下來春季跟夏季的學習活動安排，最後我們有幾個議題需要跟家長一起討論、一起決定。時間我盡量控制在兩個小時以內。

## 二、戲劇歷程回顧

在我開始分享之前，我想先把話筒交給各位。

你們的孩子在準備戲劇的那段時間，回家是什麼樣子？有沒有跟你們聊過什麼？或者你們在台下看演出的時候，心裡有什麼感覺？我很想聽聽家長這邊看到的。因為我們在學校看到的是一個版本，你們在家裡看到的是另一個版本，兩個版本拼在一起，才是比較完整的孩子。

（家長分享時間）

謝謝大家的分享。

我從老師的角度來補充一些你們可能沒看到的部分。整個戲劇排練的過程，其實比最後那場演出要精彩得多。排練的頭兩週，說實話，場面是混亂的。台詞記不住、走位搞不定、有人覺得自己的角色不夠重要想換、有人根本不想開口說話。這些都是正常的，因為戲劇要求的東西跟他們平常習慣的學習方式完全不同——你沒辦法只用腦袋，你得把整個人放進去。

但慢慢地，有些東西開始變了。不是突然之間變好，而是那種很緩慢的、幾乎看不見的位移。有些孩子開始主動幫別人對台詞，有些人在走位的時候開始注意到旁邊的人，而不是只管自己。到了最後幾次排練，我站在旁邊看的時候有一個很清楚的感覺：這群人不只是在演一齣戲，他們在學習怎麼一起完成一件事。

我覺得這是戲劇帶給九年級最重要的東西。不是演技，不是台風，而是他們經歷了一次「整個班必須一起呼吸」的過程。那個過程裡面有摩擦、有妥協、有不耐煩，但最後他們站上台的時候，每個人都知道自己該在哪裡。這件事情的意義，比演得好不好大得多。

我常常在想，我們這種教育到底在做什麼？說到底，我們是在創造一種「學習本身是快樂的」的經驗。不是那種吃到糖的快樂，而是「我做了一件很難的事，而且我做到了」的那種踏實感。戲劇給了他們這個經驗。

## 三、農場實習分享

農場實習是另一種完全不同的學習。如果說戲劇是把人放進一個藝術的框架裡面去感受，農場就是最直接的——你面對的是土地、是天氣、是動物、是你自己的身體。

沒有什麼比農場更簡單的學習方式了。你該除草就除草，該餵雞就餵雞，太陽很大你還是得站在田裡。沒有人會問「這個要考嗎」，因為答案太明顯了——田裡的草不會因為你不想拔就不長。

我在農場觀察到一件很有意思的事情。我們這群孩子在教室裡面，有些人是很會想、很會說的，有些人是安安靜靜不太表達的。但到了農場，這個排列組合完全洗牌了。平常話很多的孩子到了田裡反而安靜下來，因為身體在勞動的時候腦袋會自動切換到另一個頻道。而平常不太說話的孩子，在農場的環境裡反而放鬆了，偶爾會冒出一些讓人驚訝的觀察。

這就是專案式課程的價值。它不是讓學生「學到農業知識」——雖然那也很好——而是讓他們經歷一種不同的存在方式。在農場那幾天，他們不是學生，他們就是一個「在工作的人」。這個經驗會留在身體裡面，比任何考試成績都持久。

我跟孩子們在農場實習結束之後做了一次分享。我跟他們說的重點是：經歷過戲劇、經歷過農場之後，你們應該開始理解一件事情——人生有很多形式上的框架，有些東西看起來像是被安排好的、命定的，但真正讓你自由的，不是逃開那些框架，而是你怎麼在裡面「自己定義」你跟它的關係。這才是真正的自由。不是看你做了什麼、你喜不喜歡，而是你能不能學會跟不喜歡的工作相處。這是我覺得在九年級最後這幾個月，很想讓他們帶走的東西。

## 四、班級學習與活動報告

接下來跟大家報告一下這個學期剩下的節奏。春季還有五週、夏季還有五週，下半學年非常緊湊，活動一個接一個。我先把大的框架跟大家講一下。

**園遊會**

第一個就是園遊會，春季節慶。我們班在週一、週二的班會課已經開始籌備了。孩子們決定做食物攤跟遊戲攤。食物的部分有飯糰跟水果茶：飯糰三種口味，定價一顆三十五元、三顆一百元，預計準備五十份；水果茶三百五十毫升裝，有黑糖口味跟水果茶兩種，大概準備一百份，單價四十到五十元。遊戲攤的部分有橡皮筋射擊跟彈珠台，一次二十到三十元。下週二我們會做食物驗收試吃，看過評價之後最後定案。

**服務學習之旅**

這是一個五天四夜的行程，也已經啟動了。目前的工作模式是這樣：兩位召集人統籌，底下分成交通、住宿、景點、飲食、服務五個小組，學生自己分頭開始計畫。基本上我們讓學生自己規劃這趟旅行，老師在中間把關、協作，但主導權在他們手上。這個部分等一下在討論議題裡面會再跟大家細談。

**節慶與體育週**

下一週是春季節慶週，歡迎各位家長來參與。第七週是體育週，活動包含拔河跟跳繩。

**畢業相關**

畢業旅行預計在第八、九週，主要是第八週要確認所有細節，等一下議題裡面也會跟大家討論。升學的部分，開學之後我會開始收集所有關於慈心直升高中優免的進度。之後還會有一次模擬考，時間確定之後會再通知。

畢業紀念冊的部分，學生們已經討論過了，決定應該是不做了。說實話事情真的很多，我也覺得不需要勉強。

整體來說，接下來這幾週的節奏就是：園遊會、服務學習、畢業旅行，再加上日常的課程跟升學準備。很快地，我們就要開始做畢業前的收尾工作。旅行的細節會做最後定案跟聯絡，帶著大家真正回到當初走進這個班的初心。

接下來我們進入討論。第一個議題是升學跟畢業旅行的部分。

畢業旅行的部分，目前細節還在確認中，主要是第八週的行程安排。這個部分如果家長有任何意見或疑問，我們在這裡一起談。

## 六、討論議題二：服務學習之旅

第二個議題是服務學習之旅的規劃。

我先跟大家簡報一下目前的進度。五個小組已經啟動，學生們這禮拜開始分頭工作。相關的資料我會提供給大家參考。

這裡有一個需要家長一起決定的事情，就是關於零用錢。五天四夜的行程，吃飯的部分可能會用到零用錢，我覺得有必要大家先達成一個默契。我個人的建議是訂一個統一的額度，大家看是兩千五百元或三千元，讓學生帶著當零用錢，剩下的帶回來。

為什麼我覺得要統一？因為我不希望發生「有人吃得很好、有人吃得不好、甚至有人不吃飯」的情況。如果每個家庭給的標準差太多，孩子之間一定會比較，那個比較對誰都不好。所以我希望每個家庭有一個統一的標準，讓大家在同一個基準上。

這是我個人的建議，想聽聽家長的意見，我們再一起決定。

（家長討論時間）

## 七、討論議題三：謝師活動與畢業典禮

最後一個議題，這個比較不急，但我想趁今天大家都在，先聊一下方向。

關於謝師活動。目前班上的想法是，如果要辦，希望全班一起行動。我跟孩子們說過了，全班一起的選項就是「辦」或「不辦」，沒有一半辦一半不辦這種事。所以我想先聽聽家長的想法：你們覺得有沒有必要辦？想怎麼辦？

我先說，老師們其實真的不會在乎形式。我這邊提供兩個可能的方向，大家可以想一下：

第一個方向是以食物表達謝意。孩子們出勞力工作，家長負責提供食物。就是一種很實在的、用行動表達的方式。

第二個方向是不辦正式的宴席，改用其他方式表達心意。比方說準備一塊小蛋糕，請學生寫小卡片。不過這部分可能需要家長幫忙，因為到那時候學生會非常忙碌，我不太覺得應該在那個時間點再派工作給他們。

我覺得重點是怎麼把心意做好。研究要給老師吃什麼、送什麼，其實不是最重要的。心意到了，形式簡單一點，反而更好。今天不用馬上定案，下學期再決定就好，但我希望今天能先出一個大方向。

另外，畢業典禮的部分，除了學校跟學生這邊的安排，家長那邊有沒有想要辦什麼活動？這個也請大家想一下，我們下學期的班親會需要確定。

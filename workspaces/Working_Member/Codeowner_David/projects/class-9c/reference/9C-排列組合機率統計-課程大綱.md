# 課程大綱～九年級排組機

七年發展論中，第三個七年主要發展思考部分，把七年3等分，九年級約適逢第三個七年中的第1等分，其主要任務為思考中的意志(用意志發展思考)，與其呼應，九年級安排數學主課之一為排組機，主課一開始進行的是排列問題，學生須列舉3人排隊、4人排隊、5人排隊的全部狀況，列舉數量不少需要一定的意志力，並且學生在列舉過程中自然會產生一定的秩序規則與觀察到某些數學規律，從意志力慢慢發展思考適合這階段學生的學習狀態，發展思考能力的開頭，先進行紀錄一些不斷反覆的動作再觀察、思考，比憑空想像容易入門，學生能較穩定有方法的進入思考，並樂在其中。

## 排組機課程內容

- 列舉3人排隊、4人排隊、5人排隊的所有情況
- 完全相異物部分排列列舉
- 不盡相異物排列列舉
- 應用問題（有條件排列，曼哈頓問題）
- 組合列舉
- 機率的起源故事
- 擲爻
- 三道門的問題(蒙提霍爾問題)
- 撲克牌排型
- 抽籤

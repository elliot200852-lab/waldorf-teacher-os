# 114 學年九年級農場實習｜返校工作計畫（week 4）

**日期**：2026/3/11（三）～ 3/13（五）
**來源**：2026九年級農場實習回校工作.docx（會議定案版）

---

## 分組回饋主題

學生回校後，以分組討論方式整理農場經驗。四場討論主題：

1. 對農業和農場實習課程的認識與感受
2. 農場的價值跟困難是什麼
3. 個人的轉變；夥伴關係的轉變與體會
4. 印象最深的一件事；最大的學習是什麼

---

## 三日時程總覽

### 3/11（三）全體回校第一日

| 時段 | 內容 |
|------|------|
| 08:00-08:30 | 打掃 |
| 08:40-08:50 | 說明進行方式／優一（組別一） |
| 08:50-09:20 | 第一場討論：對農業和農場實習課程的認識與感受 |
| 09:20-09:50 | 第二場討論：農場的價值跟困難是什麼 |
| 09:20-09:50 | 團體分享／分組報告（3 分鐘/組） |
| 10:50-12:20 | 發表會準備：到分區負責老師的班級工作 |
| 放學 | — |

**分組安排**：3.3.2.2，負責分組：雍喆
**報告要求**：
- 各組有主持人、紀錄、計時，不同人報告
- 兩場討論的報告人不能重複
- 報告內容：產銷 + 特點 + 困難 + 理念 + 表達感謝
- 各組產出報告大綱

**發表格式**：
- 報告時間 5 分鐘
- 不要變成流水帳——農場介紹可參閱展板，重點放在人的參與、感動、改變
- 可用任何藝術性方式呈現（海報、代表上場、整組上場）
- 可投影 PPT（10 張以內）
- 每組一面展板、兩組一張桌子、放工作本
- 隔天交報告講綱給老師

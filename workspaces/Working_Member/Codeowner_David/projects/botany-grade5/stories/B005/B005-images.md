# B005 投影圖畫清單

### 1. 合歡山裸岩上的橙黃地衣

- 標題：合歡山裸岩先鋒——橙黃地衣
- 描述：合歡山高海拔裸露岩面上，鮮豔的橙黃色與灰綠色地衣大片覆蓋著玄武岩或板岩表面。石頭看起來像被人噴了一層漆，但仔細看可以辨認出每一塊鱗片狀的地衣體。這是孩子在台灣最容易理解「岩石先行者」概念的具體畫面。
- 來源建議：iNaturalist — 搜尋 "lichen Hehuan Mountain Taiwan" 或 Wikimedia Commons — "Lichen rock Taiwan alpine"
- 連結：https://www.inaturalist.org/observations?place_id=7485&taxon_id=54743
- 授權：CC BY-NC（iNaturalist 個別上傳者，使用前確認）
- 展示時機：課程開場，教師問「這是什麼？」讓孩子猜測後引出地衣主題

### 2. 松蘿懸垂於雲霧林枝頭

- 標題：松蘿的鬍子——枝條間的懸掛者
- 描述：阿里山或太平山霧林帶中，松蘿（*Usnea*）從闊葉樹或柳杉枝條上大量垂掛，如白鬍子一般隨著霧氣晃動。圖中可見整排樹木，每根枝條都披著絲絲垂下的地衣，光線透過霧氣形成朦朧的綠光。這張圖完美呼應 Kovacs 描述的「懸掛型地衣，像鬍子一樣垂下來」。
- 來源建議：Wikimedia Commons — "Usnea Taiwan" 或 iNaturalist — 搜尋 "Usnea" + 台灣
- 連結：https://commons.wikimedia.org/wiki/Usnea
- 授權：CC BY-SA（依個別檔案確認）
- 展示時機：Kovacs 文本段落「有一種地衣從樹枝上垂掛下來，看起來像鬍子」，讓孩子直接對應描述

### 3. 葉狀地衣（Parmotrema）在校園樹皮特寫

- 標題：校園樹上的葉狀地衣
- 描述：灰綠色的大葉狀地衣（*Parmotrema* 屬）貼伏在樹皮上，邊緣微微捲起，像一片片小耳朵。照片拍攝於低污染環境的老樹樹幹，可見到地衣體的上表面（皮層）與邊緣凹凸的輪廓。這類地衣在慈心校園中可能就有，是「空氣品質指標」的最佳在地範本。
- 來源建議：iNaturalist — 搜尋 "Parmotrema Taiwan" 或直接在校園拍攝使用
- 連結：https://www.inaturalist.org/taxa/54743-Parmotrema
- 授權：CC BY-NC（iNaturalist）或自行拍攝
- 展示時機：討論地衣作為空氣品質指標時，告訴孩子「我們學校的樹上就有這個」

### 4. 地衣橫切面顯微圖（藻類＋真菌共生）

- 標題：地衣的祕密構造——兩個朋友住在一起
- 描述：地衣葉狀體橫切面的顯微鏡照片或教學圖解，清楚標示上皮層（真菌）、藻類層（綠色球狀共生藻）、髓部（真菌絲）、下皮層（真菌）。圖中可明顯看到一粒粒圓球狀的共生藻被真菌絲包圍。這是本課最核心的「共生意象」，讓孩子看見肉眼看不到的結構。
- 來源建議：Wikimedia Commons — "Lichen cross section" 或 Smithsonian Education
- 連結：https://commons.wikimedia.org/wiki/File:Lichen_cross_section_2.jpg
- 授權：公共領域（Public Domain）或 CC BY-SA
- 展示時機：「科學家發現地衣其實是兩種植物」段落，搭配教師說明真菌保護藻類、藻類分享陽光養分

### 5. 馴鹿地衣覆蓋北極凍原

- 標題：北極的灰色軟墊——馴鹿地衣
- 描述：斯堪地納維亞或加拿大北極圈凍原上，*Cladonia rangiferina*（馴鹿地衣）大片鋪展開來，形成蓬鬆的灰白色地毯，一望無際的雪地與地衣相接。前景中一頭馴鹿低頭啃食。這張圖讓台灣孩子看見地衣在地球另一端的壯觀生態角色。
- 來源建議：Wikimedia Commons — "Cladonia rangiferina reindeer" 或 iNaturalist
- 連結：https://commons.wikimedia.org/wiki/Cladonia_rangiferina
- 授權：CC BY-SA（依個別檔案確認）
- 展示時機：「在遙遠的北方有些地衣被凍了好幾個月」段落，讓孩子感受地衣跨越南北極端環境的生命力

### 6. 演替示意圖：裸岩 → 地衣 → 苔蘚 → 植物

- 標題：生命的接力——從裸岩到森林
- 描述：生態演替四階段示意圖：第一格是光禿禿的岩石，第二格是地衣開始覆蓋（橙黃色鱗片），第三格出現苔蘚（深綠絨毛），第四格已有草本與灌木站立。圖中可加入時間軸（數十年至數百年），強調地衣是「第一步踏出的先行者」，為後來的生命準備土壤。
- 來源建議：Wikimedia Commons — "Primary succession diagram" 或 NPS 教育素材
- 連結：https://commons.wikimedia.org/wiki/File:Primary_succession.svg
- 授權：CC BY-SA（示意圖）
- 展示時機：課程後段「先行者的意義」，讓孩子體會地衣在生態中的角色不只是自己活著，而是為別人開路

### 7. 城市樹木與地衣：空氣品質的活指標

- 標題：樹上的空氣品質計——地衣還在嗎？
- 描述：兩張對比圖：一張是乾淨空氣山區的老樹，樹幹上佈滿各種地衣；一張是都市行道樹的光滑樹幹，幾乎看不到地衣。對比鮮明，無需文字說明已能傳達地衣對空氣污染的高度敏感性。NPS 的生物指標教學中有完整的說明框架。
- 來源建議：NPS（美國國家公園署）生物指標教育頁面，或 iNaturalist 對比搜尋
- 連結：https://www.nps.gov/articles/lichens-as-bioindicators.htm
- 授權：美國政府公共版權（Public Domain）
- 展示時機：課程尾聲「地衣告訴我們什麼」，引導孩子思考：我們的學校附近，地衣願意住嗎？

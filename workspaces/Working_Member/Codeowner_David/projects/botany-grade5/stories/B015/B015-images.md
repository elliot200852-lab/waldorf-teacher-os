# B015 毛蟲與蝴蝶——投影圖畫清單

## 圖片資源

### 1. 柑橘鳳蝶卵——生命的起點

- **描述**：柑橘類葉片上的圓球形黃色卵粒。故事開場時展示，讓孩子看見「一切從一顆小小的、不起眼的卵開始」。建議搭配實物觀察——若校園有柚子或金棗樹，課前帶孩子到樹下找卵。
- **來源**：亞泥生態園區
- **URL**：https://accpark.org/butterfly.php?id=28
- **授權**：教育用途引用（網站公開教育資源）
- **展示時機**：故事開場——「在一片柑橘葉子上，有一顆比芝麻還小的金黃色珠子……」

### 2. 柑橘鳳蝶幼蟲（含臭角外翻）

- **描述**：五齡幼蟲翠綠色體態，背上黑色眼狀斑清晰可見，橙黃色臭角外翻伸出。這張圖同時呈現兩個教學重點：（一）從鳥糞色到翠綠色的變態——毛毛蟲已經在改變自己了；（二）臭角像蛇的分叉舌頭，釋放柑橘味驅趕天敵——大自然賦予弱小者的智慧。建議放大投影，讓孩子仔細觀察眼狀斑的「假眼睛」。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Papilio_xuthus_(caterpillar)
- **授權**：Creative Commons
- **展示時機**：故事中段——描述毛毛蟲「渴望飛向太陽，卻沒有翅膀」的段落，讓孩子看見這條蟲的真實樣貌，感受 Kovacs 所說「並不美麗的生物」。

### 3. 柑橘鳳蝶蛹——黑暗中的等待

- **描述**：縊蛹姿態，絲帶環繞胸部固定於枝條上。建議同時展示綠色型與褐色型兩種蛹色，引導孩子發現「蛹會根據周圍環境改變自己的顏色」。蛹是故事的情感核心——Kovacs 所說的「死去的空殼」，毛毛蟲交出整個身體，在黑暗中靜靜等待。展示時放慢節奏，給孩子沉默感受的空間。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Papilio_xuthus_(pupae)
- **授權**：Creative Commons
- **展示時機**：故事轉折處——「它用自己身體吐出的細絲，纏繞在陽光上編織……然後，毛毛蟲死了。」停頓，安靜展示蛹的圖片。

### 4. 柑橘鳳蝶成蝶——光之子升起

- **描述**：展翅的柑橘鳳蝶成蝶，黑褐色底上黃色放射狀條紋與後翅橙黃色斑點清晰可見，細長尾突優雅。這是故事的高潮圖像——「從死去的空殼裡，蝴蝶，那美麗的、有翅膀的光之子，升起了。」建議用全螢幕展示，讓色彩的衝擊力自己說話。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Papilio_xuthus
- **授權**：Creative Commons
- **展示時機**：故事高潮——描述蝴蝶從蛹中羽化的段落。在蛹的沉默之後，翻到這張圖，不需要多說什麼。

### 5. 柑橘鳳蝶生活史影片（選用）

- **描述**：柑橘鳳蝶活動影片，可見飛行姿態與訪花行為。若時間允許，在故事結束後播放短片段，讓孩子從靜態圖像回到活生生的生命。影片中緩慢優雅的飛行，正好呼應「光之子」的意象。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/File:Asian_Swallowtail_(Papilio_xuthus).webm
- **授權**：Creative Commons
- **展示時機**：故事結束後的延伸——讓孩子安靜觀看，不需解說。

# B020 高麗菜（十字花科）— 素材研究

## 一、Kovacs 章節摘要

### 核心教學意象
Kovacs 以「鋼琴彈奏」比喻植物將力量送往不同部位：同一種植物可以讓某個部分「大聲」（更強壯），其他部分「小聲」（較弱）。野生甘藍（Brassica oleracea）就是這樣的「藝術家」——在人類（園丁與農夫）的幫助下，它把力量分別送往根、莖、葉、花、芽眼，演變出五種截然不同的蔬菜。

### 說故事手法
- **偽裝（disguise）比喻**：五種蔬菜是同一種植物的五套「偽裝」，讓孩子猜不到它們是同一家族。
- **四芒星／十字**：十字花科不遵循五角星或六角星，而是跟隨四角星——葉子排列成十字形。
- **力量的流動**：根（蕪菁）→ 莖（球莖甘藍）→ 葉（高麗菜）→ 花（花椰菜）→ 芽眼（球芽甘藍），每次力量流向不同部位就產生不同的「角色」。

### 科學事實
- 蕪菁（turnip）：力量全進根部
- 球莖甘藍（kohlrabi）：力量全進莖部
- 高麗菜（cabbage）：力量全進葉部，葉子層層包裹成球
- 花椰菜（cauliflower）：力量全進花部（花托），園丁折葉遮光使花托潔白柔軟
- 球芽甘藍（brussel sprouts）：力量進入芽眼，每個芽眼長出一顆迷你高麗菜
- 五種蔬菜皆源自野生甘藍（wild cabbage），原生於溫暖地區

## 二、台灣植物資料

### 1. 高麗菜（甘藍）在台灣

**歷史與產區**
- 台灣位處亞熱帶，夏季高溫。民國 49 年（1960）中橫公路修通後，農民開始利用海拔 1,000-2,000 公尺的高山氣候種植高冷蔬菜。
- 1950 年代開拓中橫公路時，為補給修路榮民糧食，在梨山開闢武陵、清境（見晴）、福壽山、西寶等農場，試種溫帶蔬果。
- 宜蘭大同、台中和平（梨山）、南投仁愛（清境）為台灣高麗菜前三大產區，佔全國總產量三至四成。

**品質特性**
- 高山日夜溫差大，高麗菜能儲存更多醣分，口感特別甜美清脆。
- 平地高麗菜與高山高麗菜口感差異明顯——高山菜葉薄、纖維細、甜度高。

**環境議題**
- 高山農業開墾涉及水土保持問題，「夏天的高麗菜是山林血肉換來的」曾引發社會討論。

### 2. 芥菜與客家醃漬文化

**芥菜的一生三變**
客家傳統中，一株芥菜可以變化出三種截然不同的醃漬品，展現客家人「愛物惜物」的生活智慧：

- **酸菜（鹹菜）**：新鮮芥菜日曬 1-3 天變軟 → 一層菜一層鹽鋪排 → 人用腳踩軟讓鹽滲入 → 石頭重壓出水 → 入甕密封 → 發酵 1-2 週產生酸味。
- **福菜（覆菜）**：酸菜再日曬風乾 → 未完全乾燥時撕成長條塞入瓶中 → 瓶子倒置讓水分流出 → 密封瓶口放正 → 發酵 4-6 個月。名為「覆菜」因瓶子要「覆」（倒放），又稱「福菜」取吉祥意。
- **梅乾菜（鹹菜乾）**：醃製福菜剩下的嫩葉繼續日曬至完全乾燥 → 捲成圓球狀保存。

**文化意義**
芥菜、酸菜、福菜、梅乾菜「本是同根生」，差別在醃製過程。這種層層轉化的傳統完整展現客家人勤奮節儉的個性。新竹關西鎮為重要的鹹菜產地，設有「鹹菜故事館」。

### 3. 蘿蔔與台灣飲食文化

**菜頭與好彩頭**
- 台語稱蘿蔔為「菜頭」（tshài-thâu），與「彩頭」諧音。
- 台灣民間凡有喜慶吉事，必備蘿蔔供作祭品，用以求取「好彩頭」。
- 早期農家秋收後在稻田裡種蘿蔔，也在番薯田的畦溝寄種。

**菜頭粿（蘿蔔糕）**
- 台灣傳統米食，使用儲放一年以上的老米磨漿，加入蘿蔔絲蒸炊而成。
- 客家蘿蔔糕用料簡單，以蘿蔔本身清甜為主體，沾醬油或客家桔醬食用。

**菜脯（蘿蔔乾）**
- 蘿蔔切片日曬、鹽醃製成菜脯，是台灣家庭常備的醃漬品。
- 「老菜脯」陳年越久越珍貴，可燉湯。

### 4. 十字花科在台灣

台灣常見的十字花科蔬菜包括：高麗菜、花椰菜、青花菜（西蘭花）、白蘿蔔、芥菜、芥藍、大白菜、小白菜、油菜等。十字花科是台灣最重要的蔬菜科別之一，幾乎每個台灣家庭的餐桌上都少不了它們的身影。

## 三、台灣俗諺

1. **「好菜頭，好彩頭」**——台語諺語，蘿蔔（菜頭）諧音「彩頭」，象徵好運氣。過年祭祀必備。
2. **「芥菜命，愈壓愈出頭」**——形容芥菜在醃漬過程中越受壓越發酵出味，比喻堅韌不屈的精神。

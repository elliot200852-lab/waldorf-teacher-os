# B017 種子與子葉

## 故事本文

孩子們，上一次我們看見球莖埋在泥土底下，安安靜靜地儲存力量。今天，我們要問一個更古老的問題：一顆種子，怎麼知道該醒來了？

你把種子放在手掌上——硬硬的、乾乾的、一動也不動。它像一本還沒有翻開的小書。根要往哪裡長，葉子長什麼形狀，花開什麼顏色——全部寫好了，可是書還合著。誰來翻開它呢？太陽給溫暖，雨水讓它脹起來像海綿，大地把它抱在懷裡，空氣對它輕輕呼吸。四個朋友一起用力，「啪」——小書翻開了。

種子裂開，根先往下鑽——總是先往下，先抓住大地，芽才敢往上長。可是芽剛從土裡探出頭的時候，真正的葉子還沒準備好。誰先出來？子葉。子葉就像國王出巡前跑在最前面的先鋒官，大聲喊著：「國王來了！」——路清好了，真正的葉子才慢慢展開。

有些植物派兩個先鋒官。你們家一定泡過綠豆湯吧？仔細看——綠豆的殼裂開，伸出兩片胖胖圓圓的子葉，像兩隻小手合掌，然後慢慢打開，中間長出真正的葉子。這種兩片子葉的植物，葉脈像一張網，交叉再交叉。台灣山上有一種植物叫紅藜——排灣族世世代代種它，種在小米田旁邊。紅藜長得高高的，穗子紅橙黃紫，像一排彩色旗子。鳥飛去吃紅藜，小米反而安全了。排灣族人叫它「穀物的守護者」——兩個先鋒官派出來，守護整片田。

有些植物只派一個先鋒官。還記得鬱金香嗎？它發芽時只冒出一片窄窄的、像劍一樣的子葉。一個先鋒官，非常專注。這類植物的葉脈一條一條平行排列，像琴弦一樣整齊。布農族種小米種了幾千年，一年十四個祭典，從播種到收割，每個階段都有歌、有祈禱。最有名的「八部合音」，八個聲部從低到高，像穗子一節節向上生長，唱到最高的音，好像穗尖碰到了天。小米就是一片子葉的植物——簡單裡面有巨大的力量。

古時候波斯的農人知道一個秘密：滿月前三天播種，種子長得最好。月亮把水往上拉，種子裡的水也跟著漲，就在那個漲滿的時刻，小書翻開了。今天很多人忘了，可是月亮還記得。

兩個先鋒官的植物，葉脈像網，複雜而豐富。一個先鋒官的植物，葉脈像弦，簡潔而專注。可是不管幾個先鋒官，每顆種子都在等同一件事——等太陽、雨水、大地和空氣一起來翻開那本書。

下一次，我們要認識一朵花——美得讓人想靠近，可是莖上長滿了刺。美麗和保護，怎麼會長在同一棵植物上？

## 教師備註

1. **引導觀察**：課前三天請學生在濕棉花上放綠豆，每天記錄變化。上課時帶來觀察成果，讓學生親眼看見子葉展開的過程。
2. **單雙子葉對比**：可準備綠豆（雙子葉）與小米或稻穀（單子葉）的發芽標本，讓學生直接比較子葉數量與葉脈形態。
3. **先鋒官意象**：Kovacs 用 herald（先鋒官）比喻子葉，這個意象對五年級學生很有效——它賦予子葉一個角色與任務，而非僅是解剖名詞。
4. **文化連結**：紅藜與布農族小米的故事可以延伸討論「人與植物的關係不只是吃」，為後續課程鋪路。布農族八部合音如有音檔可在課堂播放，讓學生感受「從大地到天空」的聲音意象。
5. **月亮播種**：此為 Kovacs 原書內容（古波斯農法）。可視班級狀況決定是否深入，重點在於「自然的節奏」而非迷信。台灣農諺「清明田，穀雨豆」可作為呼應。
6. **本課核心圖像**：種子 = 未翻開的書；子葉 = 先鋒官；雙子葉 = 網狀脈 = 複雜；單子葉 = 平行脈 = 簡潔。

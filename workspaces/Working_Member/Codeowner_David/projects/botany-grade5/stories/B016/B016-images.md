# B016 鬱金香——投影圖畫清單

## 圖片資源

### 1. 鬱金香球莖剖面——黑暗裡的秘密

- **描述**：鬱金香球莖的縱切面，清晰可見外層乾燥膜質鱗被、內層白色肉質鱗片層層包裹，以及正中央藏著的小小花芽——明年的整棵鬱金香已經在裡面了。這是本課最核心的圖像：看不見的地方藏著最大的力量。投影時引導孩子從外往內一層一層看，最後聚焦到中央那顆芽，讓他們自己發現「整棵植物的未來就在那裡」。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Tulip_bulbs
- **授權**：Creative Commons
- **展示時機**：故事前段——「你把鬱金香的球莖剝開來看，外面是一層乾乾硬硬的皮，裡面一層一層白白的鱗片……最神奇的是——鱗片的正中央，藏著一顆小小的芽，那就是明年的鬱金香。」

### 2. 台灣百合盛開——從海邊到高山的白色號角

- **描述**：台灣百合（Lilium formosanum）全株盛開的姿態，雪白喇叭形花冠外側帶酒紅色條紋，莖直立挺拔。建議選擇野外自然環境的照片（岩壁或草坡上），讓孩子感受這朵花「從海邊礁石到三千公尺高山都能活」的強韌生命力。可以先問孩子：「你覺得這朵花的力量是從哪裡來的？」再連結回鱗莖的概念。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Lilium_formosanum
- **授權**：Creative Commons
- **展示時機**：故事中段——「春天，一根莖直直地抽出來，頂端開出雪白的喇叭花，花瓣外側帶著酒紅色的條紋。」

### 3. 薑的根莖——躺在土裡慢慢變強

- **描述**：新鮮薑根莖（Zingiber officinale），展示其橫向一節一節延伸的形態，節上可見新芽萌發點。與球莖的「圓形、層疊」形成對比——薑是「橫著長的，一節一節地往旁邊延伸，像一隻手伸出很多手指」。建議帶一塊實際的薑到教室，讓孩子用手觸摸、用鼻子聞，圖片作為輔助。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Zingiber_officinale
- **授權**：Creative Commons
- **展示時機**：故事中段——「把薑挖出來看看——它不是圓的，是橫著長的，一節一節地往旁邊延伸，像一隻手伸出很多手指。」

### 4. 芋頭球莖與葉——實心的力量

- **描述**：芋頭（Colocasia esculenta）完整植株，可見地下實心球莖與地上巨大的心形（盾形）葉片。選擇能同時看到球莖和葉子的圖片，讓孩子理解「從那團厚實的球莖裡，長出一片巨大的葉子」的對比——地底下沉重結實，地面上舒展輕盈。也可與百合鱗莖（層狀）和薑根莖（節狀）做三種地下儲存器官的形態比較。
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Colocasia_esculenta
- **授權**：Creative Commons
- **展示時機**：故事後段——「芋頭的球莖不像百合那樣一片一片，也不像薑那樣一節一節。它是實心的，整整一大塊……從那團厚實的球莖裡，長出一片巨大的葉子，形狀像愛心。」

### 5. 蘭嶼達悟族拼板舟與芋田——土地的尊嚴

- **描述**：蘭嶼達悟族傳統拼板舟或芋田的影像，展現芋頭在達悟文化中的核心地位。這張圖不是生物學圖片，而是文化圖像——讓孩子感受「芋頭不只是食物，是一整個家族的尊嚴」。若能找到拼板舟下水禮堆滿芋頭的場景最佳；否則可用芋田景觀搭配口述。展示時放慢節奏，讓故事的人文層次沉澱下來。
- **來源**：上下游新聞市集
- **URL**：https://www.newsmarket.com.tw/titv/taro/
- **授權**：教育用途引用（媒體公開報導）
- **展示時機**：故事尾聲——「蘭嶼的達悟族人種芋頭，不只是為了吃。他們造好拼板舟，要下水的那一天，會把芋頭堆滿整條船——那是他們對大海說的話。」

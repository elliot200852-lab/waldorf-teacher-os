# 慈心華德福高中

## 「這座島嶼上的人」

*九年級臺灣文學主課程*

授課教師：林信宏 ｜ 114 學年度春季 ｜ 第 8–9 週 ｜ 8 天 ｜ 每堂 120 分鐘

---

### 課程對象

九年級 D 班全體學生。

### 課程目標

1. **認識** — 透過文學作品，認識戰後臺灣社會的多元面貌：離散、白色恐怖、原住民、勞工、海洋、移工。
2. **閱讀** — 培養獨立閱讀能力，學會從作品中讀出時代與社會的訊息。
3. **書寫** — 嘗試多種書寫方式：短文、評論、詩歌、生活雜記、散文詩。
4. **思考** — 從文學出發，思考一個問題：**臺灣人究竟是什麼人？**

### 課程脈絡

八天的課程沿著一條弧線展開 — 從離散到扎根，從沉默到發聲，最後回到「我是誰」：

| Day 1 | → | Day 2 | → | Day 3 | → | Day 4 | → | Day 5 | → | Day 6 | → | Day 7 | → | Day 8 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **入口** | | **漂流** | | **沉默** | | **根** | | **汗** | | **海** | | **我** | | **回音** |
| 走進故事 | | 外省離散 | | 白色恐怖 | | 原住民 | | 勞工 | | 海洋與移工 | | 誰是臺灣人 | | 回顧分享 |

每天閱讀一篇臺灣文學作品（小說、散文或詩），透過討論與書寫，走進不同族群、不同世代在這座島嶼上的故事。第八天不讀新文本，全班回顧、分享、反思。

### 每堂課的節奏

| 階段 | 大約時間 | 你會做什麼 |
|------|---------|----------|
| 開場 | 5–10 分鐘 | 安靜下來，準備進入文學的世界 |
| 回顧 | 15–20 分鐘 | 重述昨天的故事，討論、分享想法 |
| 新文本 | 40–50 分鐘 | 閱讀今天的作品，小組討論，認識作者與時代 |
| 寫作 | 25–30 分鐘 | 學習一個寫作技法，完成當日書寫練習 |
| 收束 | 5–10 分鐘 | 預告明天的主題，領取明天的閱讀文本 |

### 書寫練習

每天會有一次書寫練習，寫在工作本上。七天嘗試七種不同的書寫方式：

- **200 字短文** — 用畫面說話，不直說感受
- **FB 留言式評論** — 用最自然的語言回應作品
- **詩歌創作** — 把一個感受濃縮成幾行文字
- **個人生活雜記** — 連結自己的生活經驗
- **200 字短評** — 對一個議題表達看法
- **散文詩** — 用最少的字，說最準的感受

### 評量方式

你的成績由三個部分組成：

| 項目 | 比重 | 說明 |
|------|------|------|
| 課堂參與 | 30% | 在討論中發言、朗讀分享、小組互動。不需要「正確答案」，重要的是你願意參與。 |
| 工作本 | 50% | 七次書寫練習的完成度與深度。評量重點：是否認真嘗試，是否有自己的想法，不評文筆優劣。 |
| 期末反思 | 20% | 第八天（回顧日）的書寫：寫下三個你從這門課帶走的想法。 |

### 你需要準備的東西

- 一本空白工作本（主課程本），用來寫所有的書寫練習
- 筆（鋼筆或原子筆皆可）、彩色鉛筆（美化工作本用）
- 一顆願意慢下來、好好讀一篇故事的心

# Day 2 教案 — 漂流：誰不是外來者？

## 基本資訊

| 項目 | 內容 |
|------|------|
| 日期 | 第 8 週・週三 |
| 時長 | 120 分鐘 |
| 主題 | 外省人的離散 — 一個 11 歲男孩被帶上軍艦 |
| 主文本 | 〈岸與岸〉桑品載 |
| 寫作技法 | 敘事觀點（第一人稱回憶）+ 角色視角書寫 |
| 書寫練習 | 角色視角書寫（100-150 字）|

---

## 構想

第二天進入臺灣歷史的實質內容。〈岸與岸〉是真人真事 — 桑品載 11 歲那年，他姊姊被一個國軍連長看上，母親在一念之間決定讓姊弟倆上軍艦，從舟山到臺灣。這不是小說，是一個人對自己人生轉折的回憶。九年級學生追求「真實」，真人真事比虛構更有力。

**差異化的核心問題**：同一篇文本，有些學生讀完只看到「一個小孩搭船去了臺灣」，有些學生讀完會被母親在燈下縫銀元布袋的畫面打動。差異不是閱讀速度的問題，而是**感受力的落差**。Day 2 的 DI 設計不只分閱讀量，更要透過討論結構幫助不同感受力的學生都能進入故事。

**與 Day 1 的銜接**：昨天學生學到「用畫面寫心情」（意象），今天延伸到「用角色的眼睛看故事」（敘事觀點）。昨天的 200 字書寫練習是今天回顧的起點 — 從自己的書寫經驗出發，再進入別人的真實故事。

---

## 差異化設計（本堂核心）

### 學生分群概況（依文學感受力，非固定標籤）

| 群 | 特徵 | 本堂需要 |
|----|------|----------|
| A | 閱讀快、文字敏感，讀完會自己想很多 | 給空間去深想，別限制他們；討論時讓他們帶組 |
| B | 閱讀能力不錯但不太主動表達 | 結構化的討論任務逼他們開口；小組角色分配 |
| C | 閱讀速度慢或回家沒讀完 | 精讀重點段落 + 教師標記；角色書寫的鷹架引導 |
| D | 對文字不敏感，覺得「這跟我有什麼關係」 | 從生活經驗切入（你有沒有跟家人分開過？），不從文本分析切入 |

### DI 落實在每個環節

下方教學步驟中，每個有 DI 分層的環節都會用 **[DI]** 標記。

---

## 教學步驟

### 一、回顧與暖身（25 分鐘）

#### 1. 回顧 Day 1 書寫經驗（12 分鐘） **[DI — 全班暖身]**

先從昨天的書寫練習開始，讓學生回到自己的文字裡。

- 「昨天你們每個人寫了 200 字的畫面。現在翻開你的工作本，安靜讀一次自己寫的東西。」（2 分鐘安靜讀）
- 「等一下要跟旁邊的人分享。說三件事就好 — 」

**板書：**

```
1. 你寫的畫面是什麼？
2. 寫的時候，最深刻的感覺是什麼？
3. 有沒有卡住的地方？怎麼解決的？
```

- 兩人一組，每人 2 分鐘，互相分享。（4 分鐘）
- 全班收幾個分享：「有誰願意說說看，你昨天寫的那個畫面？」收 2-3 位。（4 分鐘）

教師收束：
- 「昨天你們練習的是**用畫面說話**。今天我們要讀的這個人，桑品載，他寫的也是畫面 — 但不是想像的，是真的。他在回憶自己 11 歲那年發生的事。」

#### 2. 回顧〈水上組曲〉+ 銜接（5 分鐘）

快速帶過，不需要花太多時間：

- 「昨天〈水上組曲〉讀完了嗎？誰可以用一句話說，船夫最後怎麼了？」
- 接 1-2 個學生回答。
- 「鄭清文整篇沒有寫一個『愛』字，但你讀完就知道了。他用**意象**說話。今天桑品載用的是另一種方式 — 他用**回憶**說話。一個六十幾歲的人回頭看 11 歲的自己。」

#### 3. 精讀時間（8 分鐘） **[DI — C/D 組安全網]**

〈岸與岸〉的文稿昨天已經發放，學生應該都讀過了。這段時間的重點不是「補讀」，而是**精讀核心段落**。

- 「昨天回家都讀了〈岸與岸〉。現在把文本拿出來，**重讀我等一下唸的這三個段落的位置**。就算你昨天已經讀過了，再讀一次，注意細節。」

**教師事先標記三個精讀段落（發文本時已用螢光筆或夾書籤）：**

| 段落      | 位置                    | 為什麼選這段                    |
| ------- | --------------------- | ------------------------- |
| 母親縫布袋   | 母親丟下碗筷進房間，取出銀元和金戒指... | 整篇的情感核心。母親流淚、姊姊哭、「我」哭不出來。 |
| 上船 / 碼頭 | 蕭連長派兵來接 → 離開家門的那一刻    | 離別的具象畫面。                  |
| 基隆睡水泥椅  | 到了基隆，睡在中正堂外的水泥椅上      | 11 歲的孩子到了臺灣之後的真實處境。       |

- 「如果昨天真的來不及讀完全文的，**先讀這三段就好**。等一下聽同學講也可以跟上。」

→ 多數學生是重讀精讀，沒讀完的學生也有一個安靜的安全網，不會被點出來。

---

### 二、進入〈岸與岸〉（50 分鐘）

#### 1. 教師鋪墊（5 分鐘）

不長篇大論，只給最必要的背景：

- 「1949 到 1950 年，中國內戰打完了，國民黨輸了。超過一百萬軍人和老百姓跟著撤退到臺灣。」
- 「桑品載那時候 11 歲，住在舟山群島。」（拿出地圖指一下，或畫在黑板上）「他不是軍人，他只是一個小孩。」
- 「這篇文章是他六十幾歲的時候回頭寫的 — 回憶自己 11 歲那年發生的事。」

#### 2. 教師朗讀核心段落（10 分鐘）

朗讀「母親縫布袋」那一整段（從「母親丟下碗筷進她房間」到「男兒志氣在四方」）。

朗讀提示：
- 「她一邊縫一邊流淚，淚流多了就用手背去擦」— 這句放慢
- 數銀元的時候一個一個數，讓教室安靜下來
- 「男兒志氣在四方」— 母親的聲音，帶著不得已

讀完停幾秒。不說話。

#### 3. 小組討論（25 分鐘） **[DI — 核心分層環節]**

**分組方式**：4 人一組，教師事先安排混合分組（每組至少一個讀完全文的學生）。

**討論分兩輪：**

**第一輪（5 分鐘）— 書寫經驗分享：**

先從昨天的書寫經驗延伸到今天的文本。教師口述引導：
- 「昨天你們寫的是自己記得的畫面。桑品載寫的也是他記得的畫面 — 但他記了五十年。在你的組裡，先聊一下：讀完〈岸與岸〉，哪一個畫面留在你心裡最深？每個人說一個就好。」

**第二輪（20 分鐘）— 討論卡：**

每組發一張討論卡，上面有三層問題。學生可以從裡面**選題討論**，不必按順序。

```
+---------------------------------------------+
|  〈岸與岸〉討論卡                              |
|                                               |
|  【每個人都要回答一題】                         |
|                                               |
|  1. 母親為什麼答應讓蕭連長帶走姊姊和弟弟？       |
|     她那天晚上在想什麼？                        |
|                                               |
|  2. 如果你是那個 11 歲的桑品載，               |
|     離開家門的那一刻，你心裡會想什麼？            |
|     （提示：你有沒有離開過家人一段時間的          |
|     經驗？去農場實習？戶外挑戰？                 |
|     那種感覺跟桑品載的遭遇一樣嗎？）                     |
|                                               |
|  3. 桑品載說他「默默在數，有五十四個」—          |
|     為什麼一個 11 歲的孩子會記住                |
|     銀元的數目？你覺得那一刻他在想什麼？          |
|                                               |
|  【讀完全文的人多選一題】                        |
|                                               |
|  4. 桑品載到基隆之後，過的是什麼樣的日子？        |
|     從一個有家的孩子，變成睡在水泥椅上的人        |
|     — 這中間發生了什麼？                        |
|                                               |
|  5. 他說「母親改變了我的一生」—                  |
|     你覺得這句話裡面有感謝嗎？有遺憾嗎？         |
|     還是更複雜的東西？                           |
|                                               |
|  【想更深的人挑戰這題】                          |
|                                               |
|  6. 五十年後桑品載回去找姊姊，兩人               |
|     「相視無言後抱首痛哭」。                     |
|     一個擁抱裡面裝了什麼？                       |
|     試著用兩三句話說出來。                        |
+---------------------------------------------+
```

**討論規則（板書）：**
- 每個人至少說一次
- 不用舉手，輪流講
- 沒有對錯，說你自己的想法

**教師巡迴重點：**
- 先到 C/D 比例高的組，確認他們有進入討論
- 如果有學生說「我不知道」，教師蹲下來問：「你有沒有跟家人分開過一段時間？去農場實習那次算嗎？離開家的那個感覺是什麼？」→ 從自身經驗引入
- **注意分寸**：學生的農場實習、戶外挑戰跟大時代的生離死別性質完全不同。不需要硬拉「你也有類似的經驗」，而是讓學生知道：你可以從自己有限的離別經驗出發，去**想像**桑品載那種更沉重的感覺。教師巡迴時如果自然帶到就好，不刻意。

#### 4. 全班分享（10 分鐘）

- 每組派一個人分享一個最有意思的回答（不限哪一題）
- 教師收束時帶出一個點：
  - 「這些人來到臺灣，很多人一輩子沒有回去。他們不是『外省人』這三個字，他們是一個一個活生生的人。有些人在基隆的水泥椅上睡了兩個月，有些人再也沒見過自己的媽媽。」
  - 「但他們後來怎麼樣了？有些人一直覺得自己是過客，有些人慢慢在這裡生了根。明天我們會讀一篇更沉重的 — 白色恐怖。一個 17 歲就在文學雜誌上發表作品的天才，因為買了幾本中文雜誌，被政府抓去關。」

---

### 三、寫作技法 + 書寫練習（35 分鐘）

#### 1. 寫作技法教學：從意象到觀點（8 分鐘）

**回顧 Day 1 的「意象」（2 分鐘）：**

- 「昨天學的是什麼？用畫面說話。鄭清文沒有寫一個『愛』字，但你讀完就知道船夫的心情。他用看得見的東西讓你自己去感覺。」

**引入「敘事觀點」（3 分鐘）：**

- 「桑品載寫這篇的時候已經六十幾歲了。他在回憶 11 歲的事。」
- 「注意他的寫法：他寫媽媽數銀元，他說『我默默在數，有五十四個』— 他記了一輩子。這是**第一人稱回憶**的力量：只有當事人才會記得那種細節。」
- 「他還有一個手法：**時間的距離**。六十歲的人回頭看 11 歲的自己，有些事看得更清楚了，有些事還是模糊的。這種距離感，讓故事產生了厚度。」

**帶入文學手法（3 分鐘）：**

- 「文學家寫回憶常用幾個手法 — 」

**板書：**

```
文學家的回憶手法：
1. 細節描寫 — 記住一個很小的東西（五十四個銀元）
2. 第一人稱 — 用「我」的眼睛看，讓讀者貼近
3. 時間距離 — 現在的我回頭看當時的我
4. 意象（昨天學的）— 用畫面承載感情
```

- 「等一下你寫的東西，要試試看把**手法**跟**真切的情感**配在一起。不是為了炫技，是讓你的文字更有力量。」

#### 2. 教師黑板示範（5 分鐘）

教師在黑板上示範一個「角色視角 + 文學手法 + 真情」的範例。

示範角色：以桑品載的母親的角度。

教師邊寫邊說：

> 「我來示範一個。假設我是桑品載的母親，我用第一人稱回憶的方式寫 — 」

**黑板示範：**

> 「那天晚上我把碗筷丟在桌上，進房間翻出銀元。五十四個。我一個一個數，手在發抖。縫進布袋的時候針腳歪了好幾次，但我不敢停，知道停下來我就會改變主意。我跟他說『男兒志氣在四方』。其實真正想說的我沒有說出口。」

教師問學生：
- 「你們看到幾個手法？」（細節描寫：五十四個、針腳歪了；第一人稱；意象：手在發抖）
- 「最重要的是最後那句 — 她說了一句話，但心裡想的是相反的。手法不是裝飾，是讓你把真的感覺表達出來。」

#### 3. 書寫（22 分鐘） **[DI — 寫作分層]**

**題目（板書）：**

```
角色書寫（150-300 字）

從下面的角色中選一個，用這個人的眼睛
寫一段話。用你今天學到的手法。
文字要精煉 — 不是聊天，是文學。

角色選項：
(1) 時代的觀察者 — 你是一個旁觀者，站在碼頭上
    看著這一幕。用冷靜的目光記錄你看到的。
(2) 一位母親 — 不一定是桑品載的媽媽，
    可以是任何一位送孩子上船的母親。
(3) 11 歲的桑品載 — 用他的眼睛，
    寫上船那一刻或到基隆之後的某個瞬間。
(4) 桑品載的姊姊 — 她被蕭連長看上，
    這件事她自己怎麼想？

選一個角色，寫一段話。
```

**分層引導：** **[DI]**

| 層 | 引導方式 |
|----|---------|
| **C/D** — 需要起手式和手法提示 | 教師在黑板上寫三個開頭讓學生選：<br>「我站在碼頭上，看見＿＿＿。」（觀察者）<br>「那天晚上，我把＿＿＿塞進他的口袋。」（母親）<br>「船開了。我回頭看，岸上＿＿＿。」（桑品載）<br>選一個接著寫。**至少用一個手法**（細節描寫或意象）。 |
| **B** — 有想法但文字容易鬆散 | 「選好角色之後，先想一個**具體的細節** — 一樣東西、一個聲音、一個動作。從那個細節開始寫。記住：精煉，不是聊天。每一句都有畫面。」 |
| **A** — 自由發揮 | 「試著在 150 字以內用上**至少兩個手法**（細節 + 意象，或第一人稱 + 時間距離）。如果你想挑戰：寫完之後，把最後一句話改成跟開頭相反的意思 — 像桑品載的母親說『男兒志氣在四方』，但心裡想的是別走。」 |

⚡ **時間提醒**：20 分鐘寫作 + 2 分鐘收尾。如果剩 5 分鐘還沒寫完，讓學生先收。100 字不多，重點是精煉。

---

### 四、收束（10 分鐘）

#### 1. 分享（7 分鐘）

- 「誰願意唸自己寫的？告訴我們你選的角色是誰。」
- 抽 3-4 位，盡量不同角色、不同層的學生（教師巡迴寫作時已經看過誰寫得好）
- 聽完每篇：「我聽到了。謝謝。」
- 如果有學生寫得特別精煉，多說一句：「你這句寫得很準，手法用對了。」
- 如果有不同角色的對比（比如母親和桑品載都有人寫），可以點出：「同一件事，兩個角度看出來完全不同。這就是觀點的力量。」

#### 2. 回家的事（3 分鐘）

發〈紅鞋子〉葉石濤（**教師標記重點段落版本**）。

- 「明天要讀的更重一點。一個 17 歲就在文學雜誌上發表作品的天才，因為戰後買了幾本中文雜誌，被政府抓去關。」
- 「全文很長，**先讀我標記的部分就好**。讀完標記的，有興趣再往下讀。」

**板書：**

```
回家：
1. 讀〈紅鞋子〉葉石濤 → 先讀標記段落
2. 明天要討論
```

## 課堂作業

| 項目 | 說明 |
|------|------|
| 書寫 | 角色視角書寫（100-150 字），寫在工作本 |
| 重點 | (1) 有沒有進入角色（不是用自己的口氣評論） (2) 有沒有用到至少一個文學手法 (3) 文字是否精煉（不是口語聊天） |
| DI 評量 | C/D 用了起手式且完成 = 達標；B 有具體細節且完成 = 良好；A 用了兩個手法且文字精煉 = 優秀 |

## 回家作業

| 任務 | 說明 |
|------|------|
| 讀〈紅鞋子〉葉石濤 | 先讀教師標記的重點段落（約 3-4 段），有餘力讀全文 |

## 教師備註

1. **回顧 Day 1 書寫經驗是今天的第一件事。** 不是可有可無的暖身，而是讓學生從自己的文字出發，再進入別人的故事。兩人分享不需要太正式，重點是讓每個人都開口說過一次。
2. **精讀時間的定位改了。** 不再是「給沒讀的人補讀」，而是全班一起精讀核心段落。多數學生昨天已經讀過，這時候是深化細節感受。真的沒讀完的學生也有安靜的空間讀三段，不會被點出來。
3. **討論卡有選題機制。** 學生可以從六題中選自己想談的，不必按順序。必答題（1-3）每個人都能回答；進階題（4-5）需要讀完全文；挑戰題（6）需要深度思考。教師不需要指定誰答哪題 — 學生自然會依能力選擇。
4. **學生經驗的引入要自然。** 9D 都有離開家人的經驗（農場實習、戶外挑戰），但這些跟大時代的生離死別性質不同。不要硬拉「你也經歷過一樣的事」，而是讓學生用自己有限的經驗去**想像**更沉重的感受。教師巡迴時自然帶到即可。
5. **書寫練習從 FB 留言改為角色視角書寫。** 不是隨意的社群留言，而是有角色定位、有文學手法要求的精煉書寫。C/D 學生有起手式和角色選項，A 學生有手法挑戰。教師黑板示範是關鍵 — 沒有示範直接寫，多數學生不知道「精煉」長什麼樣子。
6. **C/D 學生的起手式要寫在黑板上。** 不能只口述。寫在黑板上，他們隨時可以抬頭看。角色選項也要板書，不只放在口述裡。
7. **時間最容易超的是小組討論。** 討論兩輪合計 25 分鐘，到 20 分鐘時預告「再 5 分鐘收」。如果時間不夠，寫作可以縮到 18 分鐘（100 字不多）。
8. **明天的〈紅鞋子〉一定要事先標記段落。** 全文太長，不標記的話 C/D 學生回家打開直接放棄。標記四段核心場景，詳見下方「選讀段落建議」。

## 〈紅鞋子〉葉石濤 — Day 3 選讀段落建議

全文共 8 章，約 9,000 字。C/D 學生回家只讀以下四段即可參與 Day 3 討論：

| 段落 | 章節 | 起訖 | 約字數 | 為什麼選這段 |
|------|------|------|--------|-------------|
| **1. 買雜誌的起因** | 第 1 章「米街」 | 從「許尚智是我州立台南二中的後期同學」到「我邊吃冰邊想到日據時代台灣人的反帝、反封建運動」 | ~800 字 | 葉石濤 17 歲就發表作品的文學天才，光復後只是想學中文、買幾本書來讀。交代「罪行」的起點——不過是求知慾。 |
| **2. 被捕** | 第 6 章「紅鞋子」 | 從「一九五一年的秋天」到「簡阿淘禁不住眼淚滂沱的滴下來。只是他還搞不懂他為什麼被捕」 | ~1,200 字 | 整篇最戲劇性的轉折。看完電影《紅鞋子》回家，深夜被剃頭匠帶路的便衣逮捕。紅吉普車、手銬、夜寒料峭——具象畫面密度最高的段落。 |
| **3. 審問** | 第 7 章「審問」 | 從「他們來到一個小法庭」到「他再也沒有力氣站着而像一塊沒人要的磨布一般軟癱下來」 | ~1,000 字 | 特務少將的審問場景。因為買了兩本書就被指控為「資深叛徒」。荒謬與恐怖並存。少將朗讀莎士比亞《馬克白》的細節令人不寒而慄。 |
| **4. 判決書** | 第 8 章「判決」 | 從「一九五三年七月，他收到一張判決書」到文末判決書全文 | ~800 字 | 真實的判決書原文。「明知為匪諜而不告密檢舉處有期徒刑五年」——因為「看了書」就被判刑。最具衝擊力的史料。 |

**教師標記方式**：發放文本時，用螢光筆或夾書籤標出以上四段的起訖位置。A/B 學生建議讀全文（含辜雅琴、甘瑞月、本順哥三個人物的章節），這些人物豐富了「白色恐怖不只抓一個人」的理解。

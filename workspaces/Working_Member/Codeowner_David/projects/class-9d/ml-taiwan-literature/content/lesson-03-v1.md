# Day 3 教案 — 沉默：當說話變得危險

## 基本資訊

| 項目 | 內容 |
|------|------|
| 日期 | 第 8 週・週四 |
| 時長 | 120 分鐘 |
| 主題 | 白色恐怖 — 一個人因為讀書而被關進監獄 |
| 主文本 | 〈紅鞋子〉葉石濤（精讀選段） |
| 寫作技法 | 象徵 — 一件物品承載一整段歷史 |
| 書寫練習 | 詩歌創作（6-10 行）：「沉默中的一句話」 |

---

## 構想

Day 3 是第一週的情感核心。Day 1 給了學生進入故事的方法；Day 2 讓他們見到了一個被帶來臺灣的外省孩子。今天他們要見到一個**留在臺灣**的文學天才 — 他沒有離開，卻失去了自由。

**核心設計挑戰**：學生昨天帶了〈紅鞋子〉全文回家讀。有些人讀完八章，有些人只讀了標記的四段，有些人翻了幾頁就放下。Day 3 必須讓所有人都能參與。解法是**精讀** — 教師選三段核心段落（被捕、審問、判決），在課堂上朗讀帶領。讀完全文的學生會辨認出角色；只讀標記段落的學生已接觸過這些場景；幾乎沒讀的學生則透過教師的朗讀與全班討論取得必要的體驗。

**從 Day 2 到 Day 3 的轉折**：昨天的桑品載是一個被戰爭裹挾的外省孩子。今天的葉石濤是一個被同一個政權摧毀的臺灣知識份子。這個反諷不能由教師直接點破 — 它必須在討論中浮現。目標是複雜性，不是簡單的對立。被帶來的人和留下來的人都在受苦，加害的是制度。

**新技法 — 象徵**：Day 1 教意象（用畫面承載情感）；Day 2 教敘事觀點（用角色的眼睛看）。今天加一層：當一個具體的物品（紅鞋子、一本書、一副手銬）承載了整個時代的重量。這為學生的詩歌創作做準備 — 他們要用一個意象或物件，替一整個沉默的世代說話。

**David 的三個轉折點**（透過討論浮現，非講述）：
1. 從〈紅鞋子〉連結臺灣人與外省人的異同 — 兩者都是受害者，制度才是加害者
2. 葉石濤的文學地位 — 從囚犯到《臺灣文學史綱》作者
3. 臺灣主體性如何在文學作品與讀者之間經營

## 差異化設計（本堂核心）

### 學生分群概況（依文學感受力，非固定標籤）

| 群 | 特徵 | 本堂需要 |
|----|------|----------|
| A | 閱讀快、文字敏感，讀完會自己想很多 | 給空間去深想；討論時引導他們觸碰轉折點 2 和 3（文學地位、臺灣主體性） |
| B | 閱讀能力不錯但不太主動表達 | 結構化的討論問題，要求舉出文本證據 |
| C | 閱讀速度慢或回家沒讀完 | 課堂精讀提供核心場景；詩歌創作有結構引導 |
| D | 對文字不敏感，覺得「這跟我有什麼關係」 | 從內臟反應切入（深夜被銬手銬、紅吉普車、荒謬的審問），不從文學分析切入；生活經驗連結（你有沒有被冤枉過？） |

## 教學步驟

### 一、回顧與橋接（18 分鐘）

#### 1. Day 2 視角分享（10 分鐘） **[DI — 全班暖身]**

昨天學生從五個視角中選二進行書寫。讓他們的聲音先出現。

- 「昨天你們從不同的角色看同一個故事。現在翻開工作本，看一下你昨天寫的東西。」（1 分鐘安靜讀）
- 「誰來分享？告訴我們你選了哪個角色，念你寫的一句話就好。」

**板書：**

```
Day 2 回顧：
你選了哪個角色？寫了什麼？
```

- 收 4-5 位學生，盡量讓五個視角都有人代表（時代觀察者 / 桑品載的母親 / 另一位臺灣母親 / 姐姐 / 桑品載本人）
- 教師不評價，每人分享後：「我聽到了，謝謝。」
- 如果有不同視角的對比出現（比如母親和桑品載），教師可以點出：「同一條船，同一個夜晚 — 但兩雙完全不同的眼睛。這就是觀點的力量。」

#### 2. 橋接：從漂流到沉默（5 分鐘）

教師引導語（不是逐字稿 — 這些是要命中的節拍）：

- 「昨天我們讀的是一個**被帶到臺灣**的人 — 11 歲的桑品載，被戰爭裹挾上船。」
- 「今天我們要讀的是一個**留在臺灣**的人。一個台南的年輕人，在這裡出生、在這裡長大。他哪裡都沒去。」
- 「他只是想學會讀中文、寫中文。他去買了幾本書。就因為這樣，政府毀了他的人生。」
- 回扣昨天課堂上補充的國共內戰脈絡：「記得我們昨天聊到的 — 國民黨輸了內戰，帶著一百多萬人撤退到臺灣。那個帶桑品載一家來臺灣的政府，同時也在臺灣建立了一套恐怖的控制系統。任何人只要碰過不對的書、認識不對的人，都可能被抓。」
- 「今天的主題叫做『沉默』。當說話 — 甚至讀書 — 都可能讓你坐牢的時候，會發生什麼事？」

**板書（更新八天弧線，標記今天的位置）：**

```
Day 1 入口 → Day 2 漂流 → ★Day 3 沉默★ → Day 4 根 → ...
```

#### 3. 白色恐怖背景（3 分鐘）

極簡脈絡，不是講課。只給學生理解文本所需的最低限度：

- 「白色恐怖：1949 年到 1987 年，國民黨政府實施戒嚴。在這段期間，大約 14 萬人因為政治原因被逮捕。很多人被槍決，更多人被關了好幾年。」
- 「用來對付葉石濤的法條是這樣的：如果你**知道**某個人跟共產黨有關係，但你沒有去舉報，你自己就有罪。想一下 — **不做一件事**變成了犯罪。」
- 不在這裡講述 228。228 在文本第六章自然出現（「一九四七年三月發生的二二八事件」），讓文本自己完成這個工作。

### 二、精讀〈紅鞋子〉（38 分鐘）

#### 1. 葉石濤定位（3 分鐘）

教師引導語：

- 「葉石濤，16 歲就在日文文學雜誌上發表小說。他是台南的文學天才。」
- 「日本人走了、國民黨來了之後，他拼命想學中文。他是作家 — 當然想掌握新的國語。」
- 「在這篇文章裡，他給自己取了一個筆名：簡阿淘。你讀到的所有事情，都是一個真實的人的遭遇。」

**板書：**

```
葉石濤 = 簡阿淘
16 歲發表第一篇小說
光復後：拼命學中文、買書來讀
```

#### 2. 精讀三段落（25 分鐘） **[DI — 核心閱讀環節]**

學生拿出〈紅鞋子〉文本。教師宣布：「就算你回家已經讀完全文，我們現在要一起**非常仔細地**讀三個段落。這是一切改變的時刻。」

**段落一：被捕（第六章，約 1,200 字） — 教師朗讀，8 分鐘**

從「一九五一年的秋天」到「只是他還搞不懂他為什麼被捕」。

朗讀要領：
- 「他看到平常在學校替小孩理髮的剃頭匠楊雙冬那傢伙站在那兒」— 平淡地讀，一切看起來還很正常
- 「先搜他身體！」— 突然的命令，尖銳
- 「不知從哪兒拿出來的，不容他分辯，就把他雙手用手銬銬住了」— 放慢速度
- 「紅色的吉普車在深夜的『銀座通』大路靜靜地停着」— 讓紅吉普車的意象懸在空氣中
- 「簡阿淘禁不住眼淚滂沱的滴下來。只是他還搞不懂他為什麼被捕。」— 最後一句讀完，停五秒鐘，不說話

⚡ 補充情境（朗讀前簡短帶入）：「這一段發生在 1951 年秋天。距離光復已經六年了。就在那天晚上，阿淘去看了一部英國電影——《紅鞋子》，講一個芭蕾舞星在愛與藝術之間掙扎，最後穿著紅鞋跳樓自盡。他看完電影，淚眼朦朧地回家。然後——」開始朗讀被捕場景。

**段落二：審問（第七章，約 1,000 字） — 教師朗讀，7 分鐘**

從「他們來到一個小法庭」到「像一塊沒人要的磨布一般軟癱下來」。

朗讀要領：
- 特務少將朗讀莎士比亞《馬克白》的段落 — 讀得優雅，帶著反諷的矛盾感
- 「你想起了你叛國的事實了嗎？」— 語氣突然轉冷
- 荒謬的升級：「你既然看了這些書，你就是叛徒！」
- 「資深叛徒」— 少將自創的詞彙，帶著自鳴得意
- 最後的意象：阿淘癱倒「像一塊沒人要的磨布」

讀完，短暫停頓。教師只問一個問題：
- **「簡阿淘到底做了什麼？」**
- 等學生回答。期待的回應：他買了兩本書。就這樣。

**段落三：判決書（第八章，約 800 字） — 教師朗讀，5 分鐘**

從「一九五三年七月，他收到一張判決書」到判決書全文結束。

朗讀要領：
- 判決書的官僚文字要讀得平板、正式。不帶感情。
- 「簡阿淘明知爲匪諜而不告密檢舉處有期徒刑五年」— 平平地讀出判決。
- 讀完：「五年。因為買了兩本書，因為認識幾個人。」

**板書（三段精讀後）：**

```
〈紅鞋子〉— 三個時刻

1. 被捕：深夜、紅吉普車、手銬
   「他還搞不懂他為什麼被捕」

2. 審問：少將讀莎士比亞，叫他「資深叛徒」
   罪行 = 買了兩本書

3. 判決：有期徒刑五年
   「明知為匪諜而不告密檢舉」
```

#### 3. 討論（10 分鐘） **[DI — 分層討論]**

全班討論，教師用問題引導三個轉折點。不需要全部問完 — 看教室的氣氛走。

**第一層（所有人都能回答）：**

- 「這三段裡面，哪一個畫面留在你心裡最深？」（開放題，沒有錯的答案）
- 「少將一邊讀莎士比亞一邊審問人 — 這是什麼樣的人？」
- 「阿淘被捕前看了一部電影叫《紅鞋子》，講一個芭蕾舞星為藝術而死。葉石濤為什麼用這個當書名？」

**第二層（引向轉折點 1 — 臺灣人與外省人的複雜性）：**

- 「昨天桑品載是被帶來臺灣的外省孩子。今天葉石濤是被臺灣的政府抓起來的臺灣人。這兩個故事是對立的嗎？還是更複雜？」
- 如果需要引導：「想想前五章出現的那些人 — 阿才伯、辜雅琴、甘瑞月、本順哥。他們想要什麼？他們是政治激進份子，還是只是有夢想的普通人？」
- 如果有學生提到 228：「很好。文本裡提到了 228。228 之後阿淘變得謹慎 — 他待在家裡讀書寫文章。他在**躲避**麻煩。結果他們還是來抓他了。」

**第三層（引向轉折點 2 和 3 — 葉石濤的文學地位、臺灣主體性）：**

- 「阿淘坐了五年牢。出獄以後沒有人敢靠近他。找不到工作，流浪了好幾年。但是幾十年後，他做了一件了不起的事。有人知道嗎？」
  - 如果沒有人知道，教師揭示：「1987 年，他出版了《臺灣文學史綱》— 這是臺灣人自己寫的**第一部**臺灣文學史。」
- 「想想這件事：一個因為**讀書**而被懲罰的人，最後成為**定義臺灣文學是什麼**的那個人。這告訴你什麼？」

**板書（討論後增加）：**

```
葉石濤的路：
16 歲 → 發表第一篇小說
      ↓
買書、學中文
      ↓
被捕 → 五年牢獄
      ↓
出獄 → 無人敢接近
      ↓
1987 →《臺灣文學史綱》
      臺灣人自己寫的第一部臺灣文學史
```

### 三、象徵技法 + 詩歌創作（30 分鐘）

#### 1. 象徵技法教學（8 分鐘） **[DI — 技法教學]**

**回顧 Day 1 和 Day 2 技法（2 分鐘）：**

- 「Day 1 學的是什麼？意象 — 用看得見的東西承載看不見的感覺。鄭清文沒有寫一個『愛』字，但你讀完就知道了。」
- 「Day 2 學的是什麼？敘事觀點 — 用某個人的眼睛進入故事。你選了角色，用那個角色的聲音寫。」

**引入象徵（6 分鐘）：**

- 「今天：**象徵**。當一件**物品**承載了整個時代的重量。」
- 「在〈紅鞋子〉裡面，有哪些東西的意義超過了它本身？」

全班快速腦力激盪。期待的回應：
- 紅鞋子（電影裡的）— 藝術、美、為熱情而死
- 那兩本書（《新民主主義》《論聯合政府》）— 不過是紙和墨，卻毀了一個人的人生
- 手銬 — 國家暴力、自由瞬間消失
- 紅色吉普車 — 逮捕的機器，在黑夜裡等著
- 判決書 — 用官僚語言抹去一個人的存在

**板書：**

```
象徵 (Symbolism)

一件東西 → 承載一整段歷史

紅鞋子 → 藝術的代價
兩本書 → 求知的罪
手  銬 → 自由的消失
判決書 → 沉默的制度

意象是「一幅畫面承載一個感覺」
象徵是「一件物品承載一整個時代」
```

- 教師收束：「象徵跟意象不一樣。意象是一幅畫面帶出一個感覺。象徵是一件**物品**承載一段**歷史**。你看到『紅鞋子』三個字，看到的不只是鞋子 — 你看到的是每一個在危險年代裡有夢想、卻被碾碎的人。」

#### 2. 教師黑板示範（4 分鐘）

教師邊寫邊說（思考出聲）：

> 「我來試試看用象徵寫幾行。我選的物品是那兩本書。」

**黑板示範：**

```
兩本書
封面沒有作者的名字
紙頁翻久了會泛黃
但判決書上的字
比墨水更持久

五年的房間裡
沒有書架
```

教師問學生：
- 「我用了幾個物品？」（兩本書、紙頁、判決書、一個房間、一個不存在的書架）
- 「哪一個最重要？」→ 引導學生注意最後一行 — **不存在的書架**。沉默就是那些「不在」的東西。

#### 3. 詩歌創作：「沉默中的一句話」（18 分鐘） **[DI — 寫作環節]**

**題目（板書）：**

```
詩歌創作（6-10 行）

「沉默中的一句話」

在一個不能說話的時代，
有一個字、一個聲音、一件物品，
替人們說出了不能說的話。

用象徵的手法，
讓一件東西替你說。
```

**分層引導：** **[DI]**

| 層 | 引導方式 |
|----|---------|
| **D** — 需要起手式 + 物品提示 | 教師在黑板角落寫三個開頭：<br>「那本書躺在桌上，＿＿＿。」<br>「手銬的聲音像＿＿＿。」<br>「紅鞋子在黑暗的電影院裡＿＿＿。」<br>選一個，接著寫 4-6 行。**至少用一個物品當象徵**。教師在前 3 分鐘內親自走到 D 層學生旁邊確認他們動筆了。 |
| **C** — 需要結構引導 | 「選一件物品 — 可以從文本裡選，也可以從你自己的生活選。先寫它的外觀（1-2 行），再寫它承載的東西（2-3 行），最後寫一個**不在**的東西 — 一個沉默、一個缺席、一個消失的事物。」合計 6-8 行。 |
| **B** — 有想法但需要聚焦 | 「選一件物品。不要解釋它的意思 — 讓讀者自己去感覺。每一行都要有具體的東西（一個顏色、一種質感、一個聲音、一個位置）。不准用抽象詞：自由、悲傷、沉默。」6-10 行。 |
| **A** — 自由發揮 + 挑戰 | 「用至少兩個象徵。試著讓最後一行翻轉或複雜化前面的內容。挑戰：從**物品本身**的角度寫 — 如果那兩本書會說話，它們會說什麼？」6-10 行。 |

⚡ **時間管理**：15 分鐘寫作 + 3 分鐘收尾。12 分鐘時預告「還有 3 分鐘」。如果學生提早寫完，教師低聲說：「再讀一次。有沒有哪一行可以刪掉？詩越短越有力。」

### 四、收束（10 分鐘）

#### 1. 詩歌分享（6 分鐘）

- 「誰想念自己寫的？先告訴我們你用了什麼物品。」
- 收 3-4 位。盡量跨層次：一個用了起手式但寫得好的 C/D 學生、一個有強烈意象的 B 學生、一個有出人意料轉折的 A 學生。
- 每人唸完：「我聽到了，謝謝。」如果某一行特別有力：「那一行 — 象徵在起作用了。」
- 如果兩個學生用了同一個物品但寫出完全不同的詩：「同一件東西，兩首完全不同的詩。物品不決定意義 — 你決定。」

#### 2. Day 4 銜接 + 發文本（4 分鐘）

教師銜接語：
- 「三天了。Day 1 我們學會進入故事。Day 2，有人**被帶到**這座島上。Day 3，有人**留在**這座島上，卻被沉默了。」
- 「明天：一個**比他們所有人都更早到**的人。比國民黨早、比日本人早、比荷蘭人早。他家族的每一代人都上了戰場 — 但沒有一代人是為自己而戰的。」

**發下〈戰士乾杯〉黃春明，簡短框架：**
- 「黃春明去拜訪屏東深山裡一個魯凱族的部落叫『好茶』。在一戶人家的牆上掛著照片。每一張照片都是一個戰士。每一個戰士替不同的國家打仗。沒有一個人是為自己的族人打的。」
- 「今天回家讀。全文，不長。讀的時候注意一件事 — 牆上的照片。」

**板書：**

```
回家：
讀〈戰士乾杯〉黃春明 → 全文
注意：牆上的照片
```

## 教師備註

1. **三個轉折點不是講課的內容。** 它們必須透過討論浮現。教師埋下問題；學生發現連結。如果轉折點 3（臺灣主體性 / 葉石濤作為文學史家的意義）沒有自然浮現，教師可以直接點出 — 但只在審問段落的討論情緒已經落地之後。順序是：先感受，再思考。

2. **教師朗讀被捕場景是這堂課的情感錨點。** 如果有什麼要在前一天晚上排練，就是這段。從《紅鞋子》電影的柔美場景突然切換到暴力逮捕，這個轉折必須落地。在家出聲讀一遍，計時約 4-5 分鐘。

3. **莎士比亞細節的教學力量很強。** 一個邊讀莎士比亞邊毀人一生的特務頭子。九年級學生會記住這種細節。不要急著帶過。讓學生反應。有些人覺得荒謬、有些人覺得恐怖、有些人覺得兩者皆是。這些反應都是對的。

4. **Day 2 到 Day 3 的複雜性。** David 特別要求學生看到臺灣人/外省人的分野不是簡單的。文本幫了忙：阿淘的朋友們（阿才伯、辜雅琴、甘瑞月、本順哥）都只是有夢想的普通人。逮捕他的探員也是執行命令的人。228 在文本中自然出現。教師不應該製造「外省人=壞人、臺灣人=好人」的敘事。制度才是敵人，不是任何一群人。

5. **詩歌寫作是這個區塊第一次寫詩。** 有些學生會焦慮。D 層的起手式一定要**寫在黑板上**，不能只用口說。教師的黑板示範是關鍵 — 沒看到範例，多數學生不知道「象徵入詩」長什麼樣子。

6. **如果時間不夠**，第一個壓縮的是 Day 2 視角分享（10 分鐘縮到 6 分鐘，少收幾個分享）。精讀和詩歌創作**不能縮短**。象徵技法教學如果腦力激盪控制節奏，可以壓到 5 分鐘。

7. **判決書是真實的歷史文件。** 朗讀時，官僚語言的平淡**本身就是恐怖**。用沒有感情的聲音讀 — 讓內容自己做工。學生會感受到枯燥法律語言和毀滅性後果之間的落差。

8. **與區塊主題的連結**：「從沉默到發聲」。葉石濤的個人故事**就是**這個弧線。他被沉默了 — 被關、被社會排斥。他最後重新找到聲音，寫下了臺灣文學的根本文件。他的個人旅程就是整個區塊的旅程。教師今天不需要明說這一點；隨著區塊推進，這個連結會越來越清楚。

9. **發〈戰士乾杯〉時的預告。** 告訴學生明天的文本是關於原住民。如果有學生問「白色恐怖之後為什麼讀原住民」，教師可以說：「因為這座島嶼的故事不只有兩面。」說到這裡就好。Day 4 會做剩下的事。

10. **象徵的最終意涵。** 電影《紅鞋子》講的是一個芭蕾舞星在愛與藝術之間掙扎，最後為了藝術而死。葉石濤的朋友們 — 辜雅琴彈鋼琴、甘瑞月跳舞、本順哥做木工、阿才伯追求社會正義 — 都是有熱情的人。紅鞋子代表的是每一個在國家恐怖機器下被碾碎的夢想。教師應該讓學生自己透過討論抵達這個詮釋。如果他們沒有到，教師可以提供：「電影裡，舞者因為停不下來而死。現實裡，人們因為停不下來讀書、跳舞、彈琴、關心正義，而被摧毀。」

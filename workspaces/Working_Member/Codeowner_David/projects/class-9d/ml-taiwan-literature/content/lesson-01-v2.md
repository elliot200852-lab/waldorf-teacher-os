# Day 1 教案 — 入口：走進故事的方法

## 基本資訊

| 項目 | 內容 |
|------|------|
| 日期 | 第 8 週・週二 |
| 時長 | 120 分鐘 |
| 主題 | 一篇小說能告訴你什麼？ |
| 搭配文本 | 〈鋼琴〉馬查多（暖身）/〈水上組曲〉鄭清文（主文本） |

---

## 構想

第一堂課只做三件事：

1. **讓學生敢讀**：很多學生聽到「臺灣文學」會先退一步，覺得那是很遠、很嚴肅的東西。用〈鋼琴〉破冰 — 一頁紙，五分鐘讀完，但裡面藏了一整個社會。讓學生知道：讀小說不需要什麼特別的準備，你只要讀，它自己會跟你說話。

2. **讓故事進來**：〈水上組曲〉說的是一個船夫喜歡河邊洗衣服的女生，但他從來沒有說出口。這種事九年級的人懂。故事就是這麼貼近生活的東西 — 不是教科書，不是道理，是人的感覺。課堂上讀一部分，讀不完的回家繼續。

3. **練習寫一小段**：不寫作文，不寫心得。就寫一個你記得的畫面，200 字。用教室裡看得到的東西做示範，讓學生知道「寫作」可以從眼前開始，不需要靈感從天上掉下來。

---

## 教學步驟

### 一、開場（10 分鐘）

#### 1. 簡單開場白（5 分鐘）

不用晨頌，直接說：

- 「接下來八天，我們每天讀一篇故事。都是臺灣人寫的，或是跟臺灣有關的。」
- 「有的是小說，有的是散文，有的是詩。有的好笑，有的會讓你心裡很不舒服。」
- 「讀完這八天，我希望你對一個問題有自己的想法：**臺灣人到底是什麼人？**」
- 「今天先熱個身。」

#### 2. 閱讀約定（板書，5 分鐘）

在黑板上寫三行：

```
讀的時候 — 不急著下結論
讀完以後 — 先說「你看到什麼」
最後再說 — 「你覺得怎樣」
```

口述：「這八天我不會問你『作者想表達什麼』這種問題。我只會問你看到了什麼、你注意到什麼。沒有標準答案。」

---

### 二、暖身：〈鋼琴〉馬查多（20 分鐘）

#### 1. 安靜閱讀（8 分鐘）

- 一人一張 A4
- 「安靜讀，讀完放下來。」

#### 2. 聊一聊（12 分鐘）

不用舉手回答，隨意指人或讓學生自由接話：

- 「故事裡有誰？」
- 「發生什麼事？鋼琴後來怎麼了？」
- 「有沒有人覺得結局有點⋯⋯莫名其妙？」（這裡可以引出討論）
- 「你覺得那個寡婦到底在意什麼？」

收束（不要講太長）：
- 「你剛才完全不認識巴西，也不認識這個作家。但讀完這一頁，你已經感覺到一些東西了 — 那個社會的虛偽、那個人被牽著走的無奈。」
- 「故事自己會說話。接下來讀臺灣的故事，也是一樣：先讀，先感受。」

---

### 三、主文本：〈水上組曲〉鄭清文（45 分鐘）

#### 1. 教師朗讀開頭（10 分鐘）

慢慢念前面幾段 — 船夫撐竹竿、望著石階、等那個洗衣服的女生出現。

朗讀提示：
- 不要用「朗誦」的腔調，用講故事的語氣
- 念到「她果然又穿著那淡藍色的布衣」可以停一拍
- 念到船夫偷看她的段落，語氣自然放輕

念完停幾秒。不解釋，不發問。

#### 2. 學生安靜閱讀（20 分鐘）

- 發文本
- 「從我剛才念到的地方接著讀。讀到哪裡算哪裡，讀不完沒關係，回家繼續。」
- 教師走動，個別回答詞彙問題

#### 3. 討論（15 分鐘）

問題不用全問，看班上氣氛挑著問：

- 「你讀到了什麼畫面？」（讓學生用自己的話描述場景）
- 「這個船夫，他喜歡她嗎？」
- 「他有做過什麼嗎？」（大概只有幫她撈過一件掉到河裡的衣服）
- 「為什麼他不講？你覺得他是不敢還是不想？」
- 「你身邊有沒有這種人，有話不說、放在心裡的？」

教師只補一句就好：
- 「鄭清文這個作家，別人叫他臺灣的契訶夫。他寫東西有個特色：從頭到尾不直接告訴你感覺，但你讀完就是覺得心裡悶悶的。今天先記住這個感覺就好。」

---

### 四、寫作練習（35 分鐘）

#### 1. 示範：什麼叫「用畫面寫心情」（10 分鐘）

**先問一個問題（2 分鐘）：**

- 「鄭清文整篇沒有寫過一個『喜歡』或『愛』，但你知道船夫的心情。他怎麼做到的？」
- 等學生回答。引到：「他寫的全是東西 — 水珠、藍色衣服、肥皂泡泡。用看得見的東西，讓你自己去感覺。」

**現場示範（8 分鐘）：**

教師看一圈教室，指著實際看得到的東西，當場在黑板上寫：

> **示範一（用教室裡的粉筆）：**
> 「粉筆盒裡剩最後三根，兩根白的、一根黃的。黃的那根被折斷過，用紙膠帶纏起來，寫到一半經常會卡住。每次卡住的時候，黑板上就多一道刮痕。」
>
> 問學生：「你從這幾句話裡面，有沒有感覺到什麼？」（可能是：省、窮、勉強撐著、老舊⋯⋯）「但我有寫『很窮』或『很破舊』嗎？沒有。全是東西。」

> **示範二（用窗外的天氣或走廊）：**
> 「走廊盡頭的垃圾桶旁邊，永遠放著一雙藍白拖。下課的時候，有人穿著它去福利社，上課鐘響了它又回到那裡。沒有人知道是誰的。」
>
> 問學生：「你腦海裡有畫面了嗎？那雙藍白拖給你什麼感覺？」

> **示範三（帶一點幽默）：**
> 「講台上的馬克杯已經被忘在這裡三天了。杯底有一圈深褐色的咖啡漬，像年輪一樣，一圈一圈的。如果那些年輪能說話，大概每一圈都在說：主人你到底還來不來拿我。」

教師收束：「看到了嗎？不用去想什麼很偉大的東西。眼前的粉筆、走廊的拖鞋、桌上的杯子 — 你把細節寫出來，心情自己就跑出來了。」

#### 2. 書寫（25 分鐘）

**題目（板書）：**

```
寫一個你記得的畫面（200 字）

只寫你看到的東西。
不用寫「我覺得」「我很想念」「那一天我很難過」。
讓畫面自己說。
```

**寫作引導（口述 + 板書提示）：**

可以寫什麼？隨便，比如：
- 阿嬤家客廳的某個角落
- 學校某個你常待的地方
- 一個你記得的瞬間（下雨天的走廊、早餐店的蒸氣、某個人的背影）
- 甚至就寫你現在坐的這個位置

**卡住的話，從這三步開始：**

| 步驟 | 問自己 | 例子 |
|------|--------|------|
| 1. 一樣東西 | 那裡有什麼？ | 一張籐椅 |
| 2. 一個細節 | 它長什麼樣子？ | 扶手被磨得光亮，左邊那根有裂開 |
| 3. 一個動作或聲音 | 有什麼在動/在響？ | 電風扇轉著，椅子上沒有人 |

把這三樣串起來，就是一段了。不用多，三到五句就很好。

---

### 五、收束（10 分鐘）

#### 1. 分享（5 分鐘）

- 請 2–3 位學生讀自己寫的
- 不評好壞。聽完就說：「我看到那個畫面了，謝謝。」
- 沒人自願的話：「把你最喜歡的那一句抄在小紙條上，摺起來交上來，我抽幾張念。」

#### 2. 回家的事（5 分鐘）

發〈岸與岸〉桑品載的文本。

- 「今天晚上回家做兩件事：」
  1. **把〈水上組曲〉讀完**（讀到哪裡了就從那裡接著讀）
  2. **讀〈岸與岸〉**— 明天上課要用
- 「〈岸與岸〉在講一個 11 歲的男孩，被媽媽決定帶上軍艦，從中國到臺灣。那是 1950 年。讀的時候想想：如果是你，你會怎樣？」
- 板書提醒（留到明天上課前擦）：

```
回家：
1. 讀完〈水上組曲〉
2. 讀〈岸與岸〉→ 明天要討論
```

## 學生課堂作業

| 項目 | 說明 |
|------|------|
| 書寫 | 200 字短文「一個你記得的畫面」，寫在工作本 |
| 重點 | 是否有具體的東西（不是抽象感受），有沒有細節 |
| 格式 | 不限，直接寫 |

## 回家作業

| 任務 | 說明 |
|------|------|
| 讀完〈水上組曲〉 | 課堂上讀不完的部分，回家讀完 |
| 讀〈岸與岸〉 | 明天的主文本，全班共讀，帶著問題：如果你是那個 11 歲的孩子？ |

（不額外發延伸閱讀。第一天回家已經要讀兩篇了，夠了。）

## 教師備註

1. **〈水上組曲〉讀不完是正常的。** 課堂上能讀多少就多少，重點是教師朗讀的那幾段要到位，讓學生感受到文字的質感。討論只需要根據學生讀到的部分進行，不需要讀完全文才能討論。
2. **寫作示範是這堂課的關鍵。** 教師在黑板上現場寫的那三個例子，比任何口述引導都有效。可以根據自己教室的實際狀況即興發揮 — 重點是用學生眼前看得到的東西，讓他們知道寫作就是從觀察開始。
3. **第一天不要太多。** 學生記住三件事就夠了：故事自己會說話、寫東西不寫感覺、回家讀兩篇。
4. 板書建議：在黑板角落寫上八天的弧線（入口 → 漂流 → 沉默 → 根 → 汗 → 海 → 我 → 回音），每天標記「今天在這裡」。
5. 如果時間不夠，寫作練習可以縮短，但**示範不能省**。沒有示範直接寫，多數學生會卡住。

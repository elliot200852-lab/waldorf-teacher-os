# 「這座島嶼上的人」— 九年級臺灣文學主課程教學大綱

## 課程基本資訊

| 項目 | 內容 |
|------|------|
| 課程名稱 | 臺灣文學主課程 |
| 主題 | 「這座島嶼上的人 — 從離散到扎根，從沉默到發聲」 |
| 授課教師 | 林信宏 |
| 對象 | 九年級 D 班 |
| 總堂數 | 8 天（7 天新進度 + 1 天回顧） |
| 每堂時長 | 120 分鐘 |
| 時間安排 | 第 8 週：週二、週三、週四；第 9 週：週一至週五 |

---

## 核心主軸

**臺灣主體性** — 時代範圍從國民黨來臺（1945）以後至今。

透過不同族群、不同世代的文學作品，認識這座島嶼上各種人的故事。每一篇小說、散文或詩，都是一扇窗，讓我們看見某個時代、某群人走過的路。七天讀完，回到一個問題：**臺灣人究竟是什麼人？**

### 課程目標

1. **認識**：透過文學作品認識戰後臺灣社會的多元面貌 — 離散、白色恐怖、原住民、勞工、海洋、移工
2. **閱讀**：培養獨立閱讀與文本分析能力，學會從作品中讀出時代與社會
3. **書寫**：發展多元書寫能力（短文、評論、詩歌、生活雜記）
4. **思考**：建立對臺灣主體性的初步思考，從文學出發理解自己所生活的社會

### 主題弧線

```
Day 1  入口 → 走進故事的方法
Day 2  漂流 → 離散與落地（外省人）
Day 3  沉默 → 白色恐怖
Day 4  根   → 原住民的聲音
Day 5  汗   → 經濟奇蹟的代價（勞工）
Day 6  海   → 島嶼之外（海洋與移工）
Day 7  我   → 臺灣人究竟是什麼人？
Day 8  回音 → 回顧與分享
```

## 教學節奏（每堂 120 分鐘）

| 階段 | 時間 | 內容 |
|------|------|------|

| 書寫練習 | 20–25 min | 工作本書寫（每日不同文體） |
| 收束 | 5–10 min | 預告明日主題，發放閱讀材料 |

## 差異化策略

- **閱讀量分層**：每日設定主文本（全班共讀）+ 延伸閱讀（進階組自選）
  - 全班：讀主文本
  - A/B 組（能力較強）：加讀延伸文本
  - 延伸閱讀不在課堂上深入討論，僅作為個人探索
- **寫作鷹架**：
  - C/D 組（需支持）：提供結構提示（例如「先寫三樣你看到的東西」）
  - A/B 組：自由發揮，鼓勵嘗試文本中學到的技法
- **原則**：一天一篇為主，最多兩篇。第二篇不過度琢磨。

## 每日教學內容

### Day 1 — 入口：走進故事的方法

**第 8 週・週二**

| 項目 | 內容 |
|------|------|
| 暖身文本 | 〈鋼琴〉馬查多（A4 一頁，約 10 分鐘閱讀） |
| 主文本 | 〈水上組曲〉鄭清文 |
| 寫作技法 | **意象**（imagery）— 用畫面說話，不直說感受 |
| 書寫練習 | 200 字短文：「一個你反覆想起的畫面」 |
| 回家閱讀 | 〈岸與岸〉桑品載 |

**教學重點**

- 〈鋼琴〉不是臺灣文學，而是建立閱讀方法的暖身：「你不需要知道巴西的背景，光是讀文本，故事自己會告訴你。接下來讀臺灣文學也一樣。」
- 〈水上組曲〉是「臺灣的契訶夫」鄭清文的經典 — 舊鎮船夫暗戀洗衣女子的故事。表面樸實，底下藏著濃烈的情感。九年級情竇初開，對「愛在心裡口難開」有天然共鳴。
- 意象教學：「鄭清文沒有寫『他愛她』，但你讀完就知道了。因為他用畫面說話 — 竹竿上的水珠、淡藍色的布衣、肥皂的泡沫。」
- 建立閱讀契約：先感受，再思考；先說「你看到了什麼」，再說「你覺得怎麼樣」。

---

### Day 2 — 漂流：誰不是外來者？

**第 8 週・週三**

| 項目 | 內容 |
|------|------|
| 主文本 | 〈岸與岸〉桑品載 |
| 延伸閱讀 | 〈老楊和他的女人〉履彊（A/B 組） |
| 寫作技法 | **敘事觀點**（第一人稱回憶）— 時間距離如何讓故事產生厚度 |
| 書寫練習 | FB 留言式評論（100–150 字）：「讀完〈岸與岸〉，你想對那個 11 歲的男孩說什麼？」 |
| 回家閱讀 | 〈紅鞋子〉葉石濤（教師標記重點段落） |

**教學重點**

- 桑品載 11 歲被母親決定帶上軍艦，從舟山到基隆 — 那是 1950 年，比你們的阿公阿嬤還年輕。母親在微弱的美孚燈下，一邊流淚一邊縫裝銀元的布袋。
- 「離散」：超過一百萬人在那幾年從中國各地來到臺灣，再也沒有回去。他們帶來了什麼？失去了什麼？
- 對比引導（進階）：〈老楊和他的女人〉中的外省老兵選擇了不同的路 — 在恆春半島山谷中落地生根，娶了部落邊緣的聾啞女子。「落地不生根」與「落地生根」的對比。
- FB 留言格式：讓學生用最自然的語言回應，降低「寫作文」的壓力。

---

### Day 3 — 沉默：當說話變得危險

**第 8 週・週四**

| 項目 | 內容 |
|------|------|
| 主文本 | 〈紅鞋子〉葉石濤（教師選讀核心段落） |
| 延伸閱讀 | 〈台灣島上血與恨〉伯子（A 組） |
| 寫作技法 | **象徵**（symbolism）— 一件物品承載一整段歷史 |
| 書寫練習 | 詩歌創作（6–10 行）：「沉默中的一句話」 |
| 回家閱讀 | 〈戰士乾杯〉黃春明 |

**教學重點**

- 全文較長，教師事先選段：葉石濤的文學天才（17 歲入選文學雜誌）→ 戰後為學中文買雜誌被捕 → 獄中生活 → 出獄後的重生。
- 背景說明：什麼是白色恐怖？為什麼買雜誌會被抓？
- 象徵教學：「紅鞋子」在故事中象徵什麼？（自由、美好生活的渴望、被剝奪的日常）
- 討論：「如果你生活在一個說話會被抓的年代，你會選擇沉默還是繼續說？」
- 連結前日：昨天讀的桑品載被帶到臺灣，今天讀的葉石濤留在臺灣卻被關進監獄 — 不同的人，不同的苦。
- 詩歌寫作引導：不需要押韻，只要把一個感受濃縮成幾行文字。

### Day 4 — 根：聽見最老的聲音

**第 9 週・週一**

| 項目 | 內容 |
|------|------|
| 主文本 | 〈戰士乾杯〉黃春明 |
| 延伸閱讀 | 〈飛鼠大學〉撒可努（選段，A/B 組） |
| 搭配朗讀 | 莫那能詩選 2–3 首（教師課堂朗讀） |
| 寫作技法 | **對比**（contrast）— 今昔對比如何製造情感張力 |
| 書寫練習 | 個人生活雜記（200–300 字）：「我與一個地方的記憶」 |
| 回家閱讀 | 〈工廠的舞會〉楊青矗 |

**教學重點**

- 黃春明跟著魯凱青年上山，走訪舊好茶部落 → 牆上的照片 → 這個家族的每一代人都為別人而戰（清國、日本、美軍、國軍、共軍）。「祖父以降的人都不是為自己而戰。」
- 高潮場景：熊母親的靈歌 → 黃春明舉著酒杯對照片痛哭。「這不只是一個家族的故事，是整個臺灣歷史的縮影。」
- 教師朗讀莫那能詩選：讓原住民的聲音直接進入教室。
- 延伸〈飛鼠大學〉：不只有苦難，也有智慧與美 — 撒可努筆下排灣族的山林生態智慧。（撒可努曾到慈心演講，學生可能有印象。）
- 對比教學：黃春明在文中不斷以「現在」與「過去」對照 — 現在寧靜的石板屋 vs. 過去一代代離家作戰的命運。

---

### Day 5 — 汗：奇蹟的代價

**第 9 週・週二**

| 項目 | 內容 |
|------|------|
| 主文本 | 〈工廠的舞會〉楊青矗 |
| 延伸閱讀 | 〈龜爬壁水崩山〉楊青矗（重點段落，A/B 組） |
| 寫作技法 | **對話**（dialogue）— 如何用角色的說話推動故事 |
| 書寫練習 | 200 字短評：「你身邊看見的勞動」 |
| 回家閱讀 | 〈海浪之歌〉Justto Lasoo |

**教學重點**

- 〈工廠的舞會〉幾乎全篇由對話構成 — 三位女工樓長與舍監的交鋒。職員坐前排有飲料點心，女工擠後排什麼也沒有。最終女工們發動集體罷舞。
- 時代背景：臺灣七〇年代經濟奇蹟的背後 — 上百萬年輕人離開農村進入工廠，用青春血汗堆出經濟成長。
- 「水崩山龜爬壁」的比喻：老闆賺錢像水崩山，勞工賺錢像烏龜爬壁。（書名即比喻。）
- 對話教學：楊青矗幾乎不用旁白解釋，全靠角色自己說話 — 讀者從對話中自己判斷誰對誰錯。
- 連結生活：「你的阿嬤年輕時可能就在這樣的工廠工作。」「現在臺灣還有勞工被不公平對待嗎？」

---

### Day 6 — 海：島嶼之外

**第 9 週・週三**

| 項目 | 內容 |
|------|------|
| 主文本 | 〈海浪之歌〉Justto Lasoo（王磊） |
| 延伸閱讀 | 〈LIR ILIR〉Abdul Mubarok（A/B 組） |
| 搭配 | 〈討海人〉廖鴻基（教師選讀 1–2 段） |
| 寫作技法 | **文化翻譯** — 當不同語言、不同文化的人要互相理解 |
| 書寫練習 | FB 留言或短文（150 字）：「如果你漂洋過海到另一個國家工作」 |
| 回家閱讀 | 四首臺灣詩（陳秀喜〈臺灣〉、向陽〈我的姓氏〉〈一首被撕裂的詩〉〈臺灣的孩子〉）— 課前預讀 |

**教學重點**

- 〈海浪之歌〉：印尼漁工 Yadi 來臺灣討海，與雇主劉先生從嫌隙到生死與共 → 海上巨浪中的深刻對話 → 情同父子的關係。
- 數據帶入：目前在臺服務的東南亞移工總數已超過臺灣原住民總人口。他們就在我們身邊。
- 延伸〈LIR ILIR〉：穆斯林移工在齋戒月的堅持 → 工廠苦力 → 開齋節傳來的壞消息。
- 廖鴻基選段：臺灣人自己也是海洋的人 — 討海人的生命觀。
- 文化翻譯教學：移工文學用中文書寫異文化經驗，這本身就是一種翻譯行為。語言背後是整套不同的世界觀。
- 討論：「你在日常生活中遇過移工嗎？你跟他們說過話嗎？」

### Day 7 — 我：臺灣人究竟是什麼人？

**第 9 週・週四**

| 項目 | 內容 |
|------|------|
| 開場定錨 | 葉石濤語錄選段（《臺灣文學史綱》序文）— 呼應 Day 3 |
| 四首臺灣詩 | ① 陳秀喜〈臺灣〉（教師朗讀）　② 向陽〈我的姓氏〉（核心，25 分鐘）　③ 向陽〈一首被撕裂的詩〉　④ 向陽〈臺灣的孩子〉（收束）|
| 寫作技法 | **散文詩**（prose poetry）— 向陽風格：斷行、意象並置、重複 |
| 書寫練習 | 散文詩（200–400 字）：「我是 _______」 |
| 回家 | 準備 Day 8 回顧分享：① 選一段這幾天讀到的段落，明天念出來 ② 想一下這堂課中最深刻的部分（畫面、觀念、內容或上課情景），明天用說的講出來 |

**教學重點**

- **葉石濤開場定錨（15 分鐘）**：Day 3 那個因買雜誌被捕的少年，三十年後出版《臺灣文學史綱》。核心語錄：「既不是日本人，也不是中國人。」「沒有土地，哪有文學。」葉石濤是入口，不是核心。
- **向陽〈我的姓氏〉是核心文本（25 分鐘）**：一個人的名字在四百年裡被改了四次——從 A-Wu（平埔族）到漢名，到日名，到戰後改名——最後連源頭是什麼都不清楚。「姓氏不明」是裂縫，不是答案。深度討論，不可壓縮。
- **〈一首被撕裂的詩〉**：視覺衝擊為主，不需長討論。你想說的話，連字都被撕掉了。呼應 Day 3 白色恐怖——沉默不只是不說話，是語言本身被撕毀。
- **〈臺灣的孩子〉收束**：從裂縫走向修復。孩子們在河邊歌唱，星星記下希望。這首詩是寫給學生的。
- **弧線收束**：七天——離散、沉默、根、汗、海——都是這座島嶼上的人。今天回到你自己。
- **散文詩技法**：從向陽詩作分析——斷行讓一個句子呼吸、意象並置讓兩種感受碰撞、重複讓一個詞加重。讓學生用最少的字說最準的感受寫「我是 _______」。

| Day | 技法 | 從文本中學到 |

## 備註

1. **文本長度**：〈紅鞋子〉〈龜爬壁水崩山〉〈飛鼠大學〉全文較長，需教師事先選段標記，課堂以核心段落為主。
2. **文本來源**：所有文本由郭朝清老師整理分享，原檔為 .docx，已轉為 Markdown 存放於 `reference/texts/`。
3. **閱讀節奏**：前一天收束時發放次日主文本，學生在家預讀。課堂上教師朗讀開頭引入，再進入討論。如學生未能在家讀完，課堂開頭保留 15–20 分鐘安靜閱讀時間。
4. **九年級發展特質**：學生處於二極性（傾向對比的極端），因此主題弧線刻意在沉重與希望間交替 — Day 3 白色恐怖後接 Day 4 原住民智慧，Day 5 勞工苦難後接 Day 6 海洋開闊。
5. **郭朝清老師的反思提醒**：
   - 部份文本（如〈彈子王〉〈一個台灣原住民的經歷〉）有較激烈描寫，教師授課前須預做考量。
   - 選材必然是化約，教師的熱情與活力仍是學生學習的重要推力。
   - 課程是起點，不是終點。

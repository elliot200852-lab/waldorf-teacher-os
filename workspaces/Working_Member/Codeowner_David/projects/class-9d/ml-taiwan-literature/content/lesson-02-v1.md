# Day 2 教案 — 漂流：誰不是外來者？

## 基本資訊

| 項目 | 內容 |
|------|------|
| 日期 | 第 8 週・週三 |
| 時長 | 120 分鐘 |
| 主題 | 外省人的離散 — 一個 11 歲男孩被帶上軍艦 |
| 主文本 | 〈岸與岸〉桑品載 |
| 寫作技法 | 敘事觀點（第一人稱回憶） |
| 書寫練習 | FB 留言式評論（100–150 字） |

---

## 構想

第二天開始進入臺灣歷史的實質內容。〈岸與岸〉是真人真事 — 桑品載 11 歲那年，他姊姊被一個國軍連長看上，母親在一念之間決定讓姊弟倆上軍艦，從舟山到臺灣。這不是小說，是一個人對自己人生轉折的回憶。九年級學生追求「真實」，真人真事比虛構更有力。

**差異化的核心問題**：同一篇文本，有些學生讀完只看到「一個小孩搭船去了臺灣」，有些學生讀完會被母親在燈下縫銀元布袋的畫面打動。差異不是閱讀速度的問題，而是**感受力的落差**。Day 2 的 DI 設計不只分閱讀量，更要透過討論結構幫助不同感受力的學生都能進入故事。

---

## 差異化設計（本堂核心）

### 學生分群概況（依文學感受力，非固定標籤）

| 群 | 特徵 | 本堂需要 |
|----|------|----------|
| A | 閱讀快、文字敏感，讀完會自己想很多 | 給空間去深想，別限制他們；討論時讓他們帶組 |
| B | 閱讀能力不錯但不太主動表達 | 結構化的討論任務逼他們開口；小組角色分配 |
| C | 閱讀速度慢或回家沒讀完 | 課堂開頭的補讀時間 + 教師標記重點段落 |
| D | 對文字不敏感，覺得「這跟我有什麼關係」 | 從生活經驗切入（你有沒有跟家人分開過？），不從文本分析切入 |

### DI 落實在每個環節

下方教學步驟中，每個有 DI 分層的環節都會用 **[DI]** 標記。

---

## 教學步驟

### 一、開場 + 回顧 Day 1（20 分鐘）

#### 1. 回顧〈水上組曲〉（10 分鐘）

- 「昨天回家把〈水上組曲〉讀完了嗎？」（舉手調查，不追究沒讀的）
- 「誰可以用一句話說，這篇故事在講什麼？」
- 接幾個學生的回答後：「船夫從頭到尾沒有說出口。鄭清文從頭到尾沒有寫出那個字。但你讀完就知道了。這就是昨天說的——用畫面寫心情。」
- 「今天我們要讀一篇完全不同的東西。不是小說，是真的。一個人在回憶自己 11 歲發生的事。」

#### 2. 補讀時間（10 分鐘） **[DI — C/D 組關鍵]**

- 「昨天回家有讀〈岸與岸〉的，繼續讀或重讀你印象深的段落。」
- 「還沒讀完的，現在讀。**先讀我標記的三段就好**，其他的等一下聽同學講也可以。」

**教師事先標記三個必讀段落（發文本時用螢光筆或夾書籤）：**

| 段落 | 位置 | 為什麼選這段 |
|------|------|-------------|
| 母親縫布袋 | 母親丟下碗筷進房間，取出銀元和金戒指... | 整篇的情感核心。母親流淚、姊姊哭、「我」哭不出來。 |
| 上船 / 碼頭 | 蕭連長派兵來接 → 離開家門的那一刻 | 離別的具象畫面。 |
| 基隆睡水泥椅 | 到了基隆，睡在中正堂外的水泥椅上 | 11 歲的孩子到了臺灣之後的真實處境。 |

→ 讀不完全文的學生，光讀這三段就能參與後面的討論。

---

### 二、進入〈岸與岸〉（50 分鐘）

#### 1. 教師鋪墊（5 分鐘）

不長篇大論，只給最必要的背景：

- 「1949 到 1950 年，中國內戰打完了，國民黨輸了。超過一百萬軍人和老百姓跟著撤退到臺灣。」
- 「桑品載那時候 11 歲，住在舟山群島（拿出地圖指一下，或畫在黑板上）。他不是軍人，他只是一個小孩。」
- 「這篇文章是他六十幾歲的時候回頭寫的——回憶自己 11 歲那年發生的事。」

#### 2. 教師朗讀核心段落（10 分鐘）

朗讀「母親縫布袋」那一整段（從「母親丟下碗筷進她房間」到「男兒志氣在四方」）。

朗讀提示：
- 「她一邊縫一邊流淚，淚流多了就用手背去擦」— 這句放慢
- 數銀元的時候一個一個數，讓教室安靜下來
- 「男兒志氣在四方」— 母親的聲音，帶著不得已

讀完停幾秒。

#### 3. 小組討論（25 分鐘） **[DI — 核心分層環節]**

**分組方式**：4 人一組，教師事先安排混合分組（每組至少一個讀完全文的學生）。

**每組發一張討論卡，上面有三層問題：**

```
┌─────────────────────────────────────────────┐
│  〈岸與岸〉討論卡                              │
│                                               │
│  【必答】每個人都要說                           │
│  1. 母親為什麼答應讓蕭連長帶走姊姊和弟弟？       │
│     你覺得她在想什麼？                          │
│  2. 如果你是那個 11 歲的桑品載，上船的那一刻，    │
│     你心裡會是什麼感覺？                        │
│                                               │
│  【進階】讀完全文的人回答                        │
│  3. 桑品載到基隆之後，過的是什麼樣的日子？        │
│  4. 他說「母親改變了我的一生」——                 │
│     你覺得這是感謝還是遺憾？還是兩者都是？        │
│                                               │
│  【挑戰】想更深的人試試                          │
│  5. 五十年後桑品載回去找姊姊，兩人「相視無言       │
│     後抱首痛哭」。一個擁抱裡面裝了什麼？          │
└─────────────────────────────────────────────┘
```

**討論規則（板書）：**
- 每個人至少說一次
- 不用舉手，輪流講
- 沒有對錯，說你自己的想法

**教師巡迴重點：**
- 先到 C/D 比例高的組，確認他們有進入討論
- 如果有學生說「我不知道」，教師蹲下來問：「你有沒有跟家人分開過？去夏令營、或爸媽出差很久？那種感覺是什麼？」→ 從自身經驗引入

#### 4. 全班分享（10 分鐘）

- 每組派一個人分享一個最有意思的回答（不限哪一題）
- 教師收束時帶出一個點：
  - 「這些人來到臺灣，很多人一輩子沒有回去。他們不是『外省人』這三個字，他們是一個一個活生生的人。有些人在基隆的水泥椅上睡了兩個月，有些人再也沒見過自己的媽媽。」
  - 「但他們後來怎麼樣了？有些人一直覺得自己是過客，有些人慢慢在這裡生了根。明天我們會讀一篇更沉重的——白色恐怖。」

---

### 三、寫作技法 + 書寫練習（35 分鐘）

#### 1. 敘事觀點教學（5 分鐘）

- 「桑品載寫這篇的時候已經六十幾歲了。他在回憶 11 歲的事。」
- 「注意他的寫法：他寫媽媽數銀元，他說『我默默在數，有五十四個』——他記了一輩子。」
- 「這就是**第一人稱回憶**的力量：你知道這是真的，因為只有當事人才會記得那種細節。」
- 「等一下你寫的東西也可以用這種方式——從你現在的角度，回頭寫一件你記得的事。」

#### 2. 書寫（30 分鐘） **[DI — 寫作分層]**

**題目（板書）：**

```
FB 留言（100–150 字）

讀完〈岸與岸〉，你想說什麼？

可以對桑品載說，可以對他媽媽說，
也可以對自己說。
像在社群留言一樣，寫你第一個反應。
```

**分層引導：**

| 層 | 引導方式 |
|----|---------|
| **C/D** — 需要起手式 | 教師在黑板上寫三個開頭讓學生選：<br>「讀到＿＿＿那一段的時候，我覺得＿＿＿。」<br>「如果我是那個 11 歲的小孩，我最害怕的是＿＿＿。」<br>「我想跟桑品載的媽媽說＿＿＿。」<br>選一個接著寫就好。 |
| **B** — 有想法但不知怎麼開始 | 「不用管格式。就像你在 IG 或 Dcard 上看到一篇文章，底下留言 — 你會怎麼寫？」 |
| **A** — 自由發揮 | 「如果你想挑戰，試試看：不只寫你的反應，也寫這個故事讓你想到的另一件事。可以是你自己的經歷，也可以是你聽過的別人的故事。」 |

⚡ **時間提醒**：如果剩 5 分鐘還沒寫完，讓學生先收。FB 留言本來就不長，沒寫完也沒關係。

---

### 四、收束（15 分鐘）

#### 1. 分享（10 分鐘）

- 「誰願意唸自己的留言？」
- 抽 3–4 位，盡量不同層的學生（教師巡迴寫作時已經看過誰寫得好）
- 聽完每篇：「我聽到了。謝謝。」
- 如果有特別好的留言，可以多說一句：「你這句寫得很準。」

#### 2. 回家的事（5 分鐘）

發〈紅鞋子〉葉石濤（**教師標記重點段落版本**）。

- 「明天要讀的更重一點。一個 17 歲就在文學雜誌上發表作品的天才，因為戰後買了幾本中文雜誌，被政府抓去關。」
- 「全文很長，**先讀我標記的部分就好**。讀完標記的，有興趣再往下讀。」

**板書：**

```
回家：
1. 讀〈紅鞋子〉葉石濤 → 先讀標記段落
2. 明天要討論
```

## 課堂作業

| 項目 | 說明 |
|------|------|
| 書寫 | FB 留言（100–150 字），寫在工作本 |
| 重點 | 有沒有具體的回應（不是「很好看」「很感人」，而是指出哪裡、為什麼） |
| DI 評量 | C/D 用了起手式且完成 = 達標；A/B 有連結到自身經驗或更大問題 = 優秀 |

| 任務 | 說明 |
|------|------|
| 讀〈紅鞋子〉葉石濤 | 先讀教師標記的重點段落（約 3–4 段），有餘力讀全文 |

## 教師備註

1. **討論卡是這堂課的關鍵 DI 工具。** 三層問題讓不同程度的學生都有話可說。必答題每個人都能回答（只要讀了標記段落）；進階題需要讀完全文；挑戰題需要深度思考。教師不需要指定誰答哪一題 — 學生自然會依能力選擇。
2. **補讀時間不能省。** 一定有學生回家沒讀。與其假裝大家都讀了然後討論品質很差，不如花 10 分鐘讓他們至少讀完三段。
3. **C/D 學生的寫作起手式要寫在黑板上。** 不能只口述。寫在黑板上，他們隨時可以抬頭看。
4. **明天的〈紅鞋子〉一定要事先標記段落。** 全文太長，不標記的話 C/D 學生回家打開直接放棄。標記 3–4 段核心場景（被捕、獄中、出獄），讓他們先讀這些就夠。
5. **時間最容易超的是小組討論。** 設定 25 分鐘，到 20 分鐘時預告「再 5 分鐘收」。如果時間不夠，寫作可以縮到 20 分鐘（FB 留言本來就短）。

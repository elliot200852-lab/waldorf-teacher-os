# 9D 臺灣文學 W9-4：我 — 臺灣人究竟是什麼人？

## 本堂設計

Day 7 是八天弧線的收束新課（Day 8 為回顧）。主題從前六天的「看見島嶼上不同的人」轉向「回到自己：臺灣人究竟是什麼人」。設計分兩層：葉石濤的論述為開場定錨（15 分鐘），四首臺灣詩為核心體驗，其中向陽〈我的姓氏〉為全課重心（25 分鐘）。

## 設計歷程（v1 → v2）

本堂經歷一次完整的方向轉變，紀錄如下：

### 原始規劃（syllabus v1）

- 主文本：胡德夫〈大武山美麗的媽媽〉
- 搭配：莫那能詩選
- 技法：散文詩
- 邏輯：透過民歌運動與原住民身分收束弧線

### 教師決策：撤換胡德夫

教師判斷胡德夫目前的政治立場已難以代表「臺灣主體性」。要求不限純文學，評論與論說角度的文本更佳。

### v1：葉石濤 + 楊牧 + 宣言

- 核心設計：Day 3 葉石濤（被沉默者）→ Day 7 葉石濤（命名者）的弧線呼應
- 搭配：楊牧〈有人問我公理和正義的問題〉、向陽〈立場〉、陳秀喜〈臺灣〉
- 技法改為「宣言」（manifesto）
- 書寫：三步結構（我看見/我相信/我是）

### v1 的問題

教師回饋：葉石濤的宣言可保留但不應是核心（那只是臺灣文學史上應有的開頭）。核心應放在透過原住民的概念探討自我認同。楊牧移除。技法回到散文詩。討論問題需更細膩有營造感。示範詩缺乏詩意。

### v2：四首臺灣詩 + 向陽為核心

**結構性調整：**

| 項目 | v1 | v2 |
|------|-----|-----|
| 葉石濤 | 核心（25 min） | 開場定錨（15 min） |
| 楊牧 | 搭配朗讀 | 移除 |
| 向陽〈我的姓氏〉 | 無 | 課程核心（25 min） |
| 四首臺灣詩 | 無 | 貫穿全課 |
| 技法 | 宣言 | 散文詩（向陽風格分析） |
| 收束 | 葉石濤六字 | 〈臺灣的孩子〉 |

**四首詩的序列與功能：**

```
陳秀喜〈臺灣〉      → 搖籃是堅固的（穩）
向陽〈我的姓氏〉      → 四百年姓氏不明（裂）
向陽〈一首被撕裂的詩〉 → 連字都被撕掉了（暗）
向陽〈臺灣的孩子〉    → 孩子們在河邊歌唱（光）
```

## 素材選擇理由

- 〈我的姓氏〉是課程核心，因為：(1) 它把抽象的認同問題落在一個具體人物的四百年姓名流變上 (2) 原住民視角回應「臺灣人的命運」最根本的層次——最早的主人是誰？他的名字還在嗎？ (3) 詩的技法（虛構人物承載真實歷史、姓名作為身分符號、語言流失的具體化）直接可教
- 葉石濤保留但降格為開場，因為 Day 3 → Day 7 的弧線仍然有效（被沉默→為島嶼命名），只是它是鋪路，不是目的地
- 陳秀喜〈臺灣〉放在〈我的姓氏〉之前，作為從論述到詩歌的轉場——搖籃的穩定感為接下來的四百年裂痕做準備
- 〈臺灣的孩子〉收尾——從姓氏不明到孩子歌唱，弧線從暗走向光

## DI 策略（無學生資料）

| 環節 | 入門層 | 挑戰層 |
|------|--------|--------|
| 討論 | 第一層感受題（A-Wu 還活著嗎）+ 第二層連結自己（你的名字誰取的） | 第三層（姓氏不明是悲劇還是解放） |
| 書寫 | 起手式三行（我是 _____ 的孩子/我站在/我聽見） | 向陽風格融合：具體入口 + 形式即內容 + 細節替你說話 |

## 設計決策

1. **葉石濤從核心降為定錨** — 他的論述是「臺灣人是什麼人」的知性答案。但這堂課要的不是知性答案，是讓學生透過詩的體驗自己碰觸這個問題。
2. **〈我的姓氏〉為核心** — 透過原住民四百年的姓名流變探討自我認同，比任何論述都更直接觸及「臺灣人的命運」的根部。
3. **討論三層設計** — 從感受（A-Wu 還活著嗎）到連結（你的名字）到主題（姓氏不明是悲劇還是自由）。不急。每一層都讓學生停留。
4. **向陽風格分析取代泛論散文詩** — 具體分析〈我的姓氏〉的三個技法（具體小入口/形式即內容/細節替你說話），讓學生帶著可操作的工具進入書寫。
5. **〈臺灣的孩子〉收束** — 教師最後朗讀。「你們就是那些孩子。」弧線從黑暗走向光。銜接 Day 8：「你選什麼段落朗讀，就說明了你是什麼人。」

## 關鍵設計反思

v1 的核心失誤是把 Day 7 設計成論述課（宣言技法 + 多文本論證）。這偏離了主課程的精神：八天弧線是用文學體驗建立認同感，不是用論點建立立場。v2 回到文學本位——四首詩不論述、不爭辯，它們用聲音和畫面讓學生感覺到問題的重量。

向陽說：「詩，始於感性，終於理智。」這句話也是 Day 7 的設計原則。先讓學生被聲音觸動（四首詩），再讓理智幫他們找到自己的表達（散文詩書寫）。

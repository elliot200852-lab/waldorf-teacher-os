| **學生發展樣貌或需要（Why, 為何而教）：\**                                                                                                                                                                                                                                                                                                          |
| 台灣的現代文學是奠基在台灣歷史與地理交織的脈絡中，九年級學生在課堂上已經具備了台灣歷史的基礎認識，在生活經驗與課堂上亦對台灣的地理、地質有所了解，因此在這門課堂上，以台灣歷史、地理為坐標，在歷史長河與地貌空間交織中，認識台灣文學。\                                                                                                             |
| \                                                                                                                                                                                                                                                                                                                                                   |
| 從歷史切入，於日本殖民時代產出台灣意識的文學作品；民國之後的文學作品，反應了台灣從農業社會步入工商業社會，經濟起飛後，社會的多元樣貌，人民百姓生活的樣態也如實呈現在文學中。政治更牽涉著人民百姓的意識與行動，尤其當今轉型正義之推動，二二八與白色恐怖真相浮出檯面後，更有相關紀實書寫的文學作品。\                                                 |
| \                                                                                                                                                                                                                                                                                                                                                   |
| 以地理定錨，台灣的高山、海洋、鹽田、漁家、農家、鄉村與城市，多元民族匯聚島上，從種種多元視角而出發的文學情懷。開展學生對台灣現代文學的認識與賞析，藉此思索文學的意義何在，而重要的是增進文學涵養，從文學之窗再次看見台灣的風土、文史、人情與文化。\                                                                                                 |
| \                                                                                                                                                                                                                                                                                                                                                   |
| **教學內容（What, 教什麼）：**\                                                                                                                                                                                                                                                                                                                     |
| 依四大類別來領略台灣與現代文學：\                                                                                                                                                                                                                                                                                                                   |
| 一、地理：農田/森林/海洋/高山\                                                                                                                                                                                                                                                                                                                      |
| 二、族群：原住民/客家/外省/閩南/新移民\                                                                                                                                                                                                                                                                                                             |
| 三、時代：荷據台灣/清領台灣/日本時代/民國：二二八、白色恐怖、戒嚴、解嚴\                                                                                                                                                                                                                                                                            |
| 四、文化：飲食/多元族群\                                                                                                                                                                                                                                                                                                                            |
| \                                                                                                                                                                                                                                                                                                                                                   |
| 舉例說明各類別可閱讀之文學作品：\                                                                                                                                                                                                                                                                                                                   |

| 海洋：廖鴻基《鯨生鯨世》；海洋x原住民：夏曼藍波安《冷海情深》\                                                                                                                                                                                                                                                                                      |
| 高山：楊南郡／徐如林《與子偕行》；劉崇鳳《我願成為山的侍者》\                                                                                                                                                                                                                                                                                       |
| \                                                                                                                                                                                                                                                                                                                                                   |

| 外省：白先勇《臺北人》\                                                                                                                                                                                                                                                                                                                             |
| 閩南：吳晟《農婦》\                                                                                                                                                                                                                                                                                                                                 |

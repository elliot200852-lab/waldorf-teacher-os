慈心華德福高中 台灣文學九年級文本資料庫

1.  說明

    1.  本資料庫是基於慈心九年級主題課程「台灣文學」、「台灣近現代文學」、「台灣近現代歷史」等，所建立之文學文本資料，提供慈心教師在備課選材上的參考及教學使用。

    2.  文本取材原則上以單篇在一萬六千字以內的短篇小說、散文、新詩和古典漢詩為主，較適合每堂課的課程進行。中長篇小說、大河小說的文本則以節錄的方式，選擇其中部份章節，仍是以一萬六千字為上限。

    3.  文本選擇以三大類為原則：台灣人自我認同與追尋及社會發展、族群、議題(當代台灣社會)。

    4.  本資料庫僅供慈心教師團隊在課程與教學上的使用，請使用者勿他傳、嚴守教師自律及相關法律規定。

    5.  歡迎慈心教師團隊夥伴提供適合青少年的台灣文學文本，持續豐富資料庫內容。


## 二、10 原則稽核

| 原則 | 判定 | 說明 |
|------|------|------|
| P01 生成性負載 | PASS | 角色視角書寫要求學生進入非自身視角思考，無標準答案 |
| P02 差異化 | PASS | 每個主要環節（精讀、討論、書寫）都有 DI 標記與具體分層 |
| P03 學習路徑 | PASS | 頭（文本分析 + 手法教學）、心（朗讀感受 + 討論共鳴）、手（書寫練習） |
| P04 年級發展 | PASS | 九年級追求真實與絕對，真人真事回應此需求 |
| P05 口頭書寫 | PASS | 口頭 ≈ 47min / 書寫 22min，比例合理 |
| P06 Follow-up | PASS | 回家讀〈紅鞋子〉標記段落 + 預告明日主題 |

| P09 整體到部分 | PASS | 先整體朗讀感受，再透過討論卡分析局部 |

## 三、DI 適配

- **課型適配**：閱讀課 / 主課程，討論三層 + 書寫三層 = 合理。PASS
- **雙軸覆蓋**：能力 x 動機矩陣（A/B/C/D 具體策略）+ 學習優勢。PASS
- **抽象標籤**：無禁止用語，每個策略都有具體操作描述。PASS
- **策略-內容結合度**：角色選項直接來自文本人物，起手式引用文本細節。PASS

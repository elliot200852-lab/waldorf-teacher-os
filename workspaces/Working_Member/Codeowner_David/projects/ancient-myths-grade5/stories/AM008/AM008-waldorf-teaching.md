# 華德福教學指引

Block 2 波斯的核心轉折：孩子從印度的夢境意識中醒來，開始面對腳下的大地。印度篇章裡，世界是幻象，人的任務是向內冥想、融回宇宙；波斯篇章裡，大地是真實的戰場，人必須勞動、耕種、選擇立場。這堂課要喚起的核心體驗是：世界不只有美好，黑暗也真實存在——但人可以選擇站在光明這一邊。五年級的孩子正開始感受善惡、對錯的內在張力，阿胡拉·馬茲達與阿里曼的永恆對峙，正好為這份張力提供一幅宇宙級的圖景。

# 施泰納框架對應

五年級古文明課程依循人類意識的演化順序排列。印度代表最古老的夢境意識——人尚未與物質世界建立關係；波斯標誌著二元覺醒的起點——人第一次認出光明與黑暗是兩種對立的力量，並在兩者之間承擔道德選擇的責任。善惡二元論是人類意識發展的關鍵轉折：從「一切皆一」的合一狀態，走向「我必須辨別並選擇」的覺醒狀態。這正對應施泰納所描述的十歲轉折——孩子開始意識到自我與世界的分離，需要在道德領域找到自己的立足點。

# 台灣文化呼應

泰雅族的彩虹橋信仰提供了本土對照：只有遵守 gaga（祖訓）的人死後才能走過彩虹橋回到祖靈身邊。善惡不透過宇宙神話的宏大對峙來表達，而是落實在日常行為準則中。台灣民間信仰同樣有善惡審判結構——城隍爺掌管賞罰、閻羅王主持冥界裁決——但傾向「一元多層」，善惡鬼神皆在同一個天道秩序下運作，不同於波斯的宇宙論二元對立。教學切入點：引導孩子思考——「惡」是獨立存在的力量，還是善的缺席？

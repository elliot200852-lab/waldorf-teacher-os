# AM011 投影圖像清單：波斯的智者

### 圖 1：東方三博士朝拜（彼得·保羅·魯本斯工作坊）
- 說明：魯本斯工作坊繪製的三博士朝拜耶穌場景，畫面人物眾多、色彩豐富，三位東方賢者穿著華麗波斯/東方服飾，分別獻上黃金、乳香與沒藥，構圖戲劇性強、對小學生視覺衝擊力大
- 圖片描述：巴洛克風格油畫，畫面中央為聖母懷抱嬰孩，三位深色皮膚或蓄鬚的東方賢者跪伏獻禮，周圍隨從拿著各式禮物容器，背景昏黃燭光烘托神聖氣氛
- 授權：Wikimedia Commons Public Domain
- 連結：https://commons.wikimedia.org/wiki/File:'Adoration_of_the_Magi',_workshop_of_Peter_Paul_Rubens,_The_Hermitage.JPG
- 建議使用時機：故事開場引入「東方來的智者」時投影，讓學生觀察三位賢者的服飾、禮物與態度，引導提問「這些人是誰、從哪裡來、為什麼值得帶著貴重禮物長途跋涉？」

### 圖 2：諾魯茲哈夫特辛節慶桌
- 說明：波斯新年（諾魯茲）傳統的「哈夫特辛」桌，七件以波斯語字母「س」（sin）開頭的物品象徵新年祝福：嫩草芽（sabzeh）象徵新生、醋（serkeh）象徵耐心、蒜（sir）象徵健康等，是瑣羅亞斯德教春分新年文化的現代延續
- 圖片描述：鋪著精緻桌布的節慶桌上，擺放著陶碗中的翠綠嫩芽、裝飾彩蛋、玻璃碗中的金魚、蠟燭、鏡子、各種乾果與香料，色彩繁複且溫暖喜慶
- 授權：Wikimedia Commons CC BY-SA 3.0
- 連結：https://commons.wikimedia.org/wiki/File:Persian_New_Year_Table_-_Haft_Sin_-in_Holland_-_Nowruz_-_Photo_by_Pejman_Akbarzadeh_PDN.JPG
- 建議使用時機：講述諾魯茲春分新年的意義時展示，讓學生辨認桌上的各種物品，討論「為什麼波斯人用嫩草和金魚迎接新年？」，連結光明戰勝黑暗的瑣羅亞斯德教宇宙觀

### 圖 3：法拉瓦哈（Faravahar）翼形聖徽
- 說明：瑣羅亞斯德教最重要的宗教符號。有翼圓盤中央為一位長鬚智者半身像，一手持智慧之環，三層尾羽象徵善思（Humata）、善言（Hukhta）、善行（Hvarshta）三大倫理柱，是瑣羅亞斯德教「祭司智者」角色的圖像核心
- 圖片描述：對稱的有翼圓盤符號，中央為長鬚男性半身像，兩側展開弧形大翼，下方三層尾羽整齊排列，整體造型莊嚴而精確，線條清晰
- 授權：Wikimedia Commons Public Domain
- 連結：https://commons.wikimedia.org/wiki/File:Faravahar.svg
- 建議使用時機：介紹瑣羅亞斯德教祭司階層（Magi）時展示，指出三層尾羽代表的善思善言善行，讓學生理解波斯智者的「智慧」不只是知識，更是生活方式

### 圖 4：波斯波利斯阿帕達納宮殿浮雕——米底人與波斯人朝貢隊列
- 說明：阿契美尼德帝國首都波斯波利斯（Persepolis）的阿帕達納宮殿東側石階浮雕，描繪米底人（Medes）與波斯人的朝貢行列，其中米底人的服裝與頭飾被認為與瑣羅亞斯德教祭司（Magi）的起源族群有直接關聯，是現存最清晰的古波斯祭司階層形象資料之一
- 圖片描述：灰褐色石灰岩浮雕，排列整齊的人物隊伍側身前行，人物穿著長袍、頭戴圓筒形頭帽，有些手持花束或器皿，輪廓雕刻清晰，表情平靜莊嚴
- 授權：Wikimedia Commons CC BY-SA 3.0
- 連結：https://commons.wikimedia.org/wiki/File:Medes_and_Persians_at_eastern_stairs_of_the_Apadana,_Persepolis.JPG
- 建議使用時機：講述瑣羅亞斯德教祭司（Magi）作為波斯帝國的智識階層時展示，讓學生觀察這些人物的莊重姿態，說明他們不只是宗教人員，更是天文、星象、外交的知識守護者

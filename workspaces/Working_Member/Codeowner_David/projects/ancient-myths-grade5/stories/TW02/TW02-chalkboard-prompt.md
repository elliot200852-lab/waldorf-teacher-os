# TW02 黑板畫 Prompt

## 中文 Prompt（說書完成後使用）

構圖：橫幅格式。畫面中央是一道彩虹橋，從左側山峰延伸至右側山峰，橋的弧度柔和飽滿。橋面是微微發光的彩帶——紅、橙、黃、綠、藍，在粉筆的質感下顯得溫柔而莊嚴，不過分鮮豔。

橋的右端（現世的山）：深幽的山影，霧靄繞繞，山色是深青與藍紫；橋腳隱沒在雲霧之中，給人一種「這端仍是人間」的感受。

橋的左端（祖靈之家）：隱約透出溫暖的金黃光暈，彷彿有光從那裡流出來，不是強烈的光，而是深秋午後那種柔和的暖光。

橋的中央：有一個小小的人影，背對著觀者，正在向著光走去。人影非常小——渺小而莊重，讓孩子感受到那個靈魂正在踏上旅途。

橋下方：深深的山谷，以深藍與深紫的暈染表現，隱約有荊棘的剪影。不要畫得恐怖，只是讓孩子感受到「那條路不好走」。

整體氛圍：莊嚴而溫柔，有山的厚重感，也有光的溫暖感。適合五年級孩子——不是童話的甜美，也不是令人恐懼的黑暗，而是一種面對生命真實的靜謐。

黑板底色維持深綠，粉筆色調以暖金、藍紫、雲霧白為主，彩虹色彩淡雅不刺眼。

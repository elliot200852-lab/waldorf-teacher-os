# 投影圖像清單：A003 火的禮物——大屯火山與泥火山（第二版）

## 主圖

### 圖 1：大屯火山群全景
- **描述**：從空中俯瞰大屯火山群，可見七星山主峰與周圍火山地形，噴氣孔冒出白色蒸氣
- **來源**：陽明山國家公園管理處
- **URL**：https://www.ymsnp.gov.tw/News_Content.aspx?n=18005&s=299512
- **授權**：政府公開資料 / 教育用途
- **在故事中的角色**：核心視覺——二十多座火山擠在一起的氣勢
- **展示時機**：說書本文描述「轟，不是一座火山，是二十多座」時

## 輔助圖

### 圖 2：北投溫泉白煙
- **描述**：北投地熱谷或硫磺谷的溫泉蒸氣照片，白煙裊裊升起，水面帶有礦物色澤
- **來源**：大屯火山觀測站 / 陽明山國家公園管理處
- **URL**：https://tvo.ncree.narl.org.tw/knowledge/present/geology/geoheat
- **授權**：政府公開資料 / 教育用途
- **在故事中的角色**：「火山的禮物」具象化——學生可能泡過的那池水
- **展示時機**：說書本文描述溫泉湧出地面時

### 圖 3：烏山頂泥火山
- **描述**：高雄燕巢烏山頂泥火山的灰色泥丘，若有藍焰照片更佳
- **來源**：林務局自然保留區
- **URL**：https://conservation.forest.gov.tw/0000121
- **授權**：政府公開資料 / 教育用途
- **在故事中的角色**：故事高潮——「安安靜靜的藍色火焰」
- **展示時機**：說書本文描述藍色火焰時（配合停頓三秒的靜默）

### 圖 4：臺灣島全景
- **描述**：臺灣島衛星影像或地形圖，可見山脈、平原、火山分布
- **來源**：政府開放資料 / NASA Visible Earth
- **URL**：https://visibleearth.nasa.gov/
- **授權**：Public Domain / CC
- **在故事中的角色**：收尾總結——三部曲（板塊→水→火）雕刻完成的島嶼全貌
- **展示時機**：收尾段落「雕成了一座有峽谷、有平原、有溫泉、有火山的島嶼」時

## 黑板畫步驟（教師手繪參考）

1. **底色**：純黑背景
2. **地層剖面**：底部用暖橘紅色畫岩漿庫（橢圓形光暈），上方覆蓋棕灰色岩層
3. **熱脈**：從岩漿庫向上延伸的細橘色線條（裂縫中的熱）
4. **左側火山**：一座溫和的山形，頂部冒白色蒸氣 + 淡黃色硫磺
5. **右側泥火山**：小小的灰色泥丘，頂端一簇藍色火焰（用藍色粉筆輕點）
6. **色調過渡**：底部暖紅橘 → 頂部冷藍白

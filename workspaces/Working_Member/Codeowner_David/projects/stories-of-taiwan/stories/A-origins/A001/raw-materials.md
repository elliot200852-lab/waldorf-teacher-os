## 核心事實

- **時間**：約 600 萬年前至今（蓬萊造山運動開始）；更早的南澳造山運動約 1.4 億—6,500 萬年前
- **地點**：菲律賓海板塊與歐亞板塊交界處——今臺灣島位置
- **關鍵機制**：弧陸碰撞（Arc-Continent Collision）——菲律賓海板塊攜帶呂宋島弧，以每年約 7–8 公分速度向西北推進，撞上歐亞板塊東南緣
- **事件摘要**：臺灣島並非一直存在，它是兩大板塊碰撞擠壓的產物。約 600 萬年前蓬萊造山運動啟動，原本沉在海底的沉積岩層被巨大的地殼力量推擠隆起，逐漸浮出海面，形成今日擁有 258 座三千公尺以上高山的島嶼。這場造山運動至今仍在進行中。

## 關鍵知識點（交叉驗證完成）

### 1. 南澳造山運動（Ancient Taiwan 前身）
- 超過一億年前，古太平洋板塊隱沒到歐亞板塊之下
- 巨大擠壓力量使大陸棚邊緣的沉積物隆起，形成最早的「古臺灣島」
- 來源：大屯火山觀測站科普頁面、維基百科「台灣地質」條目

### 2. 蓬萊造山運動（Penglai Orogeny）
- 開始時間：約 500–600 萬年前
- 菲律賓海板塊載著呂宋島弧向西北移動（每年 7–8 cm，GPS 觀測約 8.2 cm/年）
- 「弧陸碰撞」——島弧撞上大陸邊緣
- 一度沉入水中的古臺灣島重新浮現，被抬升為將近四千公尺高山
- 蓬萊運動至今仍在進行中
- 來源：科學月刊、國家災害防救科技中心報告、大屯火山觀測站

### 3. 臺灣的地質構造分區（由東向西）
- 海岸山脈、脊樑山脈（中央山脈）、雪山山脈、西部麓山帶、海岸平原、北部張裂帶
- 來源：維基百科「台灣地質」、國家災害防救科技中心

### 4. 臺灣的山脈數據
- 玉山高度：3,952 公尺（東北亞第一高峰）
- 中央山脈：從蘇澳延伸至鵝鑾鼻，直線距離約 320 公里，總長約 500 公里，東西寬約 80 公里，有「臺灣屋脊」之稱
- 全島有 258 座三千公尺以上高山，為全球高山密度最高的島嶼
- 玉山目前仍以每年約 5–7 毫米的速度上升中
- 來源：玉山國家公園管理處、維基百科

### 5. 兩個隱沒帶系統
- 臺灣南方：歐亞板塊朝東隱沒到菲律賓海板塊之下 → 馬尼拉海溝 + 呂宋火山島弧
- 臺灣北方：菲律賓海板塊朝北隱沒到歐亞板塊之下 → 琉球海溝 + 琉球火山島弧 + 沖繩海槽
- 這種「翻轉」的隱沒方向是臺灣地質的獨特之處
- 來源：臺大地質系科普頁面、國家災害防救科技中心

### 6. 持續進行中的故事
- 菲律賓海板塊仍以每年約 8 公分速度向歐亞板塊聚合
- 臺灣每年發生超過 18,000 次地震（多數人體無感）
- 這座島嶼仍在「長大」——但同時侵蝕作用也在削減高度
- 來源：大屯火山觀測站、中央氣象署資料

## AI 研究備註

### 資料缺口
- TCMB（國家文化記憶庫）無 API Key，無法直接搜尋館藏素材；David 已申請中，到手後可補充
- data.gov.tw 與 BOCH API 在當前環境無法連線（Proxy 限制），未取得結構化資料集
- 缺少「臺灣島浮出海面」的動畫或視覺化素材——可考慮自製簡化示意圖

### 爭議點
- 蓬萊造山運動的精確起始時間：不同文獻記載為 500 萬年前或 600 萬年前，學界對此尚有討論。素材包中以「約 500–600 萬年前」表述，保留模糊性。
- 「古臺灣島」是否曾完全沉入海面後又重新浮出，還是始終有部分露出——地質學界仍在研究

### 建議敘事角度
**「島嶼是活的」**——五年級學生正處於第二七年週期末段（以太體發展期），對「活的」、「會呼吸的」自然世界有天然的親近感。建議以擬人化但不失科學準確性的方式，將臺灣島描述為一個「從海底慢慢站起來的巨人」：
1. **開場**：從海底黑暗中開始——兩塊巨大的板塊在深處推擠
2. **高潮**：島嶼浮出海面的那一刻——光線、空氣、第一場雨
3. **結尾驚喜**：「這座島現在還在長高，每年長高 5–7 毫米——就像你們一樣」

這個角度同時為下一篇故事（史前遺址/最早的足跡）預留了自然過渡：島嶼準備好了，等待第一批踏上這片土地的人。

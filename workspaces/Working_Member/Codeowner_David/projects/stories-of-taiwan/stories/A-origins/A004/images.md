# 投影圖像清單：A004 石頭裡的秘密——卑南遺址

## 主圖

### 圖 1：月形石柱
- **描述**：卑南遺址公園內的月形石柱，高 349 公分，板岩材質，頂端殘存半圓弧形孔洞，為遺址最具代表性的地標
- **來源**：Wikimedia Commons
- **URL**：https://commons.wikimedia.org/wiki/Category:Beinan_Cultural_Park
- **授權**：CC BY-SA 3.0 / CC BY-SA 4.0
- **在故事中的角色**：遺址的象徵——三千五百年前的建築遺構，開場時展示，建立視覺錨點
- **展示時機**：說書本文第一段，怪手挖出石棺後

## 輔助圖

### 圖 2：卑南遺址考古發掘現場
- **描述**：考古發掘現場照片，可見整齊排列的石板棺群與考古工作人員作業情形
- **來源**：國立臺灣史前文化博物館
- **URL**：https://www.nmp.gov.tw/cl.aspx?n=8830
- **授權**：教育用途（博物館官方資源）
- **在故事中的角色**：呈現搶救考古的實際場景——刷子、竹籤、烈日下的工作
- **展示時機**：說書本文中段，描述宋文薰與連照美九年搶救時

### 圖 3：人獸形玉玦
- **描述**：卑南遺址出土國寶級文物，片狀縷雕玉飾，雙人頂一獸造型，為臺灣獨有的玉器形式
- **來源**：開放博物館（中研院）
- **URL**：https://openmuseum.tw/muse/digi_object/75cc944aa3a925121fc7e7b933a91d34
- **授權**：依各館規定（教育展示用）
- **在故事中的角色**：三千年前的手藝巔峰——讓學生驚嘆「石器時代的人能做出這麼精細的東西」
- **展示時機**：說書本文描述「特別的東西」時，揭曉神秘物件

### 圖 4：卑南遺址公園全景
- **描述**：遺址公園草坡全景，含月形石柱與遠山，呈現遺址現今安靜的樣貌
- **來源**：Wikimedia Commons / 臺東觀光旅遊網
- **URL**：https://commons.wikimedia.org/wiki/Category:Beinan_Cultural_Park
- **授權**：CC BY-SA 3.0 / CC BY-SA 4.0
- **在故事中的角色**：收尾用——「它還在等」的意象需要一個寬闊安靜的畫面
- **展示時機**：收尾段落，描述月形石柱等待的意象時

## 黑板畫步驟（教師手繪參考）

1. **底色**：深藍黑色背景（夜空感）
2. **月形石柱**：畫面中央偏左，一根高瘦的長方形石柱，頂端有半月形缺口，用淺灰與白色粉筆勾勒
3. **玉珠**：石柱右下方散落數顆小圓點，以翠綠色粉筆輕點
4. **星空**：背景點綴幾顆星星，暗示時間的久遠
5. **大地線**：底部以褐色橫線暗示土地層

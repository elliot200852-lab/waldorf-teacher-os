# 投影圖像清單：A005 海的搖籃——黑潮與臺灣

## 主圖

### 黑潮流經臺灣東岸路線圖
- 描述：一張清楚標示黑潮從菲律賓海域北上、沿臺灣東岸流動的洋流示意圖，標示綠島、蘭嶼、花蓮、宜蘭等地
- 來源：海洋國家公園管理處 / 維基百科（Kuroshio Current 條目）
- URL：https://en.wikipedia.org/wiki/Kuroshio_Current
- 授權：CC BY-SA（維基百科）/ 教育用途（政府機關）
- 在故事中的角色：讓學生視覺化「海中的河」的流向與規模
- 展示時機：說書稿「海中的河」段落——介紹黑潮路線時

## 輔助圖

### 1. 黑潮深藍色海水近照
- 描述：從船上拍攝的黑潮海域照片，展現其特有的深靛藍色水色，與一般海水的淺藍/綠色形成對比
- 來源：黑潮海洋文教基金會
- URL：https://www.kuroshio.org.tw/
- 授權：教育用途（待確認）
- 在故事中的角色：讓學生理解「為什麼叫黑潮」——視覺證據
- 展示時機：說書稿「黑潮的顏色」段落——解釋水色深邃的原因時

### 2. 臺灣海峽黑水溝衛星影像
- 描述：從衛星拍攝的臺灣海峽影像，可見澎湖水道（黑水溝）處海水顏色明顯較深的帶狀區域
- 來源：鄭明典（中央氣象署前署長）分享之衛星影像 / 東森新聞
- URL：https://news.ebc.net.tw/news/living/57120
- 授權：教育用途
- 在故事中的角色：讓學生看到「黑水溝」不只是傳說，從太空都看得到
- 展示時機：說書稿「黑水溝」段落——講到先民渡海時

### 3. 珊瑚礁水下生態照
- 描述：臺灣東部（綠島或蘭嶼）海域的珊瑚礁水下照片，展現多彩的珊瑚與魚群共生畫面
- 來源：海洋保育署 / 農業知識入口網
- URL：https://www.oca.gov.tw/ch/home.jsp?id=345&parentpath=0,295,342
- 授權：政府公開資料 / 教育用途
- 在故事中的角色：讓學生驚嘆於「黑潮的禮物」——珊瑚礁的繽紛生命
- 展示時機：說書稿「黑潮的禮物」段落——講到珊瑚和海洋生物時

### 4. 臺灣周邊海流總覽圖
- 描述：標示臺灣四周主要海流（黑潮主流、中國沿岸流、南海暖流）的示意圖，讓學生理解臺灣在海流交匯處的特殊位置
- 來源：海洋委員會 / 教科書教學圖
- URL：https://www.oac.gov.tw/
- 授權：教育用途
- 在故事中的角色：補充說明臺灣被多條海流環繞的地理格局
- 展示時機：選用——可在開場時搭配地圖使用，或課後回顧時展示

## 投影建議

展示順序：
1. 開場：黑潮流經路線圖（讓學生先「看到」這條海中的河）
2. 黑潮顏色段落：深藍色海水近照
3. 黑水溝段落：海峽衛星影像
4. 珊瑚段落：珊瑚礁水下照片

## 替代方案（無法投影時）

- 列印 A3 大小的黑潮路線圖，貼在黑板旁邊
- 用粉筆在黑板上畫出臺灣輪廓 + 黑潮箭頭（藍色粉筆）
- 珊瑚礁照片可列印數張 A4，在教室傳閱

# 投影圖像清單：A002 水的雕刻刀——立霧溪與太魯閣（第二版）

## 主圖

### 圖 1：太魯閣峽谷垂直岩壁
- **描述**：立霧溪穿越大理岩峽谷，兩側近乎垂直的白色岩壁高聳，中間僅留窄窄的河道與天空
- **來源**：太魯閣國家公園管理處
- **URL**：https://www.taroko.gov.tw/cp.aspx?n=5760
- **授權**：政府公開資料 / 教育用途
- **在故事中的角色**：核心視覺——「水切出來的」垂直峽谷
- **展示時機**：說書本文描述「最驚人的事——切石頭」時

## 輔助圖

### 圖 2：大理岩紋理特寫
- **描述**：太魯閣大理岩近距離照片，可見白色與灰色交替的層狀紋理（變質作用留下的痕跡）
- **來源**：太魯閣國家公園管理處
- **URL**：https://www.taroko.gov.tw/cp.aspx?n=5762
- **授權**：政府公開資料 / 教育用途
- **在故事中的角色**：讓學生看到大理岩的「水墨畫」紋路，理解這些石頭的前世（珊瑚與貝殼）
- **展示時機**：說書本文描述大理岩的身世時

### 圖 3：太魯閣峽谷俯瞰全景
- **描述**：從高處俯瞰立霧溪峽谷，可見河流蜿蜒於深谷之中，兩側山壁陡峭
- **來源**：Wikimedia Commons / 文化部國家文化記憶庫
- **URL**：https://tcmb.culture.tw/zh-tw/detail?indexCode=online_metadata&id=1938429
- **授權**：CC / 教育用途
- **在故事中的角色**：鳥瞰視角——讓學生理解峽谷的規模（深度超過 1,000 公尺）
- **展示時機**：說書本文描述「兩百三十萬年」切出的峽谷深度時

### 圖 4：嘉南平原
- **描述**：嘉南平原鳥瞰照片，展示平坦的農田延伸至遠方，與背景的丘陵形成對比
- **來源**：政府開放資料 / 臺灣地理影像
- **URL**：http://pc183.hy.ntu.edu.tw/gwater/d2.php
- **授權**：教育用途
- **在故事中的角色**：收尾用——「水不只破壞，也建設」的證據
- **展示時機**：說書本文描述水帶走碎屑鋪成平原時

## 黑板畫步驟（教師手繪參考）

1. **底色**：純黑背景
2. **峽谷**：畫面中央，兩道灰白色垂直岩壁從下往上延伸，中間留一道窄縫
3. **大理岩紋理**：岩壁上用白色粉筆畫水平細線（層狀紋理）
4. **河流**：岩壁底部用碧綠色（turquoise）粉筆畫一條細窄的河
5. **雨滴**：左上角一顆銀色水滴，從天空往下落
6. **平原**：右側下方用金綠色暈染出平緩的沖積平原，延伸至海面

## 投影建議

- **展示順序**：圖 1 → 圖 2 → 圖 3 → 圖 4（跟隨說書稿段落時序）
- **無投影替代方案**：列印 A4 彩色圖片 4 張。圖 2（大理岩紋理）建議放大列印，讓學生近距離觀察。也可帶一塊實際的大理石樣本進教室。

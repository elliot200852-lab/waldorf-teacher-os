# 國家文化記憶庫 OpenAPI 申請回覆

**案件編號：CU20260318001**
**日期：2026-03-18**

---

感謝來信，以下補充說明：

**1. 預計使用的教學情境與方式**

本人任教於宜蘭慈心華德福實驗學校（依實驗教育三法運作），教授七至九年級的英文、歷史、人文與社會及跨領域主題課程。目前正在開發一套教師備課協作系統（TeacherOS），協助全校教師進行課程設計與差異化教學。

國家文化記憶庫的資料將用於以下教學情境：
- **歷史與人文課程備課**：教師在設計台灣史、地方文史、跨領域主題課程時，透過系統查詢相關的歷史影像、文物、口述紀錄等素材，融入課程設計
- **課堂展示與討論**：將查詢到的文化素材（圖片、文獻摘要）嵌入教學簡報或學習單，作為課堂討論的引導材料
- **學生專題研究**：九年級學生進行台灣主題專題時，教師預先篩選適合的資料集，提供學生作為研究素材來源

使用方式為教師端備課查詢，非學生直接操作 API。

**2. 預計取用之資料類型與規模**

主要取用類型：
- 歷史影像與老照片（地方文史、日治時期、戰後台灣）
- 文物與藝術品資料（民俗文物、原住民族文化）
- 口述歷史與人物傳記摘要
- 地方文化資產（宜蘭在地文史優先）

規模預估：每次備課查詢約 20-50 筆資料，篩選後實際使用 5-10 筆。全校約 10 位教師共用系統。

**3. 預估資料取用頻率**

以定期使用為主，非短時間大量存取：
- 學季初期（每學季約 2-3 週密集備課期）：每日約 5-10 次查詢
- 日常備課期間：每週約 3-5 次查詢
- 非備課期間：極少使用

整體而言，屬於低頻、小量的教育用途查詢。

**4. 預計使用之固定 IP**

目前學校與教師個人工作環境使用浮動 IP。若貴單位需要固定 IP 作為存取控管，請告知是否有替代方案（如 API Key 認證），或我方可評估申請固定 IP 的可行性。

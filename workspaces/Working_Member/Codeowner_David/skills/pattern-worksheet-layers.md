# Pattern: 學習單 D/C/B 三層設計

## 洞察

開放式寫作（Free Writing）作為學習單最高層，門檻過高——連 A/C 類學生也卡住，因為缺乏具體錨點。把最高層從「自由寫作」改為「有框架的應用」（如造句、改寫），學生完成率顯著提升。真正的高挑戰改由課內延伸任務二選一提供，而非寫在學習單上變成沒人做的裝飾。

## 來源情境

9C 英文 Week 6-2 連接詞學習單。原設計四層（D→C→B→A），Task A 是 Free Writing（用四個連接詞寫一段短文）。實際使用時 A 類學生也覺得突兀——前三層都是有框架的練習，最後一層突然變成開放寫作，認知切換太大。刪除 Task A 後，學習單更流暢；高挑戰改到課堂延伸任務（對話創作 vs 情境配對二選一）。

## 適用場景

- 任何分層差異化學習單
- 特別是文法、詞彙等技能型練習的學習單
- 當你想在學習單裡放「高挑戰層」時，先問自己：這層跟前面的層是同一種認知操作嗎？

## 操作要點

- D 層（基礎）：填空、配對、選擇——最低門檻，有明確答案
- C 層（進階）：有提示的造句、改寫——需要組合但有鷹架
- B 層（應用）：情境應用、短篇改寫——有框架但需要判斷
- 刪除純開放式寫作層——改放到課堂延伸任務
- 延伸任務設計為二選一（如：對話創作 vs 情境配對），讓不同學習優勢的學生有選擇
- Task B 如果涉及中英對照，加入中文填空欄降低門檻

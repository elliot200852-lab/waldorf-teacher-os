# skill: recitation — 朗讀教學設計

為指定班級設計一堂完整的朗讀教學活動，涵蓋文本選擇、朗讀節奏與差異化分組。

## 參數

可選：班級代碼 + 語言。若未提供，詢問補齊。

## 執行步驟

### Step 1 — 載入脈絡

1. `{workspace}/projects/class-[班級]/[科目]/session.yaml`（確認目前教學進度）
2. `{workspace}/projects/class-[班級]/students.yaml`（DI 分類與學習優勢）

### Step 2 — 收集設計資訊

詢問（若對話中已有資訊，直接使用）：

- 「這次朗讀的目的是什麼？」（語感培養 / 內容理解 / 戲劇準備 / 節奏體驗）
- 「有想用的文本嗎？還是需要我推薦？」
- 「預計花多少時間？」（10 分鐘暖身 / 20 分鐘主活動 / 整堂課）

### Step 3 — 文本選擇建議

根據年級發展提供文本建議：

| 年級 | 文本特質 | 範例方向 |
|------|---------|---------|
| 七年級 | 節奏鮮明、韻律感強、冒險主題 | 史詩節選、航海詩、英雄敘事 |
| 八年級 | 情感張力、對話衝突、社會正義 | 演說詞、戲劇獨白、革命詩 |
| 九年級 | 內在矛盾、哲學深度、光暗對比 | 莎劇節選、現代詩、存在主義散文 |

### Step 4 — 設計朗讀流程

輸出格式：

---

**朗讀教學設計｜class-[班級]｜[語言]**

**文本：** [文本名稱與出處]

**朗讀流程：**

| 階段 | 活動 | 時長 | DI 備註 |
|------|------|------|---------|
| 暖身 | [活動] | X 分 | |
| 示範 | [活動] | X 分 | |
| 練習 | [活動] | X 分 | |
| 呈現 | [活動] | X 分 | |

**差異化策略：**
- A 型：[策略]
- B 型：[策略]
- C 型：[策略]
- D 型：[策略]

**教師備註：** [朗讀指導要點、聲音運用提示]

---

## 注意事項

- 朗讀是具身學習（Embodied Learning）的核心活動——不只是「讀出聲音」，而是讓語言穿過身體
- B/D 類學生可能對站在眾人面前朗讀感到焦慮，提供漸進式參與選項（先同聲齊讀→小組→個人）
- 七年級需要更多肢體動作配合朗讀；九年級可以處理更長的靜態朗讀段落

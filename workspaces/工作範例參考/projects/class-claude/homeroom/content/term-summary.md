# HM Block 4：期末總結與發展評估

> **文件用途**：根據春季學期的全部導師紀錄（HM Blocks 1-3），為每位學生編寫期末發展總結。重點聚焦於「頭、心、手」的動態變化，並為親師生三角溝通提供具體的語言與觀察。本文件是導師期末評語的基礎草稿。

**完成日期**：2026-06-12（期末考前）

---

## 春季學期班級概覽

**班級代號**：Claude 班
**年級**：八年級（13-14 歲）
**學期**：2025-2026 下學期（春季，2月中～6月中）
**總人數**：18 人
**導師**：Claude

### 班級整體發展軌跡

這個春季，Claude 班展現了顯著的凝聚力與民主意識的萌芽。從班規共製的初體驗，到修復式對話的實踐，再到班委會的自主運作，同學們正在經歷「從被動接受」到「主動參與」的心理轉變。

**星芒體的特徵**在班級中清晰可見：
- 對「為什麼」的執著追問（例如在班規討論中）
- 對因果關係的敏銳觀察（例如「他的語氣傷害了他的感受」）
- 對同儕關係的高度關切，友誼小圈的穩定形成
- 初步的倫理意識萌芽（對公平性的關注、對他人感受的同理）

**班級凝聚力指標**：中等偏高。同儕間大多能和諧互動，內向與外向學生的界線正在透過活動而柔和。衝突較少，且多能透過對話而非對抗解決。

## 樣本學生發展評估

### 學生 1：CL-02 陳宇翔 | 班長

#### 個人檔案
- **主要特質**：領導傾向強、邏輯思維清晰、行動力高、個人主義色彩較濃
- **觀察來源**：班規共製、班會主持、修復式對話、課堂互動

#### 頭（Thinking / 認知發展）
**星芒體敏銳化的表現：**
陳宇翔在春季展現了顯著的因果思維與批判能力。
- 在班規共製中，他率先提出「修復」概念，說明他能思考「懲罰之外的邏輯」
- 在家族故事分享時，他敘述了父親從農民轉為工程師的奮鬥軌跡，邏輯清晰，說明他能理解多層次的人生決策
- 在班會中，他嘗試以「預算」與「預算」的角度評估班級外出方案，但有時快速決議，未充分聆聽其他聲音

**觀察深度**：他的思考較多聚焦在「系統」與「效率」，對於「感受」與「多元性」的考量仍在開發。

**發展潛力**：強。邏輯思維的基礎穩固，後續重點是培養「複雜性思維」（即同時容納多個視角）。

#### 心（Feeling / 情感發展）
**同理心的初期開發：**
- 在修復式對話中，他願意承認錯誤（「我用強硬的語氣傷害了你」），這反映出他開始能反思自己的行為對他人的影響
- 在登山活動中，他主動關照體力較弱的同學，顯示他的實際行動能力跟上了同理心
- 但在班會與日常互動中，他有時展現「急於結束對話、快速決策」的傾向，可能反映出他的內在焦慮或對控制的渴望

**導師詮釋**：他的星芒體敏銳化正在進行。他對「因果」與「責任」的關注，已經打開了同理心的大門。下一步是幫他認識「感受的複雜性」（別人可能同時感受到被傷害和被理解）。

**發展潛力**：中等。他有學習的意願，但需要導師與同儕的持續反饋。

#### 手（Doing / 行動能力與社會參與）
**領導行動的初步實踐：**
- 他承擔了班長角色，並在班會中試圖推動班級決策
- 他在修復式對話後，主動與郭宥辰修復關係
- 他在家族故事分享後，主動提議進一步的班級活動

**行動的品質**：行動力強，但有時欠缺「邀請參與」的細緻度。他傾向於「我想到一個主意，我們就這樣做」，而非「大家怎麼想？」

**發展潛力**：強。已有自主行動的基礎，後續重點是「服務領導」的內化（即「我的行動是為了班級的整體福祉」，而非「我要控制結果」）。

#### 期末評語草稿
陳宇翔是班級的責任擔當者。他的邏輯清晰、行動力強，已展現了初期的領導潛力。春季中，他在班規共製、班會運作與修復式對話中，逐漸體驗「個人力量如何影響他人」。他的下一步發展，是將「效率導向的思考」擴展到「同理心導向的思考」——也就是說，考量他人的感受與多元觀點，而非只追求快速決策。我建議他在暑期與新學期初，繼續參與班級自治的實踐，並在導師的引導下反思「好的領導長什麼樣子」。

### 學生 2：CL-03 王小芮 | 內向者的歸屬之旅

#### 個人檔案
- **主要特質**：內向、細心、聆聽能力強、信心欠缺、渴望被看見但又退縮
- **觀察來源**：班規共製、家族故事分享、陪餐時刻、日常互動

#### 頭（Thinking / 認知發展）
**靜默中的深度思考：**
王小芮在課堂上不常舉手，但她的筆記與小組內的貢獻度很高。
- 在班規共製中，她擔任紀錄者角色，認真記錄組內對話，顯示她能在「支持他人」的角色中找到安全感
- 在家族故事訪談時，她在小圈子環節首度詳細分享。導師事後察覺，她的故事敘述很有邏輯，說明她事前做了充分的思考與準備
- 在英文課的 peer feedback 環節，她的評論很細緻，捕捉到寫作者的意圖與細節

**觀察深度**：她的思考品質不遜於外向同學，但她更傾向於「深入一點」而非「廣泛發言」。

**發展潛力**：中等。她的思考能力很好，後續重點是建立「我的想法值得被聽見」的信心。

#### 心（Feeling / 情感發展）
**信心建立的初期跡象：**
- 在小圈子聆聽環節中，她顯著放鬆。與 CL-14 的自發互動（談論動畫），說明她在小而安全的環境中能開放。
- 在陪餐時刻，她與 CL-14 的靠近，反映出她在班級中正在形成「信任關係」
- 在課堂中，她對他人的故事有細緻的反應（眼神接觸、點頭確認），顯示她的情感感受力很高

**導師詮釋**：她的內向不是孤立，而是「選擇性參與」。她需要的不是被推入外向，而是被肯定「安靜的貢獻也很有價值」。

**發展潛力**：中等偏高。她有情感深度，需要的是環境的支持與同儕的確認。

#### 手（Doing / 行動能力與社會參與）
**微觀參與中的存在感：**
- 她在班規共製中的「紀錄角色」是一個關鍵的參與方式。這讓她既能貢獻，又不必承受直接發言的焦慮
- 在小圈子聆聽中，她的參與是「被動邀請」而非「自動上前」，但一旦加入，她就能專注
- 在家族故事分享的小圈子中，她的發言篇幅增加了，說明信心在逐步建立

**行動的品質**：行動力仍在開發。她傾向於被邀請而非主動，但每次被邀請時，她都認真回應。

**發展潛力**：中等。她需要更多「成功的發言經驗」來累積信心。建議導師在接下來的活動中，持續提供「安全的發言機制」（例如寫作、小圈子、書面回饋）。

#### 期末評語草稿
王小芮是班級的靜默觀察者，也是深度思考者。她的內向特質往往被外向同學的聲量所遮蔽，但在小圈子與書面作業中，她展現了清晰的思維與細緻的感受力。春季中，她在家族故事分享的小圈子環節中跨出了一大步，與同儕建立了信任關係。我鼓勵王小芮在新學年中繼續參與「小而安全」的發言機會，並在導師的陪伴下，逐步擴展至大集體環境。內向不是需要改進的缺點，而是她獨特的資源——她能傾聽、能思考、能給予細緻的回饋。讓我們一起慶祝這些力量。

### 學生 3：CL-15 洪晨翊 | 情緒調節的自我認識之路

#### 個人檔案
- **主要特質**：高敏感、情緒波動大、深層感受豐富、衝突焦慮高、渴望安定
- **特殊需求**：情緒調節困難（IEP 目標）
- **觀察來源**：晨圈、修復式對話、登山活動、個別談話、心理諮詢師回報

#### 頭（Thinking / 認知發展）
**衝突認知的敏銳化：**
洪晨翊對人際關係中的微妙變化有超乎預期的察覺。
- 在修復式對話工作坊中，他能准確識別自己的情緒狀態（「我害怕他們又會吵」），這反映出他的內省能力很強
- 在登山後的休息時刻，他與導師的對話中，能夠分辨「不行」是指體力還是心理，說明他開始有「自我認識」的意識
- 在家族故事訪談時，他選擇的訪談對象與話題（雖然敘述時略顯零散），反映出他對根源的關注

**觀察深度**：他的認知能力並無障礙，但他的思考往往被情緒激動所打斷。後續的重點是「從衝動思考轉向反思思考」。

**發展潛力**：中等。認知潛力存在，需要情緒調節工具的支持。

#### 心（Feeling / 情感發展）
**從情緒掙扎到自我命名的起點：**
這是洪晨翊春季發展的核心。
- 開學初的「調整期」顯示他對環境變化的高敏感性。導師的「非強制」邀請模式（給他說話的權利，而非要求），幫助他逐漸放鬆
- 修復式對話中，他能說出「我擔心」，這對一個情緒調節困難的孩子來說是巨大的進步。它表示他開始能「命名情緒」，而非被情緒淹沒
- 登山活動中，他經歷了「想放棄 → 同儕支持 → 克服 → 成就感」的完整弧線。這個經驗對他來說可能是深刻的，因為它表示「衝動不等於真相」

**導師詮釋**：他的高敏感性既是挑戰，也是力量。當他的情緒被看見與理解時（如同在登山中 CL-13 的邀請），他能展現出堅韌與同理心。

**發展潛力**：高。如果能持續獲得一致的情緒支持與自我認識的機會，他的星芒體敏銳化可能會轉向非常有同理心的方向。

#### 手（Doing / 行動能力與社會參與）
**從退縮到微觀參與：**
- 開學初，他試圖躲避班級活動（例如家族故事的小圈子）。導師的溫和邀請（而非強制）逐漸改變了他的參與意願
- 在修復式對話中，他以「聆聽者」與「發聲者」的雙重身份參與，顯示他能在被支持的環境中承擔責任
- 登山活動是轉捩點。他從「我不行」到「我走在最前面」，這個身體與心理的同步轉變，可能會強化他的行動信心

**行動的品質**：行動力在逐步建立。他需要的不是「推動」，而是「邀請」與「陪伴」。

**發展潛力**：中等偏高。如果情緒調節工具能持續強化，他的行動力會逐步增加。

#### 特殊支持的追蹤
**心理諮詢師合作進展：**
- 每週 1 次個案談話，重點是「情緒識別與命名」
- 導師已在教室設置「靜心角」供他使用
- 家長配合度高，在家中也避免進一步激化他的焦慮

**建議下學年的持續支持：**
- 繼續每週諮詢談話
- 在班級活動設計時，預留「退出與重新進入」的空間（例如登山時可在中途休息站停留，而非強制完成全程）
- 定期檢視「情緒調節工具」的有效性，並根據他的成長調整

#### 期末評語草稿
洪晨翊是班級的深層感受者。他的高敏感性使得他對班級的人際變化有敏銳的察覺，也使得他在情緒激動時容易陷入困境。春季中，他透過導師、心理諮詢師與同儕的支持，開始學會「命名自己的情緒」——這是情緒調節的第一步。登山活動中從「我不行」到完成的經驗，對他來說是一個里程碑。我相信，在持續的支持下，他的敏感性會逐漸轉化為同理心與社會意識。暑期建議家長與他一起慶祝這些微小的進步，並在新學年繼續深化「情緒識別」的工作。

### 學生 4：CL-07 劉冠廷 | 融合與親和力

#### 個人檔案
- **主要特質**：讀寫困難（學習障礙）、邏輯思維清晰、親和力強、幽默感足、融合度高
- **特殊支持**：國語文補救教學（每週 2 次）
- **觀察來源**：班級活動、小組協作、陪餐時刻、特教老師回報

#### 頭（Thinking / 認知發展）
**邏輯思維的隱藏強項：**
劉冠廷的讀寫困難往往掩蓋了他的認知能力。
- 在家族故事訪談中，他能邏輯清晰地選擇訪談對象、設計問題、記錄故事。雖然記錄的「量」較少，但「質」很高
- 在班規共製的小組討論中，他用簡短的詞彙表達想法，但他的邏輯方向與同組同學一致，說明他理解了討論的脈絡
- 在修復式對話工作坊中，他靜靜聆聽，之後主動詢問導師「他們現在好了嗎」，反映出他能追蹤人際動態的複雜性

**觀察深度**：他的思維品質不遜於無讀寫困難的同學。關鍵在於如何設計「非文字」的表達機制。

**發展潛力**：高。邏輯能力的基礎很好，需要的是「多元表達方式」的開發。

#### 心（Feeling / 情感發展）
**親和力與同理心的自然展現：**
- 在班級中，他展現了高度的同理心。修復式對話後他會關切「他們現在好了嗎」，說明他能感受他人的情緒狀態
- 在登山活動中，同學們自然地等待他，也樂意與他同行。這反映出他已獲得班級的接納
- 在陪餐時刻，他的幽默感與親切態度，使得不同圈子的同學都樂意與他互動

**導師詮釋**：他的讀寫困難並未影響他的人際能力。相反，他因為學習中的挑戰，可能更能同理他人的困難。

**發展潛力**：很高。他的情感品質已經是班級的正資源。

#### 手（Doing / 行動能力與社會參與）
**融合參與的模範生：**
- 他在所有班級活動中都充分參與，方式是「帶著自己的步調，而非隱身」
- 在小組任務中，他會尋找自己能貢獻的方式（例如在紀錄時，他畫圖而非寫字）
- 在班級互動中，他主動關照他人（例如在登山時，同組同學樂意扶他、陪他）

**行動的品質**：行動力強，且品質很好。他不是「被帶著走」的被動參與者，而是「主動尋找貢獻方式」的主人。

**發展潛力**：很高。他已展現了融合參與的實踐，後續可在「領導性貢獻」（例如帶領小組活動）上擴展。

#### 特教支持的追蹤
**與特教老師的協作進展：**
- 國語文補救教學的成果良好。他在認識新詞與理解文意上有進步
- 導師課中的「小組任務內嵌式支持」，讓他既能參與班級活動，又能獲得適切的協助
- 沒有出現「隔離」或「標籤化」的問題。班級對他的多元學習方式的接納度很高

**建議下學年的持續支持：**
- 繼續國語文補救教學，但逐步提高「獨立閱讀」的比例
- 在班級活動中持續提供「多元表達機制」（圖像、口頭、實作）
- 鼓勵他在非文字領域（例如美術、音樂、體育）中展現強項

#### 期末評語草稿
劉冠廷是班級的親和力典範。他的讀寫困難並未成為他參與班級生活的障礙，反而因為他的同理心與幽默感，他成為了班級的「潤滑劑」。春季中，他在班級活動中充分參與，用他獨特的方式（畫圖、口述、實作）貢獻想法。他的故事告訴我們：差異不是劣勢，而是不同的資源。他的邏輯清晰，只是需要「非文字」的表達通道。我鼓勵他在新學年中繼續發展他的強項領域，同時透過補救教學，逐步增進他的讀寫能力。最重要的是，他應該被看見為「一個完整的、被班級接納的、有貢獻的班級成員」——這一點，他已經做到了。

## 班級發展優勢與後續工作建議

### 班級的三大優勢
1. **民主參與的初步經驗**：班規共製、班會、修復式對話，已使得同學開始內化「我是班級決策的一部分」的意識
2. **修復文化的萌芽**：同學正在學會「衝突可以透過對話解決」，而非透過對抗或冷戰
3. **多元接納的開始**：內向者、讀寫困難者、情緒調節困難者，都在班級中找到了位置。這是包容性班級文化的基礎

### 後續三年（9-11 年級）的發展方向
- **9 年級**：深化民主實踐（例如更複雜的班會議題）；強化修復文化；探索個人主題與社會參與
- **10 年級**：發展社會實踐項目（例如社區服務）；深化倫理思維；為畢業準備鋪路
- **11 年級**：畢業專案；生涯探索；班級歷史的總結與展望

### 導師的自我提醒
- 持續記錄全班的日常觀察（防呆機制），確保每位學生都被看見
- 定期與心理諮詢師、特教老師、英文等科課教師協作，建立跨學科的支持網絡
- 親師溝通應以「肯定與邀請」為主，而非「問題導向」
- 對於內向、特殊需求的學生，提供「多元參與方式」，而非強制外向

## 親師生三角對話的建議架構

期末時，導師應與每位家長及學生進行「三方對話」，內容包括：

**導師視角**（來自 HM Blocks 1-3 的觀察）：
- 「在班級中，我看到你的孩子……」（具體事例）
- 「這反映了他/她在星芒體發展中的以下進展：……」

**家長視角**（邀請家長分享）：
- 「在家裡，你們發現到什麼變化嗎？」
- 「暑期時，你們希望孩子在哪些方面有所成長？」

**學生視角**（直接邀請學生表達）：
- 「這個春季，你在班級中學到了什麼？」
- 「下學年，你想要成為怎樣的班級一份子？」

這三個視角的整合，將形成對學生發展的整全性認識，也為下學年的教育規劃提供基礎。

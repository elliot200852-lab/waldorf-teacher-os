# HM Block 1：春季班級動力規劃與工作時程

## 季候背景與班級狀態

**學年/學期**：2025-2026 年度 下學期（春季，2月中～6月中）
**班級**：Claude 班（八年級，18 人）
**導師**：Claude
**上次更新**：2026-03-03

這個春季是八年級同學踏入「星芒體敏銳化」的關鍵時段。他們開始渴望因果關係的解釋，對世界秩序產生執著般的好奇；同時，同儕歸屬感的需求達到高峰。班級內部的小圈子結構也趨於穩定。

本季的導師工作重點是：
1. **維持班級凝聚力**：透過循環式的共同課題與對話圈，深化同學對彼此的理解
2. **支持自主發展**：鼓勵學生在班級事務中承擔更多責任（如班會議事、活動企劃）
3. **觀察衝突與修復**：敏銳捕捉青春期衝突，運用修復式正義對話化解隔閡
4. **銜接畢旅主題**：將班級歷程與共同冒險願景編織在一起

## 導師課堂週期規劃

**課堂時間**：每週一 08:00～09:00（晨圈整合）+ 每週三 14:30～15:30（班級圓圈時間）

### 春季主題循環（12 週規劃）

#### Phase 1：重塑與根的尋找（2 月中～3 月上旬，3 週）
- **Week 1-2**：晨圈重建儀式、班規共製（體驗民主）
- **Week 3**：家族故事分享會（準備階段）

#### Phase 2：認識彼此，深化班級動力（3 月中～4 月初，4 週）
- **Week 4-5**：班會議事、班級事務透明化
- **Week 6**：中段考支持週（壓力管理、學習策略圓圈）
- **Week 7**：衝突化解工作坊（NVC、修復式對話練習）

#### Phase 3：畢旅願景與自主性（4 月中～5 月初，3 週）
- **Week 8**：畢旅夢想分享（個人期待、班級願景）
- **Week 9-10**：籌委會工作坊與民主決策

#### Phase 4：共創與統整（5 月中～6 月初，2 週）
- **Week 11**：班誌共創工作坊
- **Week 12**：畢旅前的最後圓圈（彼此寄語）

## 班級動力評估與支持對象

### 整體班級風貌
- **凝聚度**：中等偏高。班級內已形成 3～4 個友誼小圈（較內向組、運動組、藝術組、學術組）
- **衝突傾向**：輕度。偶有小紛擾，但多能自行化解。尚未出現嚴重霸凌跡象
- **參與度**：參差不齊。約 12 人積極參與班級事務；4～5 人較為被動；1～2 人需要特別鼓勵

## 導師工作重點與自我反思項目

### 本季三大重點工作
1. **修復式正義文化的內化**：不止於「懲罰」型班規，而是教會同學如何承擔責任、修復傷害
2. **多聲部民主實踐**：班會、籌委會、圓圈對話，讓每個聲音被聽見
3. **觀察星芒體的敏銳化**：記錄同學在邏輯思考、批判能力、人際敏銳度的變化

### 自我檢視清單
- [ ] 每週是否都與至少 3～4 位學生有個別接觸（對話、靜觀）？
- [ ] HM Block 3 的日常觀察是否涵蓋全班 18 人？（預警：超過 2 週未記錄的同學）
- [ ] 畢旅籌委會的決策過程是否透明且民主？
- [ ] 衝突化解時是否運用了 NVC 的四步驟與修復對話？
- [ ] 對於 CL-07（劉冠廷）與 CL-15（洪晨翊）的支持是否足夠且個別化？

## 學季末評估框架

期末時，導師將對以下向度進行綜合評估，並產出 HM Block 4 的期末發展摘要：

1. **班級凝聚力指標**：同儕關係、參與度、衝突解決能力
2. **個人發展軌跡**：每位學生的頭（思考）、心（情感）、手（行動）變化
3. **導師工作有效性**：教室實踐、親師合作、學生反饋
4. **下學年銜接準備**：九年級課程的銜接預告、學生心理準備度
